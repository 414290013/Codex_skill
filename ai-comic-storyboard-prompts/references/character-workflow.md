# 角色工作流与 @引用系统

## 核心概念：用户绘制角色参考图

用户自行绘制角色形象，上传至即梦Seedance 2.0作为参考图。分镜中的 `@角色名` 即指向这些上传的参考图——Seedance据此锁定角色外观，确保全片41镜角色形象一致。

**标准工作流：**
1. 用户提供角色设定图（正面/侧面/半身/全身）
2. 上传至Seedance（最多9张图，≤30MB/张）
3. 分镜提示词中用 `@角色名` 引用——Seedance自动匹配角色外观
4. 无需在每镜重复描述发型/服饰/体型

**没有参考图时：** 首次提到角色时简要描述外观，后续用 `@角色名` 即可。

## @引用语法

| 素材类型 | 数量限制 | 文件限制 |
|----------|----------|----------|
| 图片 | ≤9张 | 每张≤30MB (jpeg/png/webp/gif) |
| 视频 | ≤3段 | 每段≤50MB，总时长2-15s |
| 音频 | ≤3个 | 总时长≤15s，每个≤15MB |
| 总文件 | ≤12个 | — |

常用引用句式：
- `参考@图1的角色形象和服饰，在@图2的场景中`
- `模仿@视频1的运镜节奏和动作编排`
- `首帧=@图1，尾帧=@图2，中间过渡自然`
- `将@视频1的角色替换为@图1，动作完全复刻原视频`

## 角色专属光影色彩系统

| 角色类型 | 光影属性 |
|----------|----------|
| 屌丝逆袭/好奇/纯真型 | 暖色调/鹅黄光/柔光/低反差 |
| 高冷御姐/质疑/保护型 | 冷色调/蓝调/侧光/高反差/轮廓光 |
| 神秘/意味深长型 | 温和粉紫/霓虹光/柔焦/眼神光 |
| 反派/隐藏BOSS型 | 剪影/逆光/暗绿调/低频阴影/冷金属质感 |

## 角色命名铁律

- 始终用 `@角色名`（如`@洛玉祁`、`@老者A`），不写泛称——"修士""女子""男子""年轻人"一律禁止
- 每个有名称的角色至少分配一个特写镜头

## 衣着/道具绑定角色

衣物和道具必须明确归属。角色参考图已锁定服装全部视觉信息，AI提示词中绝对禁止任何服装颜色+款式描述。

- 错误：`青色衣袍一角高速掠过` `赤色道袍在风中猎猎作响` `黑袍鼓荡`
- 正确：`洛玉祁的衣服一角高速掠过` `修士A的衣服在风中猎猎作响` `黑袍修士的衣服鼓荡`
- 错误：`青色身影飞行`
- 正确：`洛玉祁的身影飞行`

**服装通用词**：`角色名的衣服`或`角色名的衣袍`（分镜行中不带@前缀）——只描述物理运动，不描述视觉外观。

## @前缀使用规则

- **段首「出场角色」行**：使用 `@角色名` 声明参考图绑定。例如：`出场角色：@修士A。@黑袍修士。`
- **分镜提示词行**：去掉@前缀，直接写角色名。例如：`修士A双手握刀` 而非 `@修士A双手握刀`
- **参考音色行**：保留@前缀指向音频文件。例如：`参考音色：@修士A的参考音色@`

## 远景角色标记

远景中角色看不清面部时，用衣服颜色标记位置。`（角色名的衣服颜色）`中的"颜色"是占位符，由参考图自动填充：

- 错误：`青色在前` / `青色小点试图瞬移闪避` / `赤色与暗黑两点`
- 正确：`（洛玉祁的衣服颜色）在前` / `（修士A的衣服颜色）与（黑袍修士的衣服颜色）两点`

## 精简冗余形容

`@角色名` 已锁定外观，不再重复外貌细节：

- 错误：`@老者A——白发白须面容冷峻`
- 正确：`@老者A——面容冷峻`
- 错误：`@洛玉祁下颌与脖颈紧绷，喉结滚动，颈部血管微凸`
- 正确：`@洛玉祁下颌与脖颈紧绷，喉结滚动`

## VoiceID表

为每个说话者建立紧凑VoiceID，后续各行只改变状态和表达方式，不改变声音身份：

| 维度 | 说明 |
|------|------|
| 自然声线指纹 | 角色声音的本质特征 |
| 基准节奏 | 语速/停顿习惯 |
| 呼吸方式 | 呼吸深浅/节奏 |
| 音域 | 高低音范围 |
| 情感底色 | 声音中携带的默认情感 |
| 声优风格指导 | 参考声优/声音类型 |

示例：
```
洛玉祁：青年男声，沙哑，愤怒→威胁→惊恐→决绝
老者A：苍老浑厚低沉
老者B：苍老尖厉
```
