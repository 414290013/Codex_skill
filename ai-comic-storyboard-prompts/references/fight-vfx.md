## 动作拆解五镜公式（国术级打斗编排）

一切动作强制拆解为五步链——让AI理解动作的发力逻辑而不只是外形：

**起势（意图+重心）→ 劲力传导（腰胯→肢体→武器）→ 过程（速度/力量/轨迹）→ 落点（物理反馈+对手反应）→ 反应（情绪外化+战术调整）**

| 步骤 | 核心问题 | 镜头侧重 |
|------|----------|----------|
| 起势 | 重心在哪？蓄力方向？ | 全身或半身——展示重心转移+蓄力姿态 |
| 劲力传导 | 力从何处发起→经何路径→至何处释放？ | 中景或特写——展示腰胯扭转→肩→臂→腕→武器的力链 |
| 过程 | 速度/轨迹/加速度？ | 全景或中景——展示运动轨迹全貌+速度线/残影 |
| 落点 | 命中/格挡/闪避？物理结果？ | 特写——冲击点+粒子爆发+对手受力反馈 |
| 反应 | 双方情绪外化+下一招战术判断 | 近景或特写——面部+瞳孔+呼吸 |

## 国术劲力法则（修真战斗的物理真实性基石）

修真战斗虽为幻想，但角色的身体发力逻辑必须符合真实武术。以下法则确保AI生成的战斗动作"看起来是对的"：

### 劲力传导三节鞭模型

- 人体力链 = 根节（腰胯）→ 中节（肩/脊）→ 梢节（手/腕/武器）
- 力量必须从根节发起经中节加速至梢节释放——不可"只动手不动腰"
- 镜头描述中至少出现一处力链节点：如"腰胯猛然扭转带动肩臂甩出飞刀""脊椎如弓弦绷紧后弹射而出"

### 修真战斗专用劲力词汇

| 劲力类型 | 视觉化描述 | 适用场景 |
|----------|-----------|----------|
| 崩劲 | 短促炸裂，从触点向外爆发，类似爆炸冲击 | 拳掌击中、护盾震开 |
| 缠劲 | 螺旋缠绕，力线呈螺旋线从肢端延伸至武器 | 飞刀回旋、灵力锁链 |
| 透劲 | 力穿体而过，表面不动内部震颤 | 隔山打牛、真元渗透 |
| 化劲 | 以圆破直，力呈圆弧轨迹将对方攻击偏转 | 闪避+格挡+卸力 |
| 整劲 | 全身之力合为一处，身体如一张弓同时释放 | 大招爆发、全力一击 |

### 修真武器劲力映射（刀/剑/镯）

| 武器 | 核心劲力 | 力链路径 | 视觉关键帧 |
|------|----------|----------|------------|
| 飞刀 | 崩劲+缠劲 | 腰胯扭转→肩臂甩腕→指尖拨刀→刀身旋转破空 | 刀脱手瞬间腕部加速度+刀身自旋 |
| 凝聚大刀 | 整劲 | 双脚扎根→双膝微屈→腰胯下沉→双手握刀→全身之力贯入刀柄 | 握刀瞬间双臂肌肉同步隆起+刀身震颤 |
| 横劈竖砍 | 腰马合一 | 腰胯水平旋转→脊椎传导→肩臂挥刀→刀芒延伸 | 腰部旋转带动刀身划弧 |
| 真元盾 | 化劲 | 双手平撑→真元从丹田经双臂→掌心扩散成弧形盾面 | 盾面涟漪从掌心向外扩散 |
| 黑镯 | 缠劲 | 袖中腕部弹抖→镯身旋转脱手→暗紫灵力螺旋缠绕镯身 | 镯身旋转轴心+灵力螺旋纹 |

### 国术铁律（3条）

1. **发力必有重心变化**：每个攻击/防御动作描述中必须隐含重心转移（前倾/后坐/侧移/下沉/上提），不可"悬浮不动只动手"
2. **武器有重量**：大刀挥动时身体应有惯性反馈（手臂肌肉承重、身体随刀势微晃），不可"轻飘飘挥舞"
3. **修真≠无视物理**：真元可以强化力量和速度，但不能替代发力逻辑——灵力沿力链路径传导并放大，而非凭空出现

## 动作特效法则（2D国漫赛璐璐特效合成体系）

2D国漫的特效体系是**纯2D手绘合成**：所有特效——粒子、烟雾、火焰、冲击波——均以2D手绘动画实现，叠加于赛璐璐角色和手绘场景之上。不依赖3D物理模拟。

### 2D国漫特效三层结构

```
顶层 [2D手绘高光特效]  ← 黑白闪/残影拉伸/笔触特效/集中线/残像分身
─── 合成叠加 ───
中层 [2D手绘基础特效]   ← 手绘粒子爆散/手绘烟雾/手绘火焰/冲击波环/电弧/速度线
─── 合成叠加 ───
底层 [2D赛璐璐角色+场景] ← 手绘赛璐璐角色（平涂色块+清晰阴影分界线）+ 手绘场景（水彩薄涂+多层景深）
```

**铁律：2D手绘高光特效（顶层）必须与2D手绘基础特效（中层）同时出现——单一层次=特效扁平。**

### 2D手绘高光特效（顶层——作画与演出）

**冲击帧 / 黑白闪（Impact Frames）：**
- 重击命中瞬间插入1-3帧纯色画面——纯白、纯黑、或单色（红/蓝/黄）
- 常配合负片/反色帧或高反差剪影
- 作用：贩卖冲击重量、制造视觉标点、掩盖动作姿势切换
- Seedance提示词写法：`0.3s 黑白闪——纯白帧，刀盾撞击瞬间画面全白仅留撞击点高光轮廓`

**残像拉伸（Smear Frames）：**
- 快速运动中，物体/肢体沿运动方向拉伸变形的中间帧
- 横斩、直刺、闪避、旋转等高速动作必用
- 2D实现：赛璐璐角色作画+2D手绘残影叠加
- 提示词写法：`角色名横斩轨迹上拉出2D手绘残影拉伸，刀身沿运动弧线延展变形`

**集中线 / 速度线（Speed Lines）：**
- 强调速度方向的放射状或平行线
- 变体：动作线（平行）、聚焦线（向主体放射汇聚）、流线（沿运动路径弯曲）
- 2D实现：2D手绘线叠加于场景之上
- 提示词写法：`速度线集中向刀尖方向汇聚`

**笔触特效（Brush FX）：**
- 水墨/毛笔笔触风格的手绘特效——斩击痕、冲击爆发、灵气光焰、烟雾残缕
- 这是"动漫感"的核心来源，必须由2D手绘完成
- 2D实现：2D笔触合成叠加于场景和角色之上
- 提示词写法：`2D手绘笔触斩击痕沿刀芒轨迹展开，墨色刀气在斩击路径上晕染扩散`

**残像分身（After-Image）：**
- 角色在之前位置留下的虚影复制体，通常3-5个沿运动轨迹排列
- 传达极速的同时保持动作可读性
- 2D实现：角色的多帧作画残影+透明度渐变
- 提示词写法：`角色名身后拉出3个2D手绘残像分身沿闪避轨迹排列，残影透明度从近到远递减`

### 2D手绘基础特效（中层——手绘粒子/烟雾/火焰）

**手绘粒子（Particles）：**
- 浮游尘粒、灰烬余烬、火花爆散、碎片散射、能量微粒（均以2D手绘帧实现）
- 提示词写法：`2D手绘橙金色火花粒子四溅` `2D手绘浮游尘粒在光柱中缓慢飘移`

**手绘烟雾与雾效（Smoke & Fog）：**
- 手绘烟雾柱、热浪扭曲（手绘波动线）、冷气白雾、大气雾/霾、地面扬尘
- 提示词写法：`2D手绘烟雾从撞击点向上翻涌`

**手绘火焰与烈焰（Fire & Flame）：**
- 手绘火焰元素：火球、火焰拖尾（手绘动画帧）、翻腾烈焰+余烬粒子+火光映照
- 提示词写法：`2D手绘火焰从撞击点爆发+余烬粒子飘散+火光映照角色名面部`

**手绘冲击表现（Impact Feedback）：**
- 冲击波环（手绘同心圆环扩散）、地面龟裂（手绘裂纹）、环形尘环、音爆锥（手绘气浪）
- 提示词写法：`2D手绘冲击波环从撞击中心向外扩散`

**手绘能量与魔法特效（Energy & Magic）：**
- 电弧（手绘闪电线）、能量脉冲（手绘光环扩散）、魔法阵（2D手绘线稿）、光束/光焰（手绘渐变光柱）
- 提示词写法：`2D手绘电弧在刀芒与黑镯之间闪烁`

### 动作导演技法（子弹时间与速度控制）

动作导演负责打斗节奏的时间操控——何处加速、何处冻结、何处慢放。2D多平面空间中可自由操控时间流速：

#### 子弹时间（Bullet Time）

高速动作的瞬间时间冻结，摄影机绕主体旋转，观众看清每一帧的物理细节。适用于：致命一击命中瞬间、闪避毫厘之差、法宝出手定格。

**赛璐璐子弹时间三镜模块（1.5s总时长）：**
- 镜1（0.5s·正常速度）：攻击起势——主体高速运动中
- 镜2（0.7s·子弹时间）：瞬间冻结——`时间感知冻结，低角度慢环绕，极特写，飞刀刀尖距黑袍修士的衣服仅毫厘，刀刃震荡波在空气中凝结成可见的环形波纹，2D手绘粒子悬浮静止`——摄影机绕主体旋转，画面中所有粒子/布丝/水滴悬停
- 镜3（0.3s·速度释放）：时间解冻——粒子恢复运动，斩击完成

**子弹时间提示词写法**：`时间冻结，低角度慢环绕，[主体]瞬间定格，周围2D手绘粒子/碎片/布丝悬浮半空静止，仅摄影机绕主体缓慢旋转`

#### 慢动作（Slow Motion）

关键动作以0.2-0.5倍速展开，让观众看清动作工序的每一帧。适用于：大招蓄力过程、武器凝聚变形、真元从丹田→肢端的传导路径。

**慢动作镜头写法**：`慢动作，[运镜]，[动作工序以0.3倍速逐帧展开]，每帧可见[物理细节]`

#### 快慢交替（Speed Ramping）

同一镜头内速度突变——正常→慢动作→正常→冻结。适用于：连招中的节奏变化。

**快慢交替节奏模板：**
- 连斩三刀 = 正常速度(横劈0.3s) → 慢动作(竖劈核心帧0.5s) → 子弹时间(终击命中冻结0.4s) → 正常(收刀0.3s)

#### 5条速度控制铁律

1. **子弹时间每段最多1次**——过度使用=节奏拖沓
2. **冻结帧不超过0.8s**——超过则观众失去时间感
3. **冻结必有环绕**——子弹时间必须配合摄影机运动（慢环绕/微推/微摇），禁止纯静止画面
4. **慢动作必须有细节**——若慢放后画面没有值得看的物理细节，就不该慢放
5. **速度释放要干脆**——冻结后解冻瞬间加速至正常速度，不可渐变

### 动作特效铁律（6条）

1. **2D双层必须**：每次重击/爆发/大招必须同时包含2D手绘高光特效和2D手绘基础特效。单一层次=特效廉价感
2. **冲击帧节制**：黑白闪仅用于全片最重要的3-5个撞击瞬间。每段最多2个冲击帧。滥用=视觉疲劳
3. **粒子有来有去**：每次粒子爆散后必须在1-2镜内展示粒子消散（飘散/坠落/淡出），不可凭空消失
4. **特效有光源**：大型特效（爆炸/光柱/冲击波）必须描写其对场景和角色的光照影响——"火光映照面部""金光将云层染成淡金"
5. **Seedance物理化**：即梦不理解"特效"概念，只理解物理事实。不写"华丽的爆炸"，写"2D手绘金色粒子从中心向外爆散+冲击波环扩散+手绘烟雾翻涌"
6. **火焰/能量攻击禁止同框（致死级）**：攻击方释放火焰/能量时，被攻击方不得出现在同一镜头中。Seedance 在同框画面中会将两者之间的火焰/能量直接"连接"到被攻击方身上，造成烧身/击中穿帮。强制分镜——镜A：攻击方正面释放（火焰/能量向镜头方向或画外扩散），镜B：切至被攻击方反应特写（火光映面但火焰未及身）。此规则适用于所有远程能量攻击：火海、冰锥、雷电、真元冲击、灵力波动等。

  ↓
释放：掌心凭生吸力（空气扭曲波纹+气流线向掌心汇聚）
  ↓
汇聚：火海化作螺旋火线汇入掌心（宏大奇观+数十道火线+能量压缩感）
  ↓
收纳：火焰涌入翅膀/体内（流光吸入+翅膀暴涨+天空重归清明）
  ↓
余韵：吸尽后瞬间死寂（环境静默+残留火星飘散）
```

### 火焰翅膀展开链

起势：压低重心身形爆发（重心下沉+背后火焰粒子汇聚）
  ↓

```

## 打斗动作节奏控制（需求文档精华）

### 文戏→动作爆点切换

```
文戏层(3-5s)：对话/观察/对峙——固定/微推，慢节奏
   ↓ 节奏突变（L-Cut: 声音先入0.3s）
起势帧(0.5-1s)：身形爆发压低重心——快切+震动
   ↓ Match Cut
释放帧(0.5-1.5s)：火焰翅膀展开/音爆起飞——跟拍+仰拍
   ↓ Action Match
爆发后(2-3s)：高速飞行/破空——速度线+残像+音爆
```

### 偷袭/突发危机节奏

```
预兆(1-2s)：识海警示/微感应——面部特写微推
   ↓ 硬切+震动0.15s
爆点(0.5-1s)：偷袭出手/瓷瓶炸裂——快切
   ↓ Match Cut
反应(1-2s)：被偷袭方反馈——固定/微震
   ↓ L-Cut
后果(2-3s)：黑气入体/状态变化——推镜+环境暗化
```

### 三拍子战斗节奏模板

修真战斗的标准节奏——"三拍一击"：

```
第一拍·威胁(1-2s)：攻击方出手/法宝释放——攻击方的攻势建立
第二拍·应对(1-2s)：被攻击方应对/防御/闪避——防御方的即时反应
第三拍·结果(1-3s)：攻防交互结果——碰撞/格挡/命中/闪避
加一拍·余韵(1-2s)：双方情绪反馈+下一招战术判断——为下一循环铺垫
```

**铁律**：三拍循环每2-3轮插入一个"呼吸拍"——1-2s环境空镜/留白，让观众喘息。

## 生理反馈五层链（打斗命中专用）

每次重大命中必须覆盖至少3个可见层级；内部感受必须转写为姿态、呼吸、视线、衣物道具、环境声光变化：

| 层级 | 控制维度 | 时间窗口 | 火焰烧伤示例 | 重击命中示例 | 灵力侵蚀示例 |
|------|---------|---------|-------------|-------------|-------------|
| 一层·体表 | 皮肤/毛发/汗腺 | 0-0.5s | 皮肤焦黑、衣物炭化冒烟 | 脸色煞白、冷汗渗出 | 皮肤浮现暗色纹路 |
| 二层·骨骼 | 脊椎/关节/重心 | 0.2-0.8s | 身体蜷缩、重心失衡 | 身体下沉、脊椎弯曲 | 关节锁死、身体僵硬 |
| 三层·呼吸/声音 | 呼吸/心跳声/肌肉 | 0.5-1.5s | 吸入浓烟呛咳、呼吸急促 | 胸口急促起伏、呼吸断半拍 | 灵力光流一滞、呼吸收紧 |
| 四层·神经 | 瞳孔/意识/听觉 | 0.3-1.0s | 瞳孔极度收缩→意识模糊→惨叫嘶吼（非修真者）/面部扭曲无声（修真者） | 瞳孔放大、耳鸣 | 瞳孔忽大忽小、视线扭曲 |
| 五层·情绪 | 面部/声音/行为 | 1.0-3.0s | 疯狂腾挪跳转→试图扑灭火焰→身体失控 | 牙齿打颤、眼神绝望 | 面部扭曲、嘶哑低吼 |

**铁律**：
- 每层之间是递进关系（0→0.5→1.0→1.5→2.5s 时间线），不可跳跃
- 一层和二层必写（最基础的可见反馈：体表/衣物+姿态/重心），第三层从三四五层中按需选择
- 极端伤害（如雷劫、天罚）：五层全写
- **修真者疼痛表达**（与xianxia-genre.md一致）：修真者使用克制型表达——`咬牙闷哼` `面部扭曲无声` `灵力在伤口汇聚自愈` 替代外放型惨叫。非修真者/反派可使用外放型表达
- 微表情生理链模板：`瞳孔X→嘴角Y→面部肌肉Z→呼吸W`
