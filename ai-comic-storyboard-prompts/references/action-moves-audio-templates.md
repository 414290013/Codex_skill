# §A 武侠 / 仙侠特效（七大流派）

## A.1 内功气劲类（武侠基础层）

### 剑气
三种形态：
1. **物理风压** —— 挥剑产生的空气冲击波（写实派）
2. **真气剑芒** —— 注真气于剑，剑刃延伸出可见光刃
3. **真气直射** —— 体内真气直接射出，脱体而出的能量束

### 掌风 / 罡气
- 可见劲气波纹（如热浪扭曲）
- 肉眼可见的同心圆音爆环
- 透明掌印悬空、空气被压缩成实

### 真气护体
- 透明灵气漩涡环绕全身
- 淡光膜防护罩
- 罡气形成的薄膜外壳

### 凌空虚渡（御空类）
- 御剑飞行（站在剑上）

# §C 周围震荡描写库 ⭐（核心调用区）

# 武术指导动作设计库

适用范围：把招式名称、动作母体、特效、音效、环境震荡和安全改写整合成可用于分析、提取、回收的武指动作设计工作台。

## 4. 怎么分析提取一场参考动作

以后遇到“这段参考视频很好看，想抽出来给项目用”，不要只记“像袁和平”或者“很港风”。

建议按这 6 个维度提取：

1. `动作目标`
谁压谁，谁躲谁，谁逆转谁，结果是什么。

2. `动作母体`
写实都市、江湖武侠、权力压制、仙侠法术、科技修真、神话奇观，先定大类。

3. `武指参考`
再判断更像哪一路武指设计，是轻灵对拆、飘逸纵深、贴身重击、传统拆招，还是地形 stunt。

4. `动作链`
至少拆成 `起手 -> 压制 / 交换 -> 反制 -> 结果` 四段；再短也要保住结果节点。

5. `空间与器械`
人和人的距离怎么变，武器怎么过手，环境有没有被动作真正利用。

6. `镜头与声音接口`
哪一拍需要停顿，哪一拍要给反应镜，哪一拍只靠声音支撑，不要把动作和镜头拆成两张皮。

## 7. AI 项目里的使用边界

这层库是为了让分析更准，不是为了把动作写得更满。

固定原则：

- 先抽 `压制关系`，再抽 `招式外观`
- 先抽 `结果节点`，再抽 `中间花活`
- 先抽 `空间变化`，再抽 `酷炫身法`
- 一段动作最多保留 `2-4 个高辨识节点`

如果是 AI 视频：

- 武指参考只能帮你定语法，不能直接照搬复杂连招
- 参考越偏空翻、威亚、缠斗，多半越要降复杂
- 群战优先保 `站位变化`，不要保所有接触细节

# 武指动作设计案例包

适用范围：提供高复用武指案例层，方便直接分析、提取、套用。

## 1. 这份案例包解决什么

有了母体库以后，下一步最常见的问题不是“概念不懂”，而是：

1. 知道袁和平、程小东、洪金宝这些名字，但不会落到项目动作
2. 知道要写“轻灵”“飘逸”“贴身”“群战”，但不会拆动作链
3. 规则层都在，真正开写时还是不知道先保哪几个节点

所以这份案例包专门做两件事：

- 给你第一批可以直接抄的动作母体样板
- 把“武指参考”落到 `动作链 / 镜头 / 特效 / 声音 / AI 降复杂`

## 2. 最短使用方法

固定顺序：

1. 先看哪一类场景最像你现在这场戏
2. 抄它的 `动作链`
3. 再抄它的 `镜头 / 声音 / 特效`
4. 最后按你的项目题材和平台要求减复杂

一句话：

`先借编法，再换角色和招式，不要反过来。`

## 4. 案例 A · 袁和平系：竹林师徒切磋

- 适合：
  江湖切磋、门派演武、师徒过招、男女主第一次显露身法
- 动作强度：
  `L2`
- 动作链：
  - 起手：先试探，不急着真打死
  - 压制：一步进、一剑递、一掌封
  - 反制：靠身法变位，不靠硬扛
  - 结果：点到为止，但胜负已清
- 镜头 / 声音 / 特效：
  - 镜头：中景起，短促切特写，命中点后立刻回中景
  - 声音：兵刃轻响、破风、落地竹叶声
  - 特效：只给轻量尾迹和竹叶扰动，不要大光效
- 为什么像袁和平：
  这类动作值钱的不是谁更炸，而是 `你来我往、拆得清楚、人物身法有等级差`
- AI 降复杂：
  - 保留 `2 次明显变位 + 1 次反制`
  - 删掉连续翻腾
  - 不在同一镜里塞三次以上出招

## 5. 案例 B · 袁和平系：主角翻盘 Boss 战

- 适合：
  主角被压一整段后反制、师门复仇、比武大会关键翻盘
- 动作强度：
  `L3`
- 动作链：
  - 起手：Boss 先压，主角先守
  - 压制：Boss 连续进逼，主角只剩闪和挡
  - 反制：抓住半拍破绽，用一步身法抢回中线
  - 结果：主角打出第一记真正有效命中，但不立刻结束
- 镜头 / 声音 / 特效：
  - 镜头：低角度给 Boss 压迫，主角破局时急推
  - 声音：前半多喘息和压迫脚步，破局点给半拍静默再爆
  - 特效：环境裂纹、烛火熄灭、兵刃震颤，比大光更重要
- 为什么像袁和平：
  `层级差 -> 学会 -> 反拿回主动` 这种递进感，比单招大词更像这一路
- AI 降复杂：
  - 整段只保 `压制 2 节 + 破局 1 节 + 结果 1 节`
  - 破局点只给一次
  - 环境反馈替代复杂连招

## 6. 案例 C · 程小东系：月下轻功追杀

- 适合：
  女侠、刺客、追逐、古桥 / 屋脊 / 林间轻功戏
- 动作强度：
  `L2-L3`
- 动作链：
  - 起手：远距发现目标，不直接近身
  - 压制：以纵向位移和高低差逼人
  - 反制：借栏杆、飞檐、树梢转身回刺
  - 结果：一招落定后给剪影停顿
- 镜头 / 声音 / 特效：
  - 镜头：远景交代月色和高低差，中景抓落点，近景只留一记
  - 声音：衣袂、踏木、掠风、剑身轻鸣
  - 特效：冷月边缘光、轻雾、轻尾迹，比粒子更要紧
- 为什么像程小东：
  不是打得重，而是 `人物像在空中写字`，空间要有纵深和飘感
- AI 降复杂：
  - 保 `起跳 -> 落点 -> 回身` 三节点
  - 同时飞行和出招只保一次
  - 不要在一个镜头里跨三个空间层

## 7. 案例 D · 程小东系：御剑法术对轰

- 适合：
  仙侠对决、御剑斗法、女主飞升前试炼、两大宗门空战
- 动作强度：
  `L4`
- 动作链：
  - 起手：先拉开距离，先有法术蓄势
  - 压制：一方御剑推进，一方法阵拦截
  - 反制：换轨迹，不换满屏发光
  - 结果：人物仍清楚，奇观只是承托
- 镜头 / 声音 / 特效：
  - 镜头：广角、环绕、俯仰结合，但要先交代两人相对站位
  - 声音：低频共鸣、法器鸣动、剑啸
  - 特效：法阵、轨迹、云层、碎光，重点是层次，不是全屏爆
- 为什么像程小东：
  这一路最值钱的是 `御剑和人物不是两张皮`，人永远是奇观里的焦点
- AI 降复杂：
  - 一段只保 `1 种主法术 + 1 条主轨迹`
  - 不让两边同时爆满
  - 用远景尺度感替代过多粒子

## 8. 案例 E · 洪金宝系：市井群战突围

- 适合：
  帮派混战、码头追打、夜市群殴、多人围困
- 动作强度：
  `L1-L2`
- 动作链：
  - 起手：主角被围，先给站位压力
  - 压制：敌方从不同方向逼近，不急着全扑上来
  - 反制：撞、挡、推、摔，借桌椅门框破局
  - 结果：主角撕开缺口，不一定当场全赢
- 镜头 / 声音 / 特效：
  - 镜头：中近景跟拍为主，群体关系先于个人花活
  - 声音：撞击、脚步、木器翻倒、呼喊
  - 特效：基本不靠特效，靠环境动起来
- 为什么像洪金宝：
  `重、实、挤、撞` 才是关键，群战的压力来自空间而不是人数口号
- AI 降复杂：
  - 一次只保 `3 个方位`
  - 明显保一个核心障碍物
  - 不写多人长时间抱缠

## 9. 案例 F · 甄子丹系：走廊制服近身战

- 适合：
  都市悬疑、制服抓捕、地铁 / 走廊 / 电梯口近身控制
- 动作强度：
  `L1-L2`
- 动作链：
  - 起手：逼近，不绕大圈
  - 压制：短连击 + 控制对方出手线
  - 反制：打断节奏，切进中线
  - 结果：制服、压墙、放倒，迅速收束
- 镜头 / 声音 / 特效：
  - 镜头：中近景、短急推、局部特写
  - 声音：脚步、衣料、短促命中、墙面撞击
  - 特效：几乎无，重点是节奏和命中点
- 为什么像甄子丹：
  `短、狠、准、收得快`，不是长段炫技，而是控制感
- AI 降复杂：
  - 删掉连续十几手
  - 保 `逼近 -> 打断 -> 控制 -> 收束`
  - 命中后必须有停顿
