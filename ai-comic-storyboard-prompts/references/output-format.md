# 用户音色引用硬规则

- 所有人物参考音色必须由用户上传到即梦的音频文件决定。
- 规划稿可写占位符：`@角色名的参考音色@`；最终粘贴到即梦前必须替换为真实上传音频素材编号，例如 `@音频1`、`@音频2`。
- 禁止在 VoiceID、参考音色、角色设定或段首参考音色中写年龄、性别、声线、音色性格、气质判断等文字画像，例如“年轻男性”“成年男性”“清亮”“懒散”“贪念时声音收窄发亮”。
- 段首使用 `声音资产绑定` 建立角色和真实上传音频素材的映射。
- 每个有声台词镜头必须在台词前写短声源锁：`声源锁：角色=@音频X；`。
- 每镜台词格式使用：`声源锁：角色=@音频X；台词：角色："原文" 表演状态`。
- 内心独白格式使用：`口型锁：角色闭口不张嘴，无口型对白，声音为内心旁白；声源锁：角色=@音频X；台词：角色（VO）："原文" 表演状态`。
- 文字可以描述本句表演状态，如停顿、重音、呼吸、语速、尾音，但不能替代或扩写参考音色。

# 分镜输出格式规范

## 第六步：分镜输出（纯文本行，无表格）

输出为纯文本行，方便复制粘贴到即梦Seedance。不使用markdown表格。

### 段首统一声明块

全片统一使用以下段首声明（2D国漫赛璐璐画风）：

```
## 大分镜X：标题（起始时间-结束时间）

出场角色：@角色A。@角色B。@角色C。
**本段允许资产：** 人物：@角色A，@角色B，@角色C；场景：@主场景名；道具：@道具A/无；文字元素：@文字面板A/无；特效元素：特效A/无。
**声音资产绑定：** 角色A台词/角色A（VO）只使用@音频1；角色B台词/角色B（VO）只使用@音频2；禁止互换、混用、随机音色。
**场景：** [本段场景描述——地点、环境、光线、氛围、天气/时间]
**身高比例锁：** [除用户明确指定身高/体型差的角色外，所有成年主要角色统一成人身高；同框站立时头顶高度基本齐平、肩线接近，禁止一高一矮、儿童化、缩小体型]
**站位基线：** [@角色A坐标（X,Y,Z）层次/朝向/屏幕位置；@角色B坐标（X,Y,Z）层次/朝向/屏幕位置；战斗距离X米；轴线方向]（动作场景必填）
**承接上一段：** [第1段可省略；从第2段开始必须写清上一段末镜如何过渡到本段首镜，如动作承接/道具承接/视线承接/声音承接/遮挡转场/匹配转场/空间建立硬切]
**负面排除：** 无字幕，无背景音乐，仅保留环境音效和对白；如有台词、对话、旁白，使用默认字体，不透明度为0。
**画面构成：** [可选——有严格画面控制需求时填写。声明画面内的取景范围和画面外排除的内容]
```

- **出场角色**：只列出本段所有出场人物角色 `@角色名`，句号分隔。若人物在本段中途退场/新增，仍全部列出。道具、文字面板、场景、特效、武器、丹药、法器、车辆、动物群、背景群众轮廓等非人物资产不得放入 `出场角色`。
- **本段允许资产**：声明本段允许 Seedance 调用的资产类型，格式为 `人物：@角色A，@角色B；场景：@场景名；道具：@道具名/无；文字元素：@文字元素名/无；特效元素：特效名/无`。如果某类没有资产，写 `无`。凡是 `@觉耳丹`、`@丹药介绍面板` 这类非人物引用，必须放在 `道具` 或 `文字元素`，不能放在人物栏。
- **声音资产绑定**：本段有台词（含VO/内心独白）的角色必须列入，格式 `角色台词/角色（VO）只使用@音频X`。无台词角色不列。`@角色名的参考音色@` 只用于本地规划，最终进入即梦时要替换为真实上传音频引用。禁止只写一串音色占位符而不写角色归属。
- **身高比例锁**：人物同框前必须声明身高关系。只有用户明确指定身高差、儿童、巨人、矮小、坐姿/跪姿等特殊情况时才允许形成明显高低差；否则所有成年主要角色默认同一身高等级。
- **站位基线**：动作场景必填——使用坐标系统定位角色空间关系，角色用 `@角色名坐标（X,Y,Z）` 格式
- **承接上一段**：从第2个大分镜开始必填。必须描述上一段末镜的可继承元素和本段首镜的接入方式，禁止只写“转场”。

### @视觉引用铁律

- `@角色名` 只用于绑定视觉角色参考图，不是普通人名标点。
- `@道具名`、`@场景名`、`@文字元素名` 不是角色引用，必须先在 `本段允许资产` 中归类；不要把道具或面板写进 `出场角色`。
- 每个镜头行里，同一个可见角色通常只在第一次视觉出场时加一次 `@`；同一行后续动作、视线、身高、情绪、台词说话人都用普通名字。
- 台词说话人不用 `@`：写 `台词：虾仁："..."`，不要写 `台词：@虾仁："..."`。
- 音频引用只在 `声源锁` 中使用：`声源锁：虾仁=@音频1；`，不要把它和视觉角色 `@虾仁` 混用。
- 如果一个镜头里同一个角色反复写 `@角色名`，Seedance 可能理解为多个同名角色，增加复制、错位、人数膨胀风险。
- 例：`@白不分从画面右边缘快步入画，衣袖随步伐摆动；视线：看向虾仁`，不要写成 `@白不分从画面右边缘快步入画，@白不分衣袖摆动，视线：看向@虾仁`。

### 声音资产绑定铁律

- Seedance 对多角色声音的绑定不稳定时，不能只在段首写 `参考音色：@A@。@B@`，因为模型可能把它当成普通说明文字。
- 段首改用 `声音资产绑定`，明确“谁的台词/VO只使用哪个真实上传音频素材”。
- 每个有台词镜头必须在 `台词：` 前写一句短绑定：`声源锁：角色=@音频X；`。
- 单镜只有一个说话人时，也保留声源锁；多角色同镜说话时，每句台词前分别写自己的声源锁。
- 如果仍然混音，优先把一个大分镜拆成“单说话人生成单元”：一个生成片段只保留一个角色有声台词/VO，其他角色只做无声反应；或只让 Seedance 生成画面，把最终配音交给单独配音/TTS流程。
- 禁止写“参考音色：@A@。@B@”后就让模型自行判断谁说哪句。

### 内心独白口型锁

- Seedance 看到 `台词：角色（VO）` 仍可能生成角色开口说话，所以 VO 必须显式锁口型。
- 每个内心独白/旁白镜头，在 `声源锁` 前写：`口型锁：角色闭口不张嘴，无口型对白，声音为内心旁白；`。
- VO 镜头的画面表演用眼神、眉尾、嘴角压住、喉结、呼吸、肩膀、手指表达，避免写 `嘴唇微张`、`张嘴`、`开口`、`说话口型`。
- 如果角色需要真实说出口，就不要标 `（VO）`，按普通台词写。

### 人物身高比例铁律

- 有明确身高的角色，先记录数字，例如 `白不分参考身高175cm`。
- 未明确身高的成年主要角色，默认与同场成年主要角色保持同一身高等级，不得被生成成矮子、小孩、少年体型或大头小身。
- 同框站立镜头必须写可视关系：`二人同为成人身高，头顶高度基本齐平，肩线接近`。
- 如果有远近景透视，必须说明这是透视造成的视觉大小变化：`白不分在画面右后景略小是距离透视，不是身高差`。
- 拉袖、并肩、面对面交谈、双人中景等镜头，优先加入：`禁止一高一矮，禁止虾仁儿童化或缩小体型`。
- 不要只写 `175cm`，Seedance 更容易执行“头顶齐平、肩线接近、同为成人体型”这类画面关系。

### 大分镜时长铁律

- 每个大分镜是一次 Seedance 2.0 可生成片段，最长 **15秒**。
- 超过15秒必须拆成多个大分镜，不得输出 20s、30s 或更长的大分镜段。
- 文戏/对白段建议 8-15s，3-6镜；喜剧心理段建议 6-12s，4-7镜；动作段建议 10-15s，6-12镜。
- 每段首镜必须有 0.5-1.5s 的空间建立或上一段承接；每段末镜必须留 0.5-1.5s 的视觉/声音/动作钩子。
- 10s 以上单镜或一镜到底必须写镜内时间段；否则拆成多镜。

### 大分镜之间的衔接铁律

两个大分镜之间必须设计“上一段末镜 → 下一段首镜”的桥：

| 优先级 | 衔接方式 | 用法 |
|------|----------|------|
| 1 | 动作承接 | 上段末镜伸手/迈步/转身；下段首镜承接动作到达 |
| 2 | 道具承接 | 上段末镜聚焦道具；下段首镜用同一/相似道具接入 |
| 3 | 视线承接 | 上段角色看向画外；下段首镜出现目标或沿视线建立空间 |
| 4 | 声音承接 | 下一段声音先入或上一段声音延续，使用 J-Cut/L-Cut |
| 5 | 遮挡转场 | 衣袖、门框、墙面、烟尘完全遮挡后切入新空间 |
| 6 | 匹配转场 | 形状、颜色、运动方向或光源匹配，如丹药圆形→朝阳 |
| 7 | 空间建立硬切 | 仅用于明确时间/地点变化；下一段首镜必须先建立空间 |

段尾增加：

```
**下段衔接钩子：**
[说明下一段如何承接本段末镜，禁止只写“转场”]
```

下一段段首的 `**承接上一段：**` 必须与上一段钩子一致。

### 即梦AI视频提示词格式铁律（22条）

1. **运镜优先**：以运镜方式或摄影机指令开头——`固定，极特写，...` / `慢推镜，近景平拍，...`
2. **群体人数锁定（致死级——人数不符=本镜作废）**：画面中有N个确定角色时，必须逐一写出N个独立的 `@角色名`（如 `@老者A` `@老者B` `@老者C` `@老者D`），一个都不能少。**绝对禁止集体泛称**——不写"四名老者""几个修士""一群弟子""四道人影"。集体泛称不产生 @引用计数，AI 无法确定具体人数。每个不同的 @引用 = AI 的1个人物预算，少写1个 @引用 = 画面少1个人。
3. **角色命名**：使用 `@角色名`（如`@洛玉祁`、`@老者A`），**绝对禁止泛称**——不写"修士""女子""男子""年轻人"
4. **动作物理化**：情绪用身体反应表达——不写"悲伤"，写"低头咬牙眼眶泛红"
5. **主语必命名**：动作必须有 `@角色名` 作为施动者——不写"云絮被高速冲破"，写"云絮被@洛玉祁高速冲破"；禁止无主语的被动句式
6. **光线必写**：侧逆光/轮廓光/冷白顶光/暖金反光/丁达尔/霓虹光——不用"好的光线"
7. **禁止元数据侵入**：不写 `[原]/[背]/[撮]` 层级标签，不堆砌技术参数（8K/HDR/120fps等）
8. **黄金密度（致死级）**：AI提示词25-60中文字——这是AI理解精度最高的密度区间。低于20字=信息不足画面空洞，超过80字=多焦点竞争AI随机选择。每镜只有一个核心视觉焦点。写完即删：删掉所有AI不需要也能生成正确画面的字
9. **衣着绑定角色**：衣物用`角色名的衣服`或`角色名的衣袍`，**绝对禁止任何服装颜色+款式描述**。服装只描述物理运动，颜色/材质/款式全部由参考图决定。分镜行中角色名不带@前缀
10. **远景用颜色标注**：写`（角色名的衣服颜色）`——"颜色"是占位符，由参考图自动填充。不写具体颜色名
11. **远景禁止抽象化**：远景中人物保持可辨识的人形轮廓与衣袍特征——不写"五个光点呈一前四后狩猎队形""气流轨迹线将光点串联"，写"@洛玉祁的身影在前方远处飞行，身后@老者A/@老者B/@老者C/@老者D四人的人影呈扇形散开，金色灵力轨迹拖在身后"；禁止将人物降维为几何光点、禁止战术示意图式描述
12. **精简冗余形容**：已用 `@角色名` 明确身份后，删除冗余外貌词——不写"白发白须面容冷峻"，只写"面容冷峻"
13. **@视觉引用去重**：同一提示词中，同一视觉角色 `@角色名` 最多出现1次。若首次 `@角色名` 已绑定角色，后续同一行的动作、视线、身高、情绪、台词说话人使用普通名字。`声源锁` 中的 `@音频X` 属于音频引用，不计入视觉角色次数。
14. **利刃方向法（致死级）**：所有刃器（飞刀/剑/刀/匕首）在飞行、斩击、指向时，刃尖必须朝向目标——每镜含刃器时至少一处明确刃尖方向
15. **台词一字不差**：不准写"读关联原文"，不准改写，不准概括——脚本对话原封不动保留
16. **衔接逻辑必写**：每镜末尾标注 `[衔接: 主逻辑+次逻辑]`（本段末镜除外）——复合连接标签，主逻辑保证画面连续性，次逻辑保证情绪连贯性
17. **段首站位基线必填**：动作场景段首必须标注角色坐标站位基线——@角色名坐标（X,Y,Z）+ 战斗距离 + 轴线方向
18. **入画出画铁律（致死级）**：所有新出场人物必须从画面**四边之一**进入画面——绝不从画面深处凭空出现、绝不从画面中央淡入。人物退场必须从画面边缘走出去——绝不从画面中间凭空消失。出画后若镜头持续，下一秒明确声明"画面中无X角色"。
19. **出画=固定镜头（致死级）**：角色离开画面时，运镜指令必为`固定`——禁止摇镜/跟拍/推镜。摇镜方向与飞出方向矛盾=即梦画面滞留/角色悬停在画面边缘
20. **画面排除声明**：有严格画面控制需求的镜头，明确声明**画面里看不到什么**——`画面里看不到地面、看不到天花板` / `画面中无黑袍` / `全程只看下半身——画面里完全不出现任何人的上半身和脸`。排除声明 = 告诉AI不要生成什么。
21. **手部特写铁律（致死级）**：手部/手指特写必须写精确手指数量+用衣袖/手套部分遮挡——即梦不理解手部解剖结构，不约束=多指/缺指/关节反向穿帮
22. **蒙面角色禁写五官（致死级）**：兜帽/蒙面/面具完好时声明`不见任何五官`，用身体语言+声线传递情绪——即梦不理解物理遮挡，会在蒙面布上凭空生成嘴/眼/眉毛

### 动作场景专项铁律（5条——分镜师必查）

**23. 动作四镜必拆（致死级）**：任何动作（出招/拔剑/施法/受击/躲闪）必须拆为起势→过程→落点→反应四个独立镜头——每镜只承担一个叙事功能，一镜塞完整动作链=作废。
- 起势（意图）：特写/极特，0.5-1.0s——蓄力/眼神/肌肉紧绷
- 过程（力量）：中景/全景，0.8-1.2s——动作释放瞬间，速度感+视觉奇观
- 落点（物理反馈）：全景/特写，0.8-1.5s——火花/烟尘/气浪/形变等物理结果
- 反应（情绪外化）：特写/极特，0.5-1.0s——受击者/旁观者生理反应

**24. 入场必验位**：新角色入场前，必须先读段首站位基线确认到达坐标（X,Y,Z）+ 空间关系，再写入场镜。入场镜必须明确——①挡在谁身前 ②面向谁 ③从画面哪条边入画。禁止不读站位基线直接写"X出现在画面中"。

**25. 反应必拆镜**：角色出现以下情绪转变时→独立成镜，不可与动作镜合并：
- 惊呼/质问台词："是你！""怎么可能！""你……你还活着？"
- 表情极性反转：得意→震惊、愤怒→恐惧、平静→崩溃
- 反应者此前未出镜
- 格式：`X.Xs 固定，近景，以@角色名为主角，场景参考@，[身体反应+台词]`——运镜必为`固定`

**26. 质感收尾词必写**：每镜AI提示词末尾包含1-2个质感关键词——`动态模糊`/`高对比度`/`边缘高亮`/`浅景深`/`剪影`/`电影级质感`/`微距摄影`

**27. 远景观察者面对镜头**：角色远处观察时：
- 观察者面对镜头（或微偏），画面中不出现被观察者
- 末尾声明`画面中无X无Y`
- 用环境音暗示远处事件
- 运镜必为`固定`——禁止摇镜/推镜

### 即梦高风险元素黑名单（写入=穿帮）

| 黑名单 | 穿帮机制 | 安全替代 |
|--------|---------|---------|
| 残像分身 | 即梦逐镜独立，残像=多个人物→畸变 | 固定镜头+45°出画→下镜切至到达位 |
| 目光穿过X锁定Y | 渲染远处Y但无跨镜记忆→空镜 | 角色对镜头；末尾`画面中无Y`；音效暗示 |
| 眼睛映出远方火光 | 不理解"映出"→瞳孔贴错误光斑 | 用面部光影变化暗示光源方向 |
| 飞出画面+移动镜头 | 摇镜与飞出方向矛盾→滞留 | 飞出=固定镜头；下镜切到达位 |
| 入场镜内混对手台词 | 动作+文戏节奏混=即梦节奏乱 | 入场镜无台词→独立反应镜 |
| 蒙面角色五官描写 | 不理解遮挡→蒙面布上生嘴/眼 | `不见任何五官`+身体语言+声线 |

### 道具跨镜连续性铁律（致死级——道具转移/收回/销毁后必读）

即梦逐镜独立生成，不继承上一镜的道具状态。道具被收走/转移/销毁后，下一镜必须用**物理否定描述**明确声明道具的缺席，否则AI必然在画面中残留该道具。

**错误（道具残留）：**
```
镜A：@凌无策将小钟收回袖袍之内
镜B：小钟刚收起@赵玉林的症状瞬间就好了  → AI看到"小钟"就在画面中生成小钟
```

**正确（物理否定）：**
```
镜A：@凌无策五指一翻将小钟收入袖袍，袖口合拢，小钟彻底没入袖内
镜B：@赵玉林双手空空五指张开撑地，画面中无小钟，@凌无策袖袍已合拢，@赵玉林苍白着脸喘息
```

**道具连续性三原则：**
1. **道具转移后，下一镜明确新持有者**：写"小钟在@凌无策袖袍内，@赵玉林手中空无一物"
2. **道具消失后，用物理否定词描述缺席**：写"画面中无小钟""双手空空""地面仅青石地砖"，不写"小钟已消失""钟不见了"
3. **关键道具首次出现绑定角色归属**：写"@凌无策掌中的小钟"，不写"一口小钟"——防止AI将道具分配给错误角色

此规则与"角色消失=删除@引用"平行：角色消失→删除@引用；道具消失→物理否定描述其缺席。

### 自然语速反推时长

- 冷静旁白 ≤ 5中文字/秒
- 情绪旁白/内心 ≤ 4中文字/秒
- 正常对白 ≤ 3.5-4中文字/秒
- 哭腔/求饶/疼痛/恐惧/威胁/低语 ≤ 2.5-3.2中文字/秒
- 标点停顿：逗号+0.15-0.25s、句号+0.25-0.45s、问号/叹号+0.30-0.60s、省略号+0.60-1.00s

说不完就增加镜头拆分，绝不能让人声越说越快。

### 时长计算依据公式

每镜时长必须有明确计算依据，不能凭感觉赋值：

```
单镜时长 = 台词时长 + 动作缓冲 + 情绪留白
```

| 计算项 | 公式 | 说明 |
|--------|------|------|
| 台词时长 | 台词字数 ÷ 对应语速 | 冷静旁白≤5字/s / 正常对白3.5-4字/s / 情绪对白2.5-3.2字/s |
| 标点停顿 | Σ各标点停顿值 | 逗号+0.2s 句号+0.35s 问号/叹号+0.45s 省略号+0.8s |
| 动作缓冲 | 0.5-3.0s | 每个关键动作用时：蓄力0.5-2s + 释放0.3-0.8s + 反馈0.5-2s |
| 情绪留白 | 0.5-2.0s | 情绪高潮后的呼吸空间——让观众消化 |

**示例**：
```
台词"此人与销声匿迹的炼尸一脉有重大关联"=14字÷3.2字/s≈4.4s
+ 逗号×2=0.4s + 句号=0.35s → 台词总≈5.2s
+ 动作缓冲（急飞落地+按手臂）=1.5s
+ 情绪留白（恳求的急迫感沉淀）=0.8s
→ 镜时长≈7.5s → 取整为3.5s+3.0s=6.5s拆分两镜
```

### 量化时长控制参数（按镜头类型分档）

AI视频分镜的精确时长控制，不同镜头类型对应不同最优时长区间：

| 镜头类型 | 最优时长 | 说明 |
|----------|---------|------|
| 波动/冲击/信息爆发 | 0.5-1.0s/镜 | 高频动态卡点——能量炸裂/斩击命中/关键信息揭示 |
| 情绪激发/细节展示 | 2.0-3.0s/镜 | 慢节奏留白——微表情/生理反应/道具特写 |
| 对话与互动镜头 | 2.0-3.0s/镜 | 给足角色脸部表情表现空间+台词说完的时间 |
| 特殊运镜（推拉摇移/空间揭示） | 3.0-5.0s/镜 | 情绪高潮的定格——运镜需要时间完成物理运动 |

**行业铁律**：1分钟视频 ≥ 100镜。长视频项目中，1.5分钟（90秒）目标 120-150 镜。

### 快慢组合与情绪留白

- **快速节奏矩阵**：高强度的肢体动作、密集的景观信息 → 短镜（0.5-1.5s）快切
- **慢节奏矩阵**：压迫感、心理防线瓦解 → 延长单镜时长（2.5-4.0s)
- **剪辑点黄金法则**：永远在角色动作结束或台词表达完毕后切——不截断动作和话语

### 声音轨/画面轨分离格式（长镜头专用）

长镜头（≥8s）中音画关系复杂时，可将声音轨和画面轨分离标注：

```
X.Xs 运镜指令，景别

起幅 0-Xs｜[叙事阶段]：
画面轨：[前景] [中景] [背景]
声音轨：环境底噪+动作音效+台词

交锋 X-Xs｜[叙事阶段]：
画面轨：[前景] [中景] [背景]
声音轨：环境底噪+动作音效+台词

余波 X-Xs｜[叙事阶段]：
画面轨：[前景] [中景] [背景]
声音轨：环境底噪+动作音效+台词
```

**使用条件**：仅单镜≥8s且音画复杂时使用。短镜仍用紧凑AI行格式。

## 扩展/修复模式

如果产出稀疏、漏镜、缺时长、关联剧本不完整——从头重新生成完整分镜表。不补局部。

重生成强制规则：
- 纯文本行格式，每镜一行（AI提示词+对白音效+行尾情绪标签）
- 每个大分镜段 ≤ 15s
- 导演定调板（主视觉+角色设定+制作参数+情绪曲线+色彩方案）完整输出
- 每个大分镜段首包含必要资产绑定、场景、站位、承接和负面排除，段尾包含关联剧本+本段小计
- 对白/音效包含每句原文一字不差
- 每镜时长精确到0.1s，累计=总时长
- 每镜AI提示词以运镜指令开头，`@角色名`，30-100字
- 配音状态用逗号 `，` 分隔，不用 `｜`
- 自检八问逐条通过
- 全中文交付，安全脱敏完成
- AI提示词中无 `[X.Xs]` 时长前缀、无 `[原]/[背]/[撮]` 标签、无泛称、无尾部元数据
- 执行辅助物料5栏完整输出
## 最新 Skill 融合输出补充

以下规则来自用户提供的“最新skill出分镜”资料，已按 Seedance 2.0 工作流改写。默认不改变现有纯文本行格式，只在复杂、连续性敏感、动作密集或跨大分镜任务中启用。

### 段首新增可选字段

在原段首声明块中，可按需要追加：

```text
**本幕资产参考图预留区：** 上一段继承主体 / 本段登场主体 / 下一段预告资产 / 场景双轨 / 道具状态 / 全局色彩调色盘
**首帧垫图锚点：** 本段第一帧继承或建立的主体、站位、视线、道具、光线、场景结构、上一段残留动作
**尾帧生成锚点与转场：** 本段最后一帧必须保留的动态状态，以及下一段首帧如何接入
```

字段写法要求：

- `本幕资产参考图预留区` 不等于素材堆砌，必须说明每个素材用途。
- `首帧垫图锚点` 用来防止开头漂移，第一段也要写清画面建立方式。
- `尾帧生成锚点与转场` 必须与段尾 `下段衔接钩子` 一致。

### 14 秒默认输出策略

- 15 秒仍是硬上限。
- 用户没有强制要求满 15 秒时，默认将单个大分镜控制在 14.0 秒左右。
- 14 秒段内通常写 5-7 镜；动作段可更密，文戏段可更少，但每镜只承载一个核心视觉任务。
- 长台词必须按呼吸和标点拆镜，不得压缩语速硬塞进 15 秒。

### 逐镜站位基线

复杂镜头可在单行提示词中加入：

```text
本镜站位基线：继承上一镜A在画左前景、B在画右中景，A面向B，B视线看向A手中道具；
```

使用规则：

- 段首站位基线负责全局；本镜站位基线负责本镜继承和变化。
- 只在人物位置、朝向、视线、层级、道具归属或场景状态有变化时写。
- 被裁出画面的角色或道具若仍在场，必须说明其画外方向或下一次返回方向。

### 人脸与视线锁

每个需要被观众理解的人物状态，提示词中应显式包含：

```text
人脸：三分之二侧脸朝向画右，五官可见；视线：盯住A手中丹药。
```

蒙面、遮挡、背影、画外人物改写为：

```text
人脸：被兜帽遮住，不见任何五官；视线：通过头部角度指向画左目标。
```

### 尾首帧四件套检查

每个大分镜段尾和下一段段首必须同时满足：

- 资产状态：角色、道具、破损、光效、环境变化一致。
- 空间几何：9:16画面内的左右、前后、Z轴、遮挡、180度轴线一致。
- 运动向量：身体、镜头、衣袖、尘烟、光线、飞行物方向连续。
- 转场驱动：动作、道具、视线、声音、遮挡、形状匹配、光源匹配或空间建立硬切清楚。

### 动态尾帧门

段尾最后一镜必须有活性元素，禁止静态海报式收尾。可用：

- 呼吸、眨眼、手指微动、衣袖摆动
- 尘烟、蒸汽、光粒、烛火、雨线、风声延续
- 脚步未落、道具仍在滑动、遮挡物正在掠过
- 台词尾音、环境声或远处声音作为 J-Cut/L-Cut

### 群像和多角色防复制

- 不用“一群人”“几名修士”替代确定人数。
- 需要确定人数时逐一列出角色或临时编号。
- 非重点群众要降级为背景轮廓，重点人物必须用体型、年龄段、发型、服装轮廓、姿势、道具、层级或脸部可见度区分。
