# 表演层次法与音色系统 — 角色塑造与声音设计

## 用户上传音频锁定规则

所有角色音色以用户上传到即梦 Seedance 2.0 的音频为准。分镜与 VoiceID 区域只允许引用 `@角色名的参考音色@` 这类占位符，不得写“年轻男性、成年男性、清亮、懒散、低沉、贪念时声音收窄发亮”等文字音色画像。文字提示只负责当前台词的表演状态，例如停顿、重音、呼吸、语速、尾音变化；音色本身由上传音频锁定。

本文档为 SKILL.md 第三步后半段的专家知识库，包含表演层次法（影帝影后级角色塑造）、角色专属光影（2D国漫赛璐璐光照）和VoiceID音色系统（Seedance音频参考模式）。SKILL.md 中用一行摘要引用本文件。

## 表演层次法（影帝影后级角色塑造）

角色的每个情绪节点需要**三层表演递进**，缺一层=表演扁平：

| 层次 | 控制维度 | 示例（愤怒） |
|------|----------|------------|
| **表层·肢体** | 大肌肉群：姿势/步态/手势/头部角度/脊椎曲度 | 双肩耸起、脊椎前倾、五指抠入掌心、下颚前伸 |
| **中层·面部** | 五官微运动：眉/眼/鼻/嘴/颧肌的独立控制 | 眉间肌下拉、上眼睑抬高露眼白、鼻翼扩张、上唇微翻露齿 |
| **深层·生理** | 自主神经：瞳孔/呼吸/血管/汗腺/喉结/颈部肌腱 | 瞳孔收缩、呼吸变浅变快、颈动脉搏动可见、鬓角渗汗 |

### 表演铁律（3条）

1. **三层同框**：每个情绪节点至少在一个镜头中同时展现三层——不能只有肢体没有生理反应
2. **表演正反馈循环**：A的表演给B反应契机→B的反应再反哺A的下一个动作。设计镜头时检查：B看完A的表演后，B的下一镜反应是否承接了A给的情绪刺激？
3. **抑制优先于爆发**：最高级的表演是"想哭但忍住的0.3秒嘴唇颤抖"而非"嚎啕大哭"。每个情绪高潮前加一镜**抑制镜头**——角色试图控制但身体出卖了自己（嘴角强行上扬但眼眶已红）

## VoiceID表（Seedance音频参考模式）

分镜输出前，为每个说话角色准备音频参考文件并上传至即梦Seedance 2.0。音频文件作为音色参考，`@角色名的参考音色@` 为音色占位符指向对应音频。

### 工作流

1. 为每个说话角色录制/准备一段音频（对话/独白/笑声/呼吸，时长2-5s），上传至Seedance
2. 在角色设定卡或大分镜段首标注 `声音资产绑定：角色台词/角色（VO）只使用@角色名的参考音色@`，指向该音频
3. 大分镜段首的「声音资产绑定」行列出本段所有有台词角色的角色-音频映射，并写明禁止互换、混用、随机音色
4. 每镜台词前用短句强制绑定当前说话人：`声源锁：角色=@角色名的参考音色@；`
5. 内心独白/旁白必须额外锁口型：`口型锁：角色闭口不张嘴，无口型对白，声音为内心旁白；`

### 每镜配音指令格式

普通台词：`声源锁：角色=@角色名的参考音色@；台词：角色："原文" 配音状态描述，用逗号分隔`

内心独白：`口型锁：角色闭口不张嘴，无口型对白，声音为内心旁白；声源锁：角色=@角色名的参考音色@；台词：角色（VO）："原文" 配音状态描述，用逗号分隔`

配音状态仅描述本镜偏移量——**不要只写情绪名字，要写这个情绪发生时人的声音会怎么变**。

## AI配音六维发声拆解体系

AI 不一定懂"紧张""难过""愤怒"，但它更容易理解：声音大小、语速快慢、呼吸状态、停顿位置、重音位置、尾音变化。

### 六维拆解公式

```
配音指令 = 声音大小 + 语速快慢 + 呼吸状态 + 停顿位置 + 重音位置 + 尾音变化
```

### 1. 声音大小
| 变化 | 适合情绪 |
|------|---------|
| 声音放轻 | 难过、安慰、害怕、试探 |
| 声音压低 | 愤怒、警告、怀疑、反派感 |
| 声音变亮 | 开心、惊喜、兴奋 |
| 声音变弱 | 委屈、失望、舍不得 |
| 声音变稳 | 安慰、坚定、保护感 |

### 2. 语速快慢
| 变化 | 适合情绪 |
|------|---------|
| 语速变慢 | 思考、难过、压迫、怀疑、安慰 |
| 语速变快 | 开心、慌张、激动、惊喜 |
| 一字一句咬字变重 | 愤怒、质问、警告、失望 |
| 断断续续 | 害怕、哭腔、紧张、舍不得 |

### 3. 呼吸状态
| 变化 | 适合情绪 |
|------|---------|
| 呼吸变浅 | 紧张、害怕、被点名 |
| 呼吸变重 | 愤怒、压抑、快失控 |
| 带气声 | 安慰、温柔、亲密、难过 |
| 气息不稳 | 哭腔、委屈、舍不得 |
| 先吸一口气再说 | 忍了很久才开口 |

### 4. 停顿位置
| 位置 | 效果 |
|------|------|
| 句子前停顿 | 像在犹豫要不要说 |
| 中间停顿 | 像情绪卡住，或在思考 |
| 重点词前停顿 | 制造压迫感和期待感 |
| 省略号 | 难过、害怕、犹豫、舍不得 |
| 逗号 | 思考、撒娇、试探、安慰 |

### 5. 重音位置
| 位置 | 效果 |
|------|------|
| 后半句加重 | 警告、质问、情绪爆发 |
| 某个关键词加重 | 让观众听出重点 |
| 最后几个字加重 | 压迫感、怀疑感、愤怒感 |

### 6. 尾音变化
| 变化 | 适合情绪 |
|------|---------|
| 尾音上扬 | 疑问、试探、撒娇、怀疑 |
| 尾音下压 | 警告、反派、冷淡、掌控感 |
| 尾音发虚 | 难过、委屈、舍不得、害怕 |
| 尾音放柔 | 安慰、温柔、亲近 |
| 尾音拉长 | 撒娇、轻快、开心、暧昧 |
| 尾音稳稳落下 | 坚定、安抚、安全感 |

## 标点符号控制论

标点符号直接控制AI的停顿和语气。在台词原文中合理使用标点，比在配音指令中写"语速放慢"更有效：

| 符号 | 效果 | 适合情绪 |
|------|------|---------|
| `，` 逗号 | 制造真实停顿，让语气不一直往前冲 | 思考、犹豫、试探、撒娇 |
| `。` 句号 | 让语气收住、平稳落地 | 平静、冷淡、克制、陈述 |
| `！` 感叹号 | 加强情绪和重音 | 警告、愤怒、兴奋、强调 |
| `？` 问号 | 尾音自然上扬 | 疑问、试探、不确定、怀疑 |
| `？！` 问号+感叹号 | 质问+爆发 | 愤怒、震惊、不敢相信 |
| `……` 省略号 | 情绪卡住的停顿 | 难过、委屈、害怕、紧张、犹豫 |
| `～` 波浪号 | 尾音轻快或柔软 | 开心、撒娇、轻松、温柔 |
| `——` 破折号 | 突然停住、情绪转折 | 震惊、迟疑、强调、转念 |

**示例**：
- 写作"没关系……反正我也习惯了。"而非"没关系反正我也习惯了"——省略号让AI在"没关系"后自然停顿
- 写作"你还敢说……不是你？！"而非"你还敢说不是你"——省略号积压怒气，？！释放质问+爆发

## 12种情绪通用配音模板

### 1. 开心/兴奋
```
声音变亮，语速稍微变快，尾音自然上扬，带笑意地说：【台词】
```
台词标点：`～` + `！` 组合

### 2. 难过/委屈
```
声音很轻，语速慢下来，中间停顿明显，尾音发虚，像强忍情绪地说：【台词】
```
台词标点：`……` 制造情绪卡顿，`。` 收尾

### 3. 愤怒/质问
```
声音压低，呼吸变重，语速放慢，咬字变重，像压着怒气一字一句地说：【台词】
```
台词标点：`……` 积压怒气，`？！` 释放质问+爆发

### 4. 警告/压迫感
```
声音压低，语速放慢，语气冷，后半句加重，尾音不要软掉，像最后一次警告地说：【台词】
```
台词标点：`，` 前半句压住，`！` 后半句加重

### 5. 紧张/害怕
```
声音放轻，呼吸变浅，语速慢下来，中间停顿明显，尾音发虚地说：【台词】
```
台词标点：`……` 制造卡顿，`？` 不确定感

### 6. 怀疑/试探
```
声音压低一点，语速放慢，中间停顿像在观察对方反应，最后几个字微微上扬地说：【台词】
```
台词标点：`……` 制造观察停顿，`？` 尾音上扬

### 7. 安慰/温柔坚定
```
声音放轻，语速慢下来，前半句尾音放柔，后半句声音更稳、尾音稳稳落下地说：【台词】
```
台词标点：`，` 前后分层，`。` 稳稳收尾

### 8. 思考/犹豫
```
语速放慢，中间有停顿，声音不急着往前冲，像边想边说：【台词】
```
台词标点：`，` 多处思维停顿，`。` 平稳收尾

### 9. 失望/心寒
```
声音变轻，语速变慢，情绪不爆发，尾音下沉，像已经不想争了：【台词】
```
台词标点：`。` 冷淡收尾，不用`！`不用`？`

### 10. 反派/掌控感
```
声音压低，语速很慢，停顿明显，尾音微微下压，带一点轻蔑冷笑：【台词】
```
台词标点：`……` 制造压迫停顿，`。` 冷静收尾

### 11. 撒娇/亲近
```
声音放软，语速稍慢，中间有小停顿，尾音轻轻拉长：【台词】
```
台词标点：`，` 制造小停顿，`～` 尾音拉长变软

### 12. 惊讶/不敢相信
```
声音先轻后高，先停顿一下像没反应过来，然后语速稍微变快，尾音上扬：【台词】
```
台词标点：`……` 先停顿，`？！` 不可置信

## 万能配音公式

```
场景 + 人物状态 + 发声方式 + 台词
```

**不要写**：她紧张地说。
**要写**：她声音放轻，呼吸变浅，语速慢下来，中间停顿一下，尾音有点发虚地说：【台词】

**不要写**：他愤怒地说。
**要写**：他声音压低，呼吸变重，咬字变重，一字一句地说：【台词】

核心原则：**用物理化的声音变化代替抽象的情绪标签。AI 更容易理解"语速慢下来+尾音发虚"而非"难过"。**
