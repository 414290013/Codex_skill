# 导演讲戏方法论

导演不是把剧本复述一遍，而是像给演员讲戏一样，把脑海中的画面完整描述出来。讲戏是整套系统的创意起点——下游分镜师不需要理解剧本的叙事意图，只需把导演描述的画面忠实地转写成 Seedance 2.0 提示词。

## 情绪单元拆解

导演不以物理场景为单位拆解剧本，而以"情绪单元"为单位：

```
一场争吵戏 → "压抑蓄力→爆发冲突→余怒未消"三个情绪单元
一场打斗戏 → "试探→交锋→升级→绝境→决胜"五个情绪单元
一场对话戏 → "建立→质疑→揭示→沉淀"四个情绪单元
```

每个情绪单元定义该段的视觉基调（光线/色彩/景别节奏/运镜速度）。情绪转变时触发视觉基调切换。

### 视觉关键词库

将抽象情绪转化为AI可理解的视觉关键词。导演在定调时必须为每个情绪单元指定对应的视觉关键词组合：

| 情绪 | 视觉关键词 | 光影/色彩 |
|------|-----------|----------|
| 压抑 | 低角度仰拍、头顶留白压缩、阴影分割脸 | 冷蓝调、低饱和、硬光 |
| 浪漫 | 柔光、暖黄、逆光、浅景深、飘动发丝 | 暖橙调、高光柔化 |
| 危机 | 荷兰角（倾斜构图）、快速推镜、红色点缀 | 失衡构图、暗红脉冲光 |
| 恐惧 | 手持晃动、极近距离、呼吸感 | 暗调、冷色、局部高光 |
| 愤怒 | 快切、冲击变焦、倾斜 | 高反差、红/橙脉冲 |
| 悲伤 | 缓拉、留白多、固定机位 | 冷蓝/冷绿、低饱和、柔光 |
| 希望 | 缓推、逆光、镜头扬起 | 暖金、丁达尔光、高饱和 |
| 绝望 | 俯拍、拉远、角色缩小 | 灰调、去饱和、冷白 |
| 神秘 | 前景遮挡多、局部光、慢摇 | 紫/蓝、暗调、点光源 |
| 宁静 | 固定、对称构图、长镜头 | 自然光、中性色调、柔光 |

## 讲戏核心原则

### 叙述规则

1. **用具体物理动作，不用抽象概念**：不写"她很悲伤"，写"她低头咬唇，指尖颤抖着攥紧茶杯，眼眶泛红但没有泪"
2. **连续动作链**：不写孤立姿态——"她起身→走到窗前→手掌贴上冰冷玻璃→呼出的气在玻璃上凝成一小圈白雾"
3. **镜头运动有明确方向**："镜头从中景缓缓推到特写，捕捉路灯在她脸颊上的反光"
4. **光影描述具体到方向/色温/强度**："冷白路灯光从左侧45度斜切她的脸，色温4000K，另半边隐没在暗处"
5. **所有维度融合为自然叙述**：画面内容、人物动作、台词声音、镜头运动、光影氛围——不分条列举，用自然流畅的文字融在一起

### 禁止事项

- 禁止写成 bullet-point 列表（"画面：xxx / 机位：xxx / 动作：xxx"）
- 禁止抽象概括（"气氛紧张""画面唯美"）
- 禁止跳过动作链中的中间步骤
- 禁止出现"导演认为""建议使用"等元描述

## 讲戏输出模板

### 全局定调（定调板）

```
## 全局定调

### 主视觉定调
场景：[地点+时间+天气]
光线：[主光源方向+色温+氛围词]
角色：[全部出场角色在此场景中的空间位置与姿态]
色彩基调：[主色调+辅色调+点缀色]
情绪基底：[本片核心情绪的视觉化表达]

### 情绪曲线
[ASCII曲线图 + 关键时间节点标注]

### 色彩方案
主色：[色块名称 — 用途]
点缀色：[色块名称 — 功能性强调]
参考：高饱和色彩，光影对比明确，阴影用冷色补暖色光源
```

### 讲戏本（核心产物）

```
## 讲戏本 — [剧本标题]

### 段1：标题（约XXs）

[自然流畅的叙述文本，融合画面/动作/台词/运镜/光影]
[用"---"分隔段落]
[用连续动作链，不分条]

关键叙述要素（每段讲戏中自然融入）：
- 画面内容：人物位置、姿态、表情
- 动作链：连续物理动作序列
- 台词/声音：谁说什么、语气、音量、回声
- 镜头运动：推拉摇移跟升降，方向+速度
- 光影氛围：光源方向+色温+强度+投射效果
```

### 人物清单

```
## 人物清单

### 主角组
- [角色名]：[简要特征，2-3个核心性格关键词，专属光影类型；声音方向只作为后期备注，不进入 Seedance 提示词]

### 反派/配角组
- [角色名]：[同上]

提取规则：
- 剧本中每个有名称的角色 = 一个人物条目
- 提取核心性格关键词（2-4个）
- 如需记录声音方向，只放后期备注；不得写入 Seedance 正式提示词区
- 标注专属光影类型（暖黄/冷蓝/霓虹粉紫/剪影逆光）
```

### 角色出场公式（四维塑造）

每个重要角色首次出场必须覆盖四个维度——不为叙述，只为使用：

**维度一：专属人设镜头**
- 角色的第一个镜头必须展示其核心性格——通过一个具体动作而非静态pose
- 刘孝天=痴迷地研究车窗按键（好奇→人设建立），而非站在车旁
- 萧子琪=从后视镜瞥一眼后座（质疑→性格外化），而非坐在车里

**维度二：标志性微表情/微动作**
- 每个角色必须有1-2个标志性微表情——即使无台词也能传递情绪
- 好奇时双眼瞪大瞳孔收缩 / 质疑时微眯双眼眼角紧绷 / 轻蔑时嘴角微扬单侧
- 标志性微动作作为该角色的视觉签名，全片重复出现3-5次强化记忆

**维度三：他人反应链（旁观者效应）**
- 角色的气场必须通过其他角色的反应来印证——不可自说自话
- 强者出场→旁人屏息/水杯滑落/后退半步
- 反派逼近→路人低头避让/鸟兽惊飞/烛火摇曳

**维度四：环境失衡暗示（专为反派/强气场角色设计）**
- 角色出现伴随物理环境变化：光线骤然暗淡/温度骤降/空气凝固感
- 压迫感来源：保持角色极其镇定，同时周围环境开始扭曲——"他什么都没做，但房间里所有人都不敢呼吸"
- 环境失衡在角色退场后持续1-2镜作为余波

### 增补镜头三原则（提升画面丰富度30%-50%）

剧本只写核心动作，导演必须增补以下三类镜头才能达到工业级水准：

**原则一：前置铺垫（未见其人，先见其势）**
- 主角登场前1-2镜展示其影响力痕迹——不直接拍人
- 公式：视觉焦点（如地面震颤/物品晃动/影子拉长）+ 旁观者反应（抬头/惊愕）+ 主角登场
- 示例：主角出场前→桌上水杯水面震颤→配角抬头看向画外→主角从画面左侧入画

**原则二：全息反应链**
- 任何重大事件发生后，必须给至少3个反应镜头——
- ①直接受影响者的物理反馈（身体反应）→
- ②旁观者的微表情反应（他人印证）→
- ③环境/道具的连锁反应（情绪外化）
- 节奏：先给被冲击者的反应，再给旁观者的反应，最后给环境的反应

**原则三：环境叙述（情绪外化）**
- 在情绪高潮时刻，用环境元素替代直白的面部表情
- 愤怒→不是拍脸，拍紧攥的拳头+指节泛白+手中物品碎裂
- 悲伤→不是拍泪，拍雨水顺着窗户滑落+窗内人模糊身影
- 孤独→不是拍表情，拍角色小小身影在大空间中的位置

### 角色专属光影色彩系统

在生成镜头时，严格绑定以下光影规则——同一角色全片光线类型不变：

| 角色类型 | 光线类型 | 色彩方案 | 心理效果 |
|----------|---------|---------|----------|
| 热血主角 | 暖色调/鹅黄 | 顺光+柔光+逆光金边 | 观众感到温暖/共情/支持 |
| 高冷/理性角色 | 冷色调/蓝调 | 侧光+硬光+轮廓冷白 | 距离感/客观/难以接近 |
| 温柔/治愈角色 | 暖粉/紫环绕 | 柔光+逆光+浅景深 | 亲近/安全/放松 |
| 反派/隐藏BOSS | 剪影/逆光/暗调 | 无正面光+冷绿/暗红点缀 | 压迫/神秘/不可知 |
| 智者/导师 | 顶光+金色 | 丁达尔光+暖金+柔和 | 智慧/权威/希望 |

**铁律**：
- 角色专属光影一旦在定调板中确定，全片不可变更——观众通过光线颜色识别角色情绪属性
- 同一场景中两个不同光影类型的角色相遇时，光线以剧情主导方的光影类型为准
- 角色死亡/黑化/重大转变时，光影类型可以突变——这是全片唯一的例外

### 造型曲线设计

导演必须在定调板中绘制全片情绪强度曲线，确保节奏有峰值和谷值：

```
横轴 = 时间（秒），纵轴 = 情绪强度（1-10分）

10 ┤                              ╭─ 最终高潮
 8 ┤          ╭─ 第一波冲突      │
 6 ┤    ╭─ 钩子                  │        ╭─ 余波
 4 ┤   ╱    ╲                   ╱╲      ╱
 2 ┤  ╱      ╲                 ╱  ╲    ╱
 0 ┤─╱───────╲───────────────╱────╲──╱──
    0s       30s       60s       90s    120s
```

**造型曲线铁律**：
- 高压动态戏/冲突戏：波峰必须密集，情绪强度拉满（7-10分），提示词中运镜速度和景别对比最大化
- 文戏/角色内心戏：波谷必须深沉，情绪强度低（2-4分），让观众呼吸和消化
- **每一轮波峰后必须有一轮波谷**——连续高潮=麻木。波谷设计原则：让观众期待下一个波峰
- 每个大分镜段在段首标注情绪强度值：`**情绪强度：X/10**`

```
## 大分镜X：标题（起始时间-结束时间）
**情绪强度：8/10** — 本段处于全片情绪曲线的第2个波峰
```

### 场景清单

```
## 场景清单

- [场景名]：[地点类型，空间尺度（长×宽×高），光线氛围，关键道具/家具，天气/时间]
- [同上...]

提取规则：
- 每个地点/时间/天气的独立组合 = 一个场景条目
- 标注空间尺度（如"五丈见方·高两丈"）
- 标注关键家具和道具（案桌/屏风/栏杆/台阶等阻挡物）
- 同一场景的不同时间/天气 = 不同条目
```

## 两步审核流程

导演在每个阶段产出完成后执行审核。审核时：
1. 加载对应的 review skill 文件
2. 逐项评分（1-10分）
3. 平均分 < 8 或任一单项 < 6 = FAIL
4. FAIL 时输出：评分表 + 具体不达标项 + 修改指令

### 阶段一审：讲戏质量
加载 `references/review-script-analysis.md`
- 画面感 / 动作链完整性 / 节拍密度 / 光影具体性 / 情绪传达 / 剧本覆盖 / 运镜合理性 / 空间连续性

### 阶段二审：服化道设计
加载 `references/review-art-direction.md`
- 人物对照 / 造型准确性 / 风格一致性 / 描述精确性 / 格式正确性 / 出图可行性 / 素材管理

### 阶段三审：Seedance 提示词
加载 `references/review-seedance-prompt.md`
- 讲戏忠实度 / 动作可执行性 / 运镜可实现性 / @引用正确性 / 节拍密度 / 安全区 / 音频设计 / 敏感词

### 合规审核（跨阶段）
加载 `references/review-compliance.md`
- 通用红线 + 即梦专用 + Gemini 专用
