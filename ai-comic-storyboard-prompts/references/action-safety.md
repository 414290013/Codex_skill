# Action Safety Rules and Rewrite Cases

## Knowledge Categories

- `禁用词扫描`: 血腥、具体身体部位、真实武器、粗口、低俗/强迫/羞辱语义。
- `安全替换`: 战损痕迹、深色烟雾、冲击光效、压制、倒地静止、能量余波。
- `动作改写案例`: 都市街斗、Boss战、仙侠法术战、群战围攻、都市制服。
- `提示词自检`: 图像提示词、英文描述、声音列和关联剧本的边界分离。


# AI 视频平台安全审核规则

用途：动作、悬疑、打斗、修仙冲突等分镜提示词容易触发平台内容过滤。本知识库提供通用禁用词清单、替代词库、自检扫描规则和动作安全改写案例。图像提示词优先使用风格化冲击、能量粒子、战损痕迹、倒地静止、烟雾消散等安全表达，避免写实血腥、具体身体伤害和低俗敏感描述。

## §A 五大类禁用词清单

### A.1 血腥类（🔴 最高危，必拦）

| 禁用词 | 替代词 |
|--------|--------|
| 血、鲜血、淤血、血珠、血滴、血沫、血浆、血雾 | 汗水、水花、气浪、雨水、水珠、尘雾 |
| 血痕、血线、血迹、血渍、血污 | 擦痕、污渍、水渍、灰痕 |
| 血飞、血溅、血洒、血飞溅、血花四溅 | 汗飞、水花飞溅、气浪翻卷、尘土飞扬 |
| 血染、血洗、染红、血泊、血洼 | 雨水浸透、积水扩散、水面同心圆 |
| 吐血、咯血、口吐鲜血、七窍流血 | 唇角一抹擦痕、喘息、咳嗽 |
| 内出血、脑出血、动脉喷涌 | 完全不写（改描写气息紊乱） |

**铁律**：打戏任何场景**禁用"血"字及其所有组合**，无例外。

### A.2 具体身体部位攻击（🔴 高危）

| 禁用部位名 | 替代广义词 |
|-----------|----------|
| 咽喉、喉结、颈动脉、锁骨凹陷 | 肩、上身、胸口 |
| 太阳穴、后脑、颅底、眉骨、眼眶 | 头侧、上盘 |
| 下颌、下巴、下颚、颌骨 | 面颊、下盘 |
| 鼻梁、鼻骨 | 面部中心 |
| 心脏、心口、肋骨、肝区 | 胸口、躯干中段 |
| 肾区、腰椎、脊椎、尾椎 | 后腰、背部 |
| 裆部、膝盖反折、膝弯、内踝 | 下盘（完全不写反折） |

**铁律**：不写"攻击点精确到某个易受伤部位"。打戏描写只用"胸口 / 上身 / 肩 / 上盘 / 下盘 / 躯干"等**广义词**。

### A.3 骨骼 / 器官损伤（🔴 高危）

| 禁用词 | 替代写法 |
|--------|---------|
| 骨折、断骨、骨裂、粉碎性骨折 | 踉跄、失衡、站不稳 |
| 脱臼、关节错位、卸下颌 | 踉跄后退、扶墙喘息 |
| 断牙、牙齿飞出、门牙打掉 | 唇角一抹擦痕（不写牙） |
| 眼白上翻、瞳孔涣散、视线涣散 | 眼神一滞、眼神空白（不写眼白） |
| 昏迷、昏过去、失去意识 | 单膝跪地、靠墙坐下、不再动作 |
| 抽搐、痉挛、颤抖不止 | 喘息、停顿 |
| 内伤、五脏俱损、七经八脉俱断 | 气息紊乱、呼吸急促 |

**铁律**：不写任何生理损伤细节。人物击中后只写外部状态（踉跄/失衡/后退/撑地）。

### A.4 真实武器（🟡 中-高危，按场景分级）

#### 🔴 必禁（现代真实武器）
- 枪类：手枪、步枪、冲锋枪、霰弹枪、狙击枪、AK、M4、子弹、弹匣、枪口、扳机
- 刀类：砍刀、菜刀、开山刀、匕首、军刺、尖刀、水果刀
- 棍棒类：钢管、铁棍、铁链、铁棒、铁锤、斧头、伸缩警棍
- 改制武器：土枪、土炸药、爆炸物

#### 🟢 允许（道具化"非武器"）
- 港片黑帮片常用：**折凳**、**木棒**、**短棒**、**雨伞**、**卷报纸**、**拖把柄**、**扫帚**
- 工具类：**卷尺**、**钳子**、**扳手**（具体场景斟酌）

#### 🟢 武侠 / 仙侠语境允许（幻想层加持）
- 冷兵器：**剑**、**刀**（武侠意义）、**枪**（武侠意义）、**棍**、**鞭**、**扇**
- 法器：**法剑**、**符箓**、**法印**、**飞剑**、**法宝**

**铁律**：
- 现代写实题材（`photorealistic_human`）→ 武器必须道具化，禁真实军警武器
- 武侠 / 仙侠（`stylized_2d` / `cg_render` 的武侠子风格）→ 冷兵器 OK，但避免"砍向 + 具体部位 + 血"三连
- 真实军警场景（抗战剧 / 警匪）→ 审核最严，建议不走 AI 视频，或整体改武侠化

### A.5 死亡与极端暴力（🔴 最高危）

| 禁用词 | 替代写法 |
|--------|---------|
| 死、死亡、死去、断气、咽气、殒命、毙命、丧命 | 不再动作、静止、闭眼、失去气息（最后一个仍危险，尽量不用） |
| 杀、击毙、格杀、诛杀、处决、处死、枪决 | 制服、放倒、击退、震退 |
| 尸体、遗体、横尸、陈尸、碎尸、残尸、全尸 | 完全不写（场景切走）|
| 虐杀、屠杀、灭门、血洗、大开杀戒、赶尽杀绝 | 交锋结束、局势已定 |
| 自尽、自杀、自戕、自焚 | 完全不写（或场景改为"决意离开"）|
| 割喉、剜心、剖腹、挖眼 | 完全不写，此类动作禁止出现 |

**铁律**：打戏叙事改为**"制服 / 突围 / 击退"**，不走**"击杀 / 屠杀"**。即使剧情需要死亡，用**外部叙事旁白**表达（"他再也没有站起来"），不在视频分镜里具体描写。

### A.6 粗口与敏感言论（🟡 中危）

| 禁用词 | 替代写法 |
|--------|---------|
| 操、操你、操他、草、他妈的、妈的、TM、卧槽、靠 | 唔、哼、啧、呵、"——!" |
| 滚你妈的、去你妈的、你妈逼 | 滚、让开、走开 |
| 鸡巴、婊子、贱人、贱货、狗东西 | 改中性描述（"那家伙"）|
| 去死吧、找死、活腻了 | "别动"、"收手" |

#### 涉政 / 涉教敏感（🔴 必禁）
- 任何涉及现实政治人物、党派、国家冲突、宗教对立、民族冲突的直接描写
- 建议走纯虚构架空叙事，避免真实地名、国名、机构名

## §B 写作策略（按 realismLevel 差异化）

不同 realismLevel 的审核严格度不同，打戏写作策略也不同：

| realismLevel | 审核严格度 | 打戏写作策略 |
|--------------|-----------|------------|
| `stylized_2d`（2D 漫剧） | 🟢 最宽松 | 可保留水墨动态 / 剑气迸发 / 定格冲击，但仍避免粗口和精确部位。血可换成"墨色飞溅"（作为水墨特效）|
| `stylized_3d`（3D CG 动画） | 🟢 较宽松 | 卡通化降低真实感，允许夸张动作和非真实武器。仍避免真实武器+真实部位攻击 |
| `cg_render`（次世代仿真人 CG） | 🟡 中等 | 因为质感接近真人，审核偏严。走武侠 / 奇幻叙事（气劲/剑气）最稳；走现代写实（黑悟空级仿真人但现代街斗）高危 |
| `photorealistic_human`（真人实拍） | 🔴 最严 | **所有真人肢体冲突都是高危**。必须走"突围 / 制服"叙事，禁真实武器、精确部位、血腥、粗口、死伤 |

**核心经验**：**越是写实题材，审核越严**。追求"超级激烈打斗"的最佳选择是**武侠 / 仙侠 / 魂系奇幻**（有虚构层加持），而不是现代真人街斗。

## §C 推荐替代叙事模式

### C.1 港片黑帮 / 现代动作片合规写法

写**突围**不写**击杀**：
- ❌ 男主手起刀落，斩杀三名反派，血溅三尺
- ✅ 男主贴身寸劲推出，三名对手踉跄后退，手中折凳脱手落在积水里

武器改**道具化**：
- ❌ 带头反派抽出砍刀砍向男主
- ✅ 带头反派挥起折凳砸向男主

目标改**制服**：
- ❌ 男主一记肘击打碎对方下颌，对方昏死过去
- ✅ 男主一记寸劲推出，对方上身一震踉跄后退撞墙，单膝撑地喘息

### C.2 武侠 / 仙侠合规写法

走**招式 + 环境反馈**，不走**具体伤害细节**：
- ✅ 『白鹤亮翅』剑尖划弧，剑气带起三片竹叶；对方横剑一架，两人各退三步
- ❌ 剑尖刺入对方肩胛骨，血从伤口喷涌而出

环境震荡体现威力，不直接描写身体伤害：
- ✅ 一掌拍出，脚下青砖寸裂蛛网纹，周围烛火齐齐扑灭；对方倒飞数丈撞在廊柱上
- ❌ 一掌拍在对方胸口，肋骨应声断裂，当场吐血身亡

### C.3 魂系黑暗奇幻合规写法

用**幻化 / 气劲 / 符文**替代真实血腥：
- ✅ 对方身形开始光点离散，化作一缕黑烟消散
- ✅ 致命一击命中瞬间，对方身上浮现裂纹光效然后碎成光尘
- ❌ 对方内脏被击碎，倒地痉挛而亡

# 平台安全动作改写对照案例页

更新日期：2026-04-23
适用范围：把高风险动作写法和安全改写写法并排对照，方便直接照着改

## 1. 这份页解决什么

最常见的问题不是“不知道哪些词危险”，而是：

1. 知道危险词，但一删就把戏删空了
2. 只会机械替换，改完以后没有压制关系
3. 分镜、声音、动作没有一起改，结果前后不一致

这份页专门做一件事：

`把“原危险写法 -> 为什么会撞线 -> 怎么改还能保住爽点”并排写清。`

## 2. 用法

每次碰到高风险动作段，固定这样走：

1. 先找到最像的案例
2. 抄它的 `保留什么`
3. 再抄它的 `替换成什么`
4. 最后把镜头和声音也一起改

一句话：

`安全改写不是删动作，而是换表达载体。`

## 3. 对照案例 A · 都市街斗：刀刺改成制服突围

### 原危险写法

```text
反派掏出匕首猛刺男主腹部，男主侧身躲开后反手夺刀，一刀划开对方手臂，鲜血立刻溅到墙上。对方捂着伤口惨叫倒地。
```

### 为什么高危

- 有真实武器：`匕首`
- 有具体伤害：`刺腹部`
- 有血腥细节：`鲜血溅到墙上`
- 有生理痛感强化：`惨叫倒地`

### 安全改写版

```text
反派抄起短棒猛冲上来，直逼男主中线。男主侧身让过，顺势扣住对方手腕把短棒击偏，借墙面反推把对方震退半步。对方撞上墙后单膝撑地，呼吸发乱，男主已经拉开身位冲出走廊。
```

### 这一改保住了什么

- 压制关系还在：
  反派先逼，男主后反制
- 爽点还在：
  男主不是被动挨打，而是夺回主动
- 结果还清楚：
  男主成功突围

### 镜头 / 声音也要一起改

- 镜头：
  中近景跟拍，重点给 `击偏 -> 反推 -> 拉开身位`
- 声音：
  短棒破风、鞋底急停、墙面闷撞、喘息

## 4. 对照案例 B · Boss 战：断骨改成压制链 + 结果停顿

### 原危险写法

```text
Boss 一掌打在主角胸骨上，骨头断裂声清晰可闻。主角口吐鲜血跪倒，Boss 再一脚踩住他的喉咙，逼得他几乎断气。
```

### 为什么高危

- 具体身体部位：`胸骨`、`喉咙`
- 生理损伤：`骨头断裂`
- 血腥：`口吐鲜血`
- 死亡趋近描写：`几乎断气`

### 安全改写版

```text
Boss 一掌震在主角上身，掌风压得主角连退数步，背后石柱都跟着一震。主角单膝撑地，呼吸紊乱，视线短暂停住。Boss 已经逼到近前，脚步压住中线，主角却在下一拍借地起身，把局势硬生生翻回来。
```

### 这一改保住了什么

- Boss 的强压迫还在
- 主角的低谷还在
- 翻盘前的停顿更清楚了

### 镜头 / 声音也要一起改

- 镜头：
  低角度给 Boss 压迫，主角失势时给短停顿
- 声音：
  掌风低频、石柱闷震、半拍静默、主角再起时给爆点

## 5. 对照案例 C · 仙侠法术战：穿心改成护体碎裂

### 原危险写法

```text
飞剑直接刺穿对方心口，剑尖从背后透出，血线顺着衣襟流下。她低头看着胸前伤口，身体发颤，最后无力倒下。
```

### 为什么高危

- 具体部位：`心口`
- 贯穿伤：`刺穿`、`透出`
- 血腥：`血线流下`
- 生理死亡过程：`发颤`、`倒下`

### 安全改写版

```text
飞剑贯空而至，对方胸前护体灵光骤然裂开，碎光向四周散开。她被这股冲击震得倒退数步，衣摆和长发同时掀起，脚下法阵都出现一圈震纹。她抬眼时气息已乱，手中法诀也被硬生生打散。
```

### 这一改保住了什么

- 命中感还在
- 胜负变化还在
- 法术压制更像仙侠

### 镜头 / 声音也要一起改

- 镜头：
  远景先交代两人站位，命中时切中景看护体碎裂
- 声音：
  飞剑尖啸、护体碎裂脆响、法阵嗡鸣中断

## 6. 对照案例 D · 群战围攻：砍杀改成撞散缺口

### 原危险写法

```text
主角挥刀一路砍翻围上来的敌人，刀锋划过时血雾四起，几个人接连倒在台阶上不再动弹。
```

### 为什么高危

- 高危武器：`刀`
- 血腥：`血雾`
- 死亡结果：`不再动弹`

### 安全改写版

```text
主角抄起木棒冲向台阶缺口，先挡开左侧来人的动作线，再借栏杆把正前方两人撞散。后排人群一乱，他顺势从空出的半步身位突过去，木栏和脚边杂物被带得四处翻响，场面一下子散开。
```

### 这一改保住了什么

- 围攻压力还在
- 主角破阵的爽点还在
- 群战的空间关系更清楚

### 镜头 / 声音也要一起改

- 镜头：
  先给站位，再跟主角切缺口，不要一上来就乱切
- 声音：
  木棒挡击、栏杆震动、杂物翻倒、密集脚步

## 7. 对照案例 E · 都市制服：掰关节改成控制收束

### 原危险写法

```text
男主拧住对方手臂猛力反折，只听一声脆响，对方手肘当场脱臼，痛得惨叫不止。
```

### 为什么高危

- 关节损伤：`反折`、`脱臼`
- 痛感强化：`惨叫不止`

### 安全改写版

```text
男主贴身切进中线，扣住对方手腕向下压，顺势把整个人按到墙边。对方重心一乱，另一只手也来不及抬起，只能被迫停住动作。男主借势把人彻底控制在墙面和臂弯之间，局势瞬间收住。
```

### 这一改保住了什么

- 控制感还在
- 近身压迫还在
- “他赢了” 仍然非常明确

### 镜头 / 声音也要一起改

- 镜头：
  中近景，贴身，不要过分拆手部细节
- 声音：
  衣料摩擦、鞋底急停、墙面轻撞、短促喘息

## 8. 平台安全改写最稳的 5 个原则

1. 保 `压制关系` 不保 `伤害细节`
2. 保 `命中点` 不保 `伤口`
3. 保 `胜负变化` 不保 `死亡过程`
4. 保 `环境反馈` 不保 `生理反馈`
5. 保 `镜头节奏` 不保 `危险词`
