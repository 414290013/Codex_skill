# 音效设计体系 — 动画声音全流程

音效设计师负责全片听觉体验。在分镜阶段，每镜必须标注音效——让即梦Seedance理解该镜的声音环境。

---

## 声音五层模型

每镜的声音由下至上叠五层，缺层则声音扁平：

```
五层 [台词/VO]     ← 人声：对话/独白/旁白
四层 [音乐/BGM]    ← 配乐：旋律/和声/节奏
三层 [冲击音效]    ← 重音：撞击/爆炸/破裂/黑白闪
二层 [动作音效]    ← Foley：衣袍/脚步/武器/呼吸
一层 [环境音]      ← Ambience：风/云/空气/空间感
```

**铁律**：每镜至少覆盖一层+二层+三层（无台词时）或一层+五层（有台词时）。

---

## 音效时间线结构（双层并行）

每镜音效由两条并行时间线组成。标注时按时间顺序分段，确保音效与画面动作帧级对齐：

```
背景层（Ambience）：贯穿全镜的持续环境音——风、空调低鸣、云层翻涌、人群底噪
事件层（Events）  ：时间点触发的离散音效——脚步、台词、碰撞、关门、道具掉落
```

### 标注格式

```
【音效时间线】
背景：摄影棚环境声——灯架"咔嗒"+对讲机碎片+普通话讨论+脚步"哒哒"+器材轮子+空调低鸣
事件：
  0-1秒：墙体闷声打开+脚步浮现+环境声变清晰
  1-5秒：制片人连珠炮对白+丹眉脚步"哒哒"
  3-5秒（叠加）：场务小跑"踏踏踏"+"姑娘您的剑！"+桃木剑轻微木头摩擦声
  8-10秒：制片人退场脚步远去+丹眉深呼吸"吸——"+整理桃木剑木头摩擦声
  10-11秒：环境声压低→监视器电流声取代
  11-12秒：邵氏场务洪亮"Ready——Action！"+环境声0.3s淡出
```

### 叠层音效

同一时间点多事件并行时，用 `（叠加）` 标记：
```
3-5秒（叠加）：[事件A音效]+[事件B音效]+[事件C音效]
```

## 拟声词标注法

关键音效使用拟声词让 AI 精确理解音色和节奏，比抽象描述更有效：

| 类型 | 拟声词 | 适用场景 |
|------|--------|---------|
| 脚步硬地 | `"哒"` | 地坪漆/石面/水泥 |
| 脚步木地 | `"咚"` | 舞台木板/木地板 |
| 脚步急促 | `"踏踏踏"` | 小跑/快走 |
| 金属碰撞 | `"叮当"` | 钥匙串/金属饰品 |
| 厚重关门 | `"砰"`（重、闷） | 大门/户门关闭 |
| 开门锁 | `"咔哒"`+`"吱呀"` | 门把手+门轴 |
| 深呼吸 | `"吸——"` | 准备/蓄力/平复 |
| 灯架 | `"咔嗒"` | 金属灯具机械声 |
| 瓷器碎裂 | `"哗啦"` | 瓷瓶/瓷片破碎 |

拟声词用引号包裹（`"哒"`），后跟括号说明质感（`"砰"（重、闷）`）。

## 音效节奏对比

同一段落中多个同类音效需设计节奏对比如"三声门"模式：

```
【三声门音效节奏对比】
第9秒  "砰"=出门关门声（重、闷）
第12秒 "咔哒+吱呀+气流"=回来开门声（轻、有层次）
第14秒 "砰"=再次出门关门声（重、闷，和第一次形成回收）
→ 开门和关门的音效必须明显不同，不要混淆
```

对比要素：**轻重、快慢、距离、混响**——至少两个维度形成差异。

---

## Beat-Sync 音效

动作音效与节奏同步时，明确标注同步关系：

```
丹眉脚步cadence严格sync to [参考节奏]——each footstep lands precisely on a beat
步幅步频一致+较慢但坚定+脚步"哒"和beat精确同步
```

Beat-sync 适用场景：
- 角色行进（脚步声 = 画面节奏锚点）
- 打击乐式动作（锤击/劈砍/拍桌——每击一个重音）
- 心跳声（紧张场景——心跳与镜头推拉同步）

## 环境声空间演变

一镜到底穿越多个物理空间时，环境声必须随空间变化同步演变——不能全段一个环境底噪：

```
通道阶段：工作噪音（对讲机+灯架+脚步+空调）+深灰色地坪漆空间混响
  ↓ 踏上舞台瞬间
舞台阶段：暖色环境声（电流嗡+空旷混响）+观众席低声窃窃私语+脚步"咚"（木地板）
```

### 空间转变触发点

空间转变瞬间必须同时写出：
1. **视觉触发**：地面纹理变化 / 光线色温跳变 / 空间开阔度变化
2. **听觉触发**：混响特征变化 / 新环境声入场 / 旧环境声退场
3. **动作触发**：脚步音色改变 / 衣袍声音在新空间中的混响变化

**示例**（踏上舞台瞬间）：
```
地坪漆→温暖木质舞台板（视觉）+脚步"哒"→"咚"（听觉）+密闭混响→空旷扩散感（空间）+观众席哗然窃窃私语（新环境声入场）
```

## 音效标注格式

### 无台词镜头

```
【无台词】环境音+动作音效+冲击音效（如有）
```

示例：`【无台词】高空劲风啸+飞刀破空尖啸+衣袍甩动`

### 有台词镜头

```
台词：角色："原文" 配音状态 【音效】环境音+动作音效
```

示例：`台词：修士A："狂风斩！" 爆发怒吼，声带撕裂 【音效】衣袍翻飞+刀身破风沉重啸`

### 音效层级叠加

当一镜有多层音效时，用+连接，按重要性排序：

`【无台词】劲风啸（一层）+衣袍猎猎（二层）+刀尖撞击爆响+火花迸射（三层）`

## 声音设计铁律（5条）

1. **五层不缺**：每镜至少覆盖环境音+动作音+冲击音（或人声）
2. **写实优先**：粒子/火焰/烟雾/爆炸的VFX音效必须是写实级——不是卡通音效
3. **静默是音效**：黑白闪/冲击帧配0.3s绝对安静——静默比巨响更有冲击力
4. **法宝有专属音色**：每个法宝有独特的声音签名——飞刀尖啸/黑镯暗紫嗡鸣/真元盾金色共鸣
5. **空间感一致**：高空场景=开放空间混响（声音有扩散感）；室内=封闭混响（声音有反射）
