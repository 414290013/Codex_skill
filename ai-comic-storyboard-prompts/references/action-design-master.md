## 一、戏型/镜头语法分类体系（15型）

每段动作戏必须标注戏型。戏型决定镜头节奏、景别选择、运镜方式和节拍密度：

### 修仙打戏类

| 戏型 | 核心特征 | 镜头节奏 | 适用场景 |
|------|---------|---------|----------|
| **宏观环境建立** | 大远景→远景，无人物或人物极小，场景叙事 | 3-5s/镜，慢摇/航拍 | 片头/场景切换/战斗开幕 |
| **宏观-中观链式建立** | 大远景→中景逐步推进，从环境到人物 | 3-6s/镜，缓推/摇 | 战斗场景逐步聚焦到角色 |
| **蓄力-释放-接触锁** | 蓄力→释放→飞行/扩散→接触目标→反馈 | 1.5-3s/镜，快切伴随运镜 | 远程能量攻击/法宝释放/神通出手 |
| **情绪锚点-状态锁** | 角色情绪姿态定格→环境响应→状态持续 | 2-4s/镜，缓推/固定 | 角色性格外化/实力展示/对峙 |
| **状态锁延续-余波** | 上一镜状态持续→新动作打破→环境响应 | 2-5s/镜，环绕/跟拍 | 实力碾压展示/反转揭示 |
| **攻防释放-物理反馈** | 攻击释放→飞行→接触→物理反馈→角色反应 | 1.5-3s/镜，快切+震动 | 战斗高潮/大招对轰 |
| **物理反馈-状态锁** | 角色被击中→身体反馈→生理反应→状态定格 | 2-4s/镜，推镜/特写 | 被击中/承受伤害/极限状态 |
| **追逐节拍快切** | 追逐者→被追者→速度线→环境快速切换 | 0.8-2s/镜，极快切 | 追击/逃跑/高速移动 |

### 文戏类

| 戏型 | 核心特征 | 镜头节奏 | 适用场景 |
|------|---------|---------|----------|
| **信息炸弹特写链** | 关键信息→角色微表情→环境静默→反应 | 2-4s/镜，缓推/固定 | 重大信息揭示/身份曝光/秘密揭晓 |
| **对话呼吸链** | 对话→呼吸间隙→微动作→下一句 | 2.5-5s/镜，固定/微摇 | 审问/盘问/重要对话 |
| **情绪高峰锚点链** | 情绪逐步攀升→峰值锚点→生理反馈→释放 | 2-4s/镜，推镜/特写 | 情感决裂/生死抉择/重大决定 |

### 混合转场类

| 戏型 | 核心特征 | 镜头节奏 | 适用场景 |
|------|---------|---------|----------|
| **文戏压迫→动作爆点** | 对话/静态→突然动作→音爆→空间跳切 | 文戏部分2-4s→爆点0.5-1s | 主角入场/冲突突然升级 |
| **微动作反应-爆点** | 微小预兆→角色感应→突然爆发→连锁反应 | 感应1-2s→爆点0.5-1.5s | 预感/偷袭/突发危机 |
| **动作反馈-情绪压迫** | 动作结果→身体反馈→情绪蔓延→环境压迫 | 2-4s/镜，推镜+环境响应 | 诅咒/中毒/负面状态降临 |
| **对话呼吸-环境响应** | 对话→环境开始变化→对话被打断→环境爆发 | 对话2-3s→环境响应1-2s | 天劫/天地异象/灵力暴走 |

## 二、动作节拍扩展公式（五段法·升级版）

国术五镜公式的升级扩展。每个完整动作拆解为五段。**重要**：完整五段通常跨3-5镜分布，单镜仅承载1-2段——不可在单个5s镜内塞入全部五段（违反节拍密度限制）。近战动作中"飞行段"折叠为"运动轨迹段"（时长≤0.3s），肢体接触发生在同一镜内。

```
蓄力清晰段 → 释放帧 → 飞行/扩散段 → 接触帧 → 物理反馈/反应链
```

### 2.1 蓄力清晰段（0.5-2.0s）

动作前的能量聚集阶段——让观众预期即将发生什么：

| 蓄力类型 | 视觉关键词 | 示例写法 |
|----------|-----------|---------|
| 呼吸蓄力 | 胸腔鼓胀→血管凸显→内部发光 | `猛然深吸一口气，胸腔极度夸张地吸得鼓起，脖颈血管凸显，胸腔内部一团刺目的高热橙光亮起` |
| 重心蓄力 | 压低重心→肌肉绷紧→脚下地面龟裂 | `双腿微屈压低重心，腰胯下沉，双脚踩实地面，脚下泥土龟裂崩碎` |
| 灵力蓄力 | 丹田发光→经脉传导→肢端凝聚 | `丹田处金色真元自发光透衣隐约可见，金色脉冲沿手臂经脉传导至掌心，掌心光球急速膨胀` |
| 武器蓄力 | 引刀/引剑至极限位→刀身震颤→刃口寒光凝聚 | `脊椎如弓弦向后绷紧，刀身引至身后极限位，刀身震颤发出低沉嗡鸣，刃口凝聚冷白寒光` |
| 法宝蓄力 | 法宝旋转加速→符文亮起→灵力缠绕 | `袖中黑镯旋转加速，暗紫符文逐一亮起，紫色灵力呈螺旋缠绕镯身` |

**蓄力段铁律**：
- 蓄力占主动作时长的20-30%（与animation-principles.md的预备动作占比统一）
- 必须有至少一个"能量/力量正在聚集"的视觉信号
- 蓄力段不能跳过——无蓄力=动作无重量

### 2.2 释放帧（0.2-0.8s）

能量/力量从蓄力状态转为释放状态的临界瞬间：

| 释放类型 | 视觉关键词 | 示例写法 |
|----------|-----------|---------|
| 喷吐释放 | 张口→火焰/能量喷薄→反冲力 | `大口一张，一股猛烈无比的柱状高压火焰（抖动线+高压火柱）汹涌喷薄而出，反冲力使身体后仰` |
| 挥击释放 | 腰胯扭转→脊椎传导→肩臂甩出 | `腰胯猛然扭转带动脊椎，肩臂如鞭甩出，刀身在最高速处脱手` |
| 蹬踏释放 | 脚下炸裂→音爆云→身形弹射 | `双脚猛然蹬踏，脚下岩石轰然炸裂，一圈白色音爆云炸开，身形瞬间弹射而出` |
| 掌指释放 | 五指张开/结印→掌心能量喷射 | `右手猛然前伸，五指张开呈爪，掌心旋涡状吸力（空气扭曲波纹）凭生` |
| 法宝释放 | 法宝脱手→灵力尾迹→锁定目标 | `袖中腕部弹抖，黑镯脱手而出，暗紫灵力呈螺旋缠绕镯身，破空飞向目标` |

**释放帧铁律**：
- 释放帧必须有"力从何处发出→经何路径→向何处释放"的完整力链
- 释放瞬间必须有反冲/惯性反馈——不写反冲=动作轻飘飘
- 所有远程能量攻击的释放帧：被攻击方不得同框（致死级——即梦同框能量穿帮）

### 2.3 飞行/扩散段（0.5-3.0s）

能量/武器/角色从释放点到接触目标的中间过程：

| 飞行类型 | 视觉关键词 | 示例写法 |
|----------|-----------|---------|
| 能量飞行 | 火柱/光柱/能量束→沿途环境响应 | `火柱贯穿天空，沿途云层被高温蒸发形成空洞，云层边缘被染成炽热橘红` |
| 法宝飞行 | 法宝旋转→灵力尾迹→破空轨迹 | `飞刀破空，刀尖向前，速度线向刀尖收敛，身后拉出2D手绘白色气刃轨迹` |
| 角色突进 | 音爆云→尾焰轨迹→速度线→出画 | `周身炸开音爆云，身形化作一道带尾焰的火光以45度角瞬间冲出画面——镜头固定不跟随` |
| 火焰扩散 | 火海蔓延→火舌翻涌→覆盖区域 | `一片暗红火海从释放点瞬间铺满整个上空，火舌翻涌如狂潮，云层被火舌绞碎` |
| 吸力/引力 | 旋涡→物质流向→空间扭曲 | `漫天狂躁的火海如涓涓流水般化作数十道螺旋火线，不断朝掌心汇聚压缩` |

**飞行段铁律**：
- 能量/火焰飞行途中：攻击方和被攻击方不得同框
- 必须有明确的方向性——速度线/轨迹线/尾迹
- 飞行距离越长→镜头时间越长→环境响应越重要

### 2.4 接触帧（0.3-1.0s）

能量/武器命中目标的瞬间：

| 接触类型 | 视觉关键词 | 冲击帧 | 示例写法 |
|----------|-----------|--------|---------|
| 能量命中 | 爆炸/火光吞噬/全身包裹 | 纯白帧(0.2s)→火光 | `迎面一股灼热气浪（画面极度扭曲）扑面而来，瞬间将其整个人完全吞没进火海，镜头被火光吞噬白屏` |
| 利刃命中 | 火花爆散/金属形变/切入 | 黑白闪(0.3s) | `刀尖撞击盾面，金属应力嘎吱声，撞击点爆出密集火花，盾面涟漪急剧扩散` |
| 钝器命中 | 冲击波环/身体形变/地面碎裂 | 纯黑帧(0.2s) | `巨掌拍下，冲击波环从命中点向外扩散，地面龟裂呈蛛网状，身体被压入地面` |
| 灵力命中 | 光爆/能量涟漪/身体震颤 | 分色帧（画面以接触线为界，左半暗紫右半金色在撞击点激烈对撞闪烁0.3s） | `金色光柱轰击胸口，暗红与金色能量涟漪从撞击面爆散，身体内部透出震颤光纹` |

**接触帧铁律**：
- 重击接触必须配冲击帧（黑白闪/纯色帧）——全片3-5个最重要的撞击瞬间
- 接触帧必须有"物理反馈"：火花/碎片/冲击波/身体形变——不能只有光
- 接触点必须在画面中可见

### 2.5 物理反馈/反应链（1.0-4.0s）

命中后的身体反应和情绪反馈：

**可见反馈层级链（由外到内的感受必须改写成镜头可见信息）**：

```
第一层·体表/衣物反应（0-0.5s）：衣袍震颤→发丝扬起→冷汗渗出→手指收紧
第二层·姿态/重心反应（0.2-0.8s）：肩背一沉→膝盖弯折→重心后退→手扶支撑物
第三层·呼吸/声音反应（0.5-1.5s）：胸口急促起伏→呼吸中断半拍→闷哼→耳鸣声
第四层·视线/感知反应（0.3-1.0s）：瞳孔收缩→眨眼失焦→视线偏移→短暂停顿
第五层·行为外化（1.0-3.0s）：后退半步→握紧道具→重新站稳→反击准备
```

**标准可见反馈链示例**：

| 伤害类型 | 第一层 | 第二层 | 第三层 | 第四层 | 第五层 |
|----------|--------|--------|--------|--------|--------|
| 火焰烧伤 | 皮肤瞬间焦黑→衣物炭化冒烟 | 身体蜷缩→双手护面 | 呼吸急促→吸入浓烟呛咳 | 瞳孔极度收缩→惨叫嘶吼 | 疯狂腾挪跳转→试图扑灭火焰 |
| 重击命中 | 脸色瞬间煞白→冷汗渗出 | 身体下沉→脊椎弯曲→膝盖弯折 | 胸腔震动→呼吸断裂 | 瞳孔放大→意识模糊0.3s | 牙齿打颤→眼神绝望→声音颤抖 |
| 灵力侵蚀 | 皮肤浮现暗色纹路→血管变黑 | 身体僵硬→关节锁死 | 经脉刺痛→灵力运转受阻 | 瞳孔忽大忽小→视线扭曲 | 面部扭曲→嘶哑低吼 |
| 雷击 | 全身瞬间僵硬→发丝竖立 | 身体弓起→肩背猛收 | 胸口急促起伏→呼吸断半拍 | 眼前白光用瞳孔缩小和白光反射表现 | 身体后退扶住墙面 |

**反应链铁律**：
- 每个重大命中至少覆盖3个可见层级
- 内部痛感必须转写为姿态、呼吸、视线、衣物道具、环境声光变化
- 禁止只写情绪不写身体——"痛苦"不如"身体蜷缩+牙齿打颤+冷汗渗出"

## 三、衔接逻辑系统（Transition Logic）

每镜末尾标注衔接逻辑，指导即梦Seedance理解镜间连续性：

### 3.1 衔接逻辑分类

| 衔接类型 | 适用场景 | 上一镜末→下一镜首的连续性要求 |
|----------|---------|------------------------------|
| **空间建立+硬切** | 新场景/新空间首次出现 | 上一镜末：无特殊要求；下一镜首：0.5s场景建立期 |
| **内容逻辑+形式逻辑** | 同一空间内的连续事件 | 保持空间坐标+光方向连续性 |
| **因果关系+Match Cut** | 动作A导致动作B | 上一镜末：动作结果；下一镜首：承接结果的立即反应 |
| **动作顺序+Match Cut** | 同一动作的分解步骤 | 上一镜末：动作中段；下一镜首：动作延续 |
| **动作匹配+Action Match** | 同一动作跨镜延续 | 上一镜末：动作50%完成；下一镜首：动作51%起，运动轨迹无缝衔接 |
| **视线引导+硬切** | 角色视线引导切换目标 | 上一镜末：角色视线方向明确；下一镜首：视线目标(沿视线方向) |
| **情绪呼应+L-Cut** | 情绪递进/反应链 | 上一镜末：情绪起点；下一镜首：情绪递进(声音先入) |
| **情绪呼应+J-Cut** | 声音先入引导情绪转换 | 上一镜末：画面持续；下一镜声音已开始 |
| **视线引导+J-Cut** | 角色视线望向某处+声音先行 | 上一镜末：角色视线方向+面部反应；下一镜声音提前0.3s |
| **空间跳切+动作匹配** | 快速场景切换/追逐 | 上一镜末：角色做动作；下一镜首：动作在新空间延续 |
| **因果关系+L-Cut** | 因→果，声音跨镜 | 上一镜末：原因动作；下一镜首：结果+上一镜声音延续 |

### 3.2 衔接逻辑铁律

- 同一大分镜段内所有镜头按"上一镜末→下一镜首"逻辑逐一标注衔接类型（本段末镜除外——末镜无下一镜，不写衔接逻辑）
- 不同场景之间的切换必须使用"空间建立+硬切"或"空间跳切+动作匹配"
- 对话场景衔接优先使用L-Cut/J-Cut——声音跨镜平滑过渡

## 四、站位基线系统（坐标空间定位）

### 4.1 站位基线格式

每个大分镜段首声明角色空间位置：

```
此时此刻站位基线：[@角色A坐标（X,Y,Z）层次/朝向/屏幕位置；@角色B坐标（X,Y,Z）层次/朝向/屏幕位置；战斗距离X米；180度轴线]
```

### 4.2 本镜站位基线格式

每镜时间行声明本镜角色位置变化：

```
本镜站位基线：@角色A坐标（X,Y,Z）[状态描述]；@角色B坐标（X,Y,Z）[状态描述]；[空间关系变化]
```

### 4.3 坐标系统约定

- X轴：屏幕左右（正值=画右，负值=画左）
- Y轴：屏幕上下（正值=画上方，负值=画下方）
- Z轴：屏幕深度（正值=远离镜头/远景，负值=靠近镜头/前景）
- 坐标原点(0,0,0)=画面正中心

### 4.4 站位基线铁律

- 段首必须声明所有出场角色的空间位置
- 角色在本段内移动时，每镜必须更新站位基线
- 180度轴线锁定后全段不变——摄影机不越轴
- 战斗距离必须标注——影响景别选择和节拍密度

## 五、运镜速度与镜头运动词典（扩展版）

### 5.1 运镜速度控制

| 速度等级 | 中文指令 | 适用场景 |
|----------|---------|----------|
| 极慢 | `镜头极缓慢推近` | 情感凝视/细节揭示/灵异感 |
| 慢 | `镜头缓慢推近` / `缓推` | 情绪升温/顿悟同步/仪式感 |
| 中 | `镜头平稳推进` / `平稳摇` | 跟踪主体/场景扫描 |
| 快 | `镜头快速推近` / `快推` | 冲击/威胁/突然发现 |
| 极快 | `镜头急速推近` / `急速摇` | 危险瞬间/爆点/追逐 |

### 5.2 特殊运镜指令

| 运镜类型 | 中文指令 | 用途 |
|----------|---------|------|
| 水平环绕 | `镜头水平环绕X度` | 展示角色/场景的空间关系 |
| 360度大环绕 | `镜头360度大环绕` | 极致奇观/宏大场面 |

## 六、对话场景分镜法则（升级版）

### 6.1 文戏镜头语法

结合相机艺术铁律与需求文档实战经验：

**信息炸弹（重大信息揭示）镜头序列**：
```
镜1(3-4s)：信息揭示者全景/中景——说话+表情——固定/微推
镜2(2-3s)：听者ECU面部特写——瞳孔变化+微表情生理链——缓推
镜3(2-3s)：环境空镜——静默/停顿——固定
镜4(3-4s)：听者反应——身体语言+台词——固定
```

**对话呼吸链（审问/盘问）镜头序列**：
```
镜1(4-5s)：双人中景定场——3-5句台词不切镜——固定
镜2(3-4s)：问者近景——1-2句追问——微推
镜3(2-3s)：答者ECU面部——微表情+呼吸+回答——固定
镜4(3-4s)：双人中景——反应+下一轮对话——固定
```

### 6.2 混合转场镜头语法

**文戏压迫→动作爆点**：
```
镜1(3-4s)：文戏——角色对话/观察——固定/微推
镜2(0.5-1.5s)：起势——角色身形爆发（重心下沉+能量聚集）——快切+震动
镜3(1-2s)：释放——火焰翅膀展开/音爆起飞——跟拍/仰拍
```

**微动作反应-爆点**：
```
镜1(1-2s)：预兆——角色感应/识海警示——微推
镜2(0.5-1s)：爆点——偷袭/突发攻击——快切+震动
镜3(1-2s)：反应——被攻击方反馈——固定/微震
```

### 7.2 修真战斗专用动作词汇

| 动作类型 | 核心词汇 | 示例写法 |
|----------|---------|---------|
| 神通释放 | 凌空挥手/剑指一点/抬手一扫 | `黑袍凌空挥手，一片火海瞬间铺满整个上空` |
| 神通吸收 | 掌心吸力/漩涡汇聚/涓流归海 | `右手伸开后凭生一股吸力，狂躁火海如涓涓流水般向手掌汇去` |

| 喷火反击 | 胸腔蓄力+张口喷薄+火焰柱状 | `猛然吸了口气将胸腔吸得鼓起，一口气喷薄而出，猛烈火焰汹涌喷去` |
| 炭化反馈 | 衣物炭化+身体焦黑+冒烟 | `火海中一道身影被烧得浑身焦黑，衣袍炭化成片状剥落，黑烟滚滚` |
| 法宝偷袭 | 袖中暗器+阴毒手段+诡笑 | `袖袍一挥，一道暗光从袖中飞出，嘴角勾起诡异笑容` |

### 7.3 火焰/能量攻击强制分镜规则（致死级）

即梦在同框画面中会将攻击方和被攻击方之间的火焰/能量直接连接到被攻击方身上，造成烧身/击中穿帮。

**强制分镜三段式**：

```
镜A（攻击方释放）：攻击方正面释放火焰/能量，能量向镜头方向或画外扩散。被攻击方不出镜。
镜B（能量飞行/扩散）：纯能量/火焰画面，或环境响应。双方角色均不出镜（或仅保留环境）。
镜C（被攻击方反应）：被攻击方特写/近景。只写火光照亮面部+环境光变化——禁止出现火焰本体。
```

**铁律**：镜A和镜B、镜B和镜C之间用"切至"明确断开，切断即梦在同框中建立的空间连续性。

## 八、动作音效集成体系

### 8.1 动作类型→音效映射

| 动作类型 | 环境音（一层） | 动作音（二层） | 冲击音（三层） | 特殊音 |
|----------|-------------|-------------|-------------|--------|
| 蓄力 | 空气低鸣 | 衣袍紧贴身子的摩擦声 | — | 内部能量嗡鸣渐强 |
| 释放（火焰） | 风噪加剧 | 衣袍被气浪掀动 | 高压喷火轰鸣 | 反冲力造成的低沉音爆 |
| 释放（法宝） | 环境回音 | 法宝脱手Foley | 尖锐破空声 | 法宝专属音色 |
| 飞行/突进 | 狂风呼啸 | 衣袍高速猎猎 | 音爆轰鸣（Sonic Boom） | 残影分身消散声 |
| 接触（能量命中） | 环境瞬间静默(0.2s) | 火焰扑面的轰爆声 | 被命中者的倒抽气/惨叫 | 身体炭化滋滋声 |
| 接触（利刃命中） | 撞击点环境震动 | 金属应力嘎吱 | 撞击爆响+火花迸射 | 武器震颤余音 |
| 急停/制动 | 风噪骤停 | 衣袍惯性前冲猎猎 | 急停制动音爆 | 反向气浪扩散声 |
| 吸收/汇聚 | 环境声音被吸入 | 能量流动呼啸 | 压缩/汇聚的尖锐啸叫 | 吸尽后的瞬间死寂 |

### 8.2 音效铁律

- 每镜至少覆盖环境音+动作音+冲击音（或人声）
- 重击命中瞬间：冲击帧（黑白闪）= 0.2-0.3s绝对安静——静默是最高境界的音效
- 法宝/武器有专属音色——全片保持一致
- 音效标注格式：`【无台词】环境音+动作音+冲击音` 或 `台词：角色："原文" 配音状态 【音效】...`

## 九、动作设计的全局约束

### 9.1 风格锚定

所有动作描述以**参考图主体画风**为唯一锚点：
- 角色外观/服饰/体型/发色/瞳色全部由参考图决定
- 动作描述中只写"角色名的衣服""角色名的衣袍"——绝不描述颜色/款式/材质
- 远景角色用`（角色名的衣服颜色）`标记，"颜色"为参考图自动填充占位符

### 9.2 即梦Seedance 2.0适配

- 所有特效必须物理化描述——不写"华丽的爆炸"，写"2D手绘金色粒子从中心向外爆散+冲击波环扩散"
- 每镜只有1个运镜指令
- @引用：N个确定角色 = N个独立@引用
- 单镜内动作节拍：5s ≤ 2节拍，3s ≤ 1节拍
- 前0.5s安全区（场景建立），后0.5s安全区（动作收束）

### 9.3 国术动作铁律（所有动作场景的物理真实性基石——引自 fight-vfx.md）

1. **发力必有重心变化**：每个攻击/防御动作描述中必须隐含重心转移（前倾/后坐/侧移/下沉/上提），不可"悬浮不动只动手"。示例：`腰胯下沉重心前压，肩臂如鞭甩出`，而非`手臂一挥`。
2. **武器有重量**：重武器挥动时身体应有惯性反馈（手臂肌肉承重、身体随刀势微晃），不可"轻飘飘挥舞"。示例：`大刀沉重挥下，双臂肌肉隆起承重，身体随刀势前倾——从起势到落点用时1.2s`。
3. **修真≠无视物理**：真元可以强化力量和速度，但不能替代发力逻辑——灵力沿力链路径传导并放大，而非凭空出现。示例：`金色真元从丹田经腰胯→脊椎→肩臂→刀身传导并放大`，而非`刀身突然发光威力无穷`。

### 9.4 合规约束

- 画面无写实可辨识人脸
- 无血腥暴力元素（血液→暗红液体，死亡→失声/停滞）
- 无低俗违规内容
- 无版权保护IP元素

## 十一、动作场景致死级陷阱汇总

动作场景是即梦Seedance 2.0陷阱最高发区。以下汇总动作场景特别容易触发的致命错误及规避要点：

### 11.1 同框能量穿帮（致死级——远程能量攻击必读）

**陷阱**：即梦在同框画面中会将攻击方和被攻击方之间的火焰/能量直接连接到被攻击方身上，造成烧身/击中穿帮。

**规避**：强制分镜三段式——镜A（攻击方释放，被攻击方不出镜）→ 镜B（纯能量/环境，双方均不出镜）→ 镜C（被攻击方反应，只写火光照亮面部，禁止火焰本体出现）。详见本文 7.3 节。

### 11.2 手指畸变（致死级——结印/掐诀/掌法/手部特写必读）

**陷阱**：即梦对手部解剖结构理解极弱，含手部动作的镜头极易出现多指/并指/断指。

**规避**：①每镜含手部动作必须写出精确手指数量——`五指张开呈爪` `食指+拇指掐诀` `双手十指交叉结印`，不写模糊的"手指""手势"；②能用遮挡就遮挡——袖口/衣袍/灵力特效部分覆盖手背；③手部特写前加缓冲镜——前1镜给手部中景/全景预备动作。详见 `references/seedance-spec.md` 陷阱四。

### 11.3 道具穿帮（致死级——武器脱手/法宝收回必读）

**陷阱**：即梦逐镜独立生成，道具被收走/转移/销毁后，下一镜若未明确声明道具的缺席，AI在画面中残留该道具。

**规避**：①道具转移后下一镜明确新持有者——`飞刀在@角色A掌中，@角色B手中空无一物`；②道具消失后用物理否定描述——`画面中无飞刀` `双手空空` `地面仅青石地砖`；③关键道具首次出现绑定角色归属。详见本文 9.2 节及 `references/output-format.md` 道具跨镜连续性铁律。

### 11.4 角色消失残留（致死级——遁逃/传送/隐身必读）

**陷阱**：即梦不理解镜头间的叙事连续性。角色消失后，后续镜头若出现@角色名引用，AI必然在画面中生成该角色。

**规避**：①消失瞬间用物理化工序描述消解过程——`人形轮廓急速模糊消解为暗色粒子流，粒子彻底消失在虚空中`；②消失后从该镜起彻底删除@引用；③用否定物理描述替代状态描述——`画面中无任何人形轮廓`而非`角色已消失`。详见 `references/seedance-spec.md` 陷阱三。

### 11.5 物理障碍穿透（致死级——室内战斗/绕障必读）

**陷阱**：即梦不理解场景中的物理障碍物。角色需要绕过/跨过障碍时，若不写绕行动作，AI会让角色直接穿透。

**规避**：①段首场景声明必加入阻挡物位置和尺寸；②人物位置必相对于障碍物——`@角色A站于案桌后方`；③穿越障碍必写绕行动作——`从案桌后猛然起身绕至案桌前`。详见 `references/seedance-spec.md` 陷阱七。

### 11.6 飞行冲击缺失（致死级——起飞/破空/急停必读）

**陷阱**：即梦默认生成的飞行/瞬移镜头缺乏物理重量感——角色轻飘飘浮起或平移。

**规避**：起飞必有反冲破坏（蹬踏点碎裂/气浪）+ 破空必有音爆（环形音爆云+速度线+尾焰轨迹）+ 急停必有制动冲击（反向气浪+衣袍惯性前冲）。禁用"轻轻一跃""飘然而起""腾空而去"。

**特别注意——残像分身是即梦致死级陷阱**：即梦逐镜独立生成，无跨镜记忆。`身后拉出N个2D手绘残像分身`——即梦会将残像理解为画面中同时存在多个独立人物，导致畸变、粘连、或残像被当作实体角色。**替代方案**：用`以45度角瞬间冲出画面`（固定镜头不跟随）+ 下一镜直接切至到达位置。用镜头切换代替残像拖尾——这是2D动画中"出画→入画"的经典技法，即梦执行安全。

详见 `references/animation-principles.md` 飞行冲击三要素。

### 11.7 动作节拍密度超限（致死级）

**陷阱**：5s单镜塞4-5个节拍=动作糊、跳、不自然=作废。完整五段链被错误压缩进单个镜头。

**规避**：单镜内5s≤2节拍、3s≤1节拍；完整动作五段链跨3-5镜分布执行。详见本文第二章五段法及 `references/seedance-storyboard-skill.md` 节拍密度规则。

### 11.8 角色入场空间定位错误（致死级——瞬移/破空/突入必读）

**陷阱**：角色从远处破空而至时，分镜师容易忽略该角色与在场其他角色的相对空间关系——角色可能出现在错误位置（如挡在A身前却写成了挡在B身前），导致后续对话和动作逻辑全段崩塌。

**规避**：
① 入场前必读段首站位基线——确认入场角色起飞前位置、到达后位置、以及其他角色的当前位置；
② 入场镜必须明确"挡在谁身前"或"落在哪个角色前方"，且方向描述必须与该角色朝向一致——如"挡在四旬男人身前正面对向黑袍"而非模糊的"出现在战场中"；
③ 如果入场是拦截/保护性质——入场角色必须在被保护者和威胁源之间，站位基线的Z轴坐标要体现这层空间关系（入场者Z轴在被保护者之前、威胁源之后）；
④ 入场后的第一反应镜——如果是重要对手的惊讶/识别反应，必须单独拆镜，不得与入场镜合并。入场镜=动作镜（无声或环境音），反应镜=对白镜（对手台词），二者不可混镜。

### 11.9 远景穿帮——观察者望向远处时即梦渲染缺失（致死级——观望/监视/隔空观察必读）

**陷阱**：角色在远处观察高空/远方发生的事件时，若镜头描写"目光穿过云层锁定远处角色"或"眼睛映着远方火光"，即梦会试图渲染角色目光方向的远处人/物。但即梦不理解跨段叙事——它不知道上一段中远处正在发生什么（如火海包围、战斗进行中），导致渲染出的远处场景为空镜、错误的静态人物、或与上一段状态矛盾的穿帮画面。

**规避**：
① 观察者镜头中角色面对镜头——用表演传达"观察到什么"而非让即梦去渲染被观察物；
② 在提示词末尾加入物理否定声明：`画面中无[被观察角色名]` `背景仅见[场景元素]`，明确切断即梦生成远处人物的可能；
③ 如果必须展示被观察物——切至独立的主观POV镜（纯环境/纯远处场景），观察者和被观察者绝不同框；
④ 声音提示：用环境音暗示远处正在发生的事（如火海轰鸣、打斗声），画面不依赖即梦去渲染；
⑤ 禁止在观察者镜头中写"目光穿过X锁定Y的身影""瞳孔中映出Y的倒影"——这些需要即梦同时准确渲染Y，属于跨镜记忆依赖。

### 11.10 蒙面/兜帽角色的面部五官穿帮（致死级——蒙面巾、兜帽、面具角色必读）

**陷阱**：角色设定中面部被蒙面巾/兜帽/面具遮挡（如"兜帽深垂遮住大半面容""嘴唇被灼烧黏连"），但分镜提示词中描写了"嘴角勾起""嘴唇微张""瞳孔放大""面部肌肉变化"等五官细节——即梦不理解遮挡关系，会在蒙面布/兜帽表面直接生成嘴巴和面部器官，造成"布上长嘴"的严重穿帮。

**规避**：
① 兜帽/蒙面完好时——提示词中必须声明`兜帽阴影完全吞没面部不见任何五官`，禁止描写任何面部器官动作。用身体语言替代面部表情：肩膀后展/塌缩、头略上扬/前倾、身体一震/后缩、姿态舒展/僵硬；
② 声音替代表情——蒙面角色的情绪转折通过声线变化（鼻腔轻哼→声音戛然而止、倒抽气）传达，不依赖面部；
③ 如果剧情需要展示面部——必须先确认兜帽/蒙面是否被破坏。如被火焰烧毁→写`兜帽已被烧毁大半露出下方焦黑面部`，之后再描写面部细节；
④ 检查人物设定中的面部状态——若设定写"嘴唇被灼烧黏连仅留干裂缝隙"，则该角色不具备张嘴/嘴角活动能力，全剧所有镜头禁止描写嘴唇动作。

## 十二、世界顶级动作设计核心法则

> 以下法则提炼自世界顶级动作电影（成龙、甄子丹、John Wick、Mad Max、The Raid）的动作设计哲学，适配即梦2D国漫赛璐璐分镜。

### 12.1 动作升级阶梯（Escalation Ladder）

打斗不是匀速的——每场动作戏必须有逐级攀升的强度阶梯。观众会适应当前强度，必须持续"加码"：

```
第一级·试探（10%篇幅）：轻击→闪避→试探距离→建立对手实力
第二级·交锋（30%篇幅）：连击→格挡→反击→环境利用开始
第三级·升级（35%篇幅）：大招/法宝释放→环境大规模破坏→受伤
第四级·绝境（15%篇幅）：一方濒死→绝地反击→极限状态
第五级·决胜（10%篇幅）：最终一击→冲击帧→余波→静默
```

**铁律**：
- 每一级的能量输出必须比上一级大——火焰从小火球→火海→火柱→核爆级
- 每一级的环境破坏必须比上一级广——从地面龟裂→建筑崩塌→地形改变→天穹撕裂
- 每一级的速度必须比上一级快——从1.5s/镜→1.0s→0.8s→0.5s
- 绝境级插入0.5s慢镜（全片仅1-2次）——慢镜用在"最不该慢的瞬间"
- 不可跳过任何一级——跳过=观众失去情绪锚点

### 12.2 战斗空间地理学（Combat Geography）

世界顶级动作设计的铁律：**观众在任何时刻都必须清楚每个角色在空间中的位置**。

**空间建立三步骤**：
```
步骤1·定场镜（段首1镜，2-3s）：大远景/全景——框入所有角色+关键环境特征
步骤2·轴线锁定：选定180度轴线后全段不越轴——观众的空间坐标系不变
步骤3·每镜更新位置：角色移动后，下一镜的环境背景必须与移动方向一致
```

**空间清晰度检查点**：
- 任何时候暂停画面，观众能说出"A在B的某方向，距离约X米"？
- 角色移动方向是否和背景流动方向匹配？（向右移动→背景向左流动）
- 切镜后角色的屏幕位置是否与上一镜末一致？（A在画左→下一镜A仍在画左）

**禁止**：角色在镜A中位于画左→镜B切到特写后背景完全陌生→观众失去空间感。

### 12.3 动作揭示性格（Character Through Action）

每个角色的战斗风格 = 性格外化。同样的"出拳"，不同性格的角色打出来完全不同：

| 性格类型 | 战斗风格 | 动作关键词 | 镜头语言 |
|----------|---------|-----------|---------|
| 狂傲型（虾仁） | 正面硬刚、不闪不避、碾压式 | 单手吸收、漫步火海、单字控法 | 固定机位+低角度仰拍——"让观众仰视" |
| 阴狠型（黑袍） | 暗中偷袭、法宝暗器、逃遁快 | 袖中摸物、回头诡笑、透支灵力逃跑 | 倾斜构图+快切——不稳定性 |
| 沉稳型（四旬男人） | 长剑横于身前、不轻易出手、观察分析 | 持剑戒备、眉头拧起、剑尖指向 | 平稳横摇+中景——客观冷静 |
| 疯狂型 | 不计代价、以伤换伤、越痛越狠 | 嘶哑狂笑、身体颤巍巍、残存力量全押 | 手持晃动+极特写——混乱感 |

**铁律**：同一角色全片战斗风格统一——虾仁从不躲闪、黑袍永远有后手。

### 12.4 打斗节奏控制（Fight Rhythm）

顶级动作片从不"全程高速"。节奏必须有峰值和谷值：

```
节奏曲线（15s打斗段标准模板）：
秒0-2   ：缓（空间建立+对峙）
秒2-5   ：急（第一轮交锋——3-4镜快切）
秒5-7   ：停（反应镜——角色评估伤害/调整策略）
秒7-11  ：更急（第二轮升级——4-5镜极快切+能量释放）
秒11-13 ：顿（0.5s黑白闪/纯白帧——冲击顶点）
秒13-15 ：缓（余波+状态定格——给观众喘息）
```

**节奏控制技术**：
| 技术 | 效果 | 适用时机 |
|------|------|---------|
| 快切加速（0.8-1.2s/镜） | 制造窒息感、速度感 | 连击/追逐/爆发 |
| 慢镜强调（2.5-4s/镜） | 放大关键瞬间的每个细节 | 致命一击/绝地反击/重大揭示 |
| 静默帧（0.3s黑白闪） | 冲击顶点——声音真空 | 重击命中/爆炸核心 |
| 呼吸镜（1.5-2s空镜） | 给观众消化情绪 | 高烈度段落后 |
| 叠加快切（0.5-0.8s/镜） | 极限速度——多种攻击同时发生 | 大招对轰/决死一战 |

**禁止**：连续6镜以上全是快切（<1s）——观众眩晕失焦。快切段落后必须跟1镜≥2s的呼吸镜。

### 12.5 动-反应链清晰度（Action-Reaction Clarity）

观众必须在0.3秒内理解"谁做了什么→导致什么结果"。混乱=观众出戏。

**清晰度铁律**：
1. **一镜一动作主体**：每镜只展示一个角色的关键动作——不在一镜内同时展示A出拳和B中拳
2. **动作→独立反应镜**：A的攻击镜→B的反应镜（单独切出）——除非轻击/格挡
3. **反应必须承接动作**：A用火球攻击→B的下一镜必须有"火光映照面部+闪避动作"
4. **禁止动作跳跃**：镜A出拳→镜B已是B倒地——中间缺了"拳命中B"的接触帧
5. **能量路径可视化**：火焰/能量从A→B的飞行轨迹必须在独立的"飞行镜"中展示

**标准动作→反应片段模板**：
```
镜A（1.5s）：A释放攻击——蓄力→释放→能量出画（画面中无B）
镜B（1.0s）：能量飞行——能量在画面中飞行+环境响应（无A无B）
镜C（2.0s）：B被命中——能量抵达→命中B→B的身体反馈（画面中无A）
镜D（1.5s）：A的反应——A观察命中效果（画面中无B）
```
