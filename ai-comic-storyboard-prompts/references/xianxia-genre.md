# 修仙/仙侠类动漫制作规范 — 类型专家指导

修仙类动漫有其独特的视觉语法和战斗逻辑。本规范确保分镜不犯类型错误。

---

## 武器物理铁律（致死级）

### 利刃方向法则

所有刃器（飞刀/剑/刀/匕首）在飞行、斩击、指向时，**刃尖必须朝向目标**。这是物理常识——违反则观众立刻出戏。

| 场景 | 正确 | 错误 |
|------|------|------|
| 飞刀出手 | 刀尖向前，刀身沿直线/弧线刺向目标 | 刀柄向前、刀身横飞、刀身翻滚无方向 |
| 飞刀飞行 | 刀尖破空，速度线向刀尖收敛 | "飞刀在空域中旋转飘浮" |
| 飞刀斩落 | 刀刃/刀尖斩在盾面/目标上 | 刀柄或刀背碰撞目标 |
| 飞刀弹飞 | 撞击后刀身失稳旋转倒飞（此时可乱转） | 弹飞前方向正常但弹飞后仍方向正常 |
| 飞刀回返 | 掉头后刀尖重新指向目标再飞 | 刀身横移回来 |

**提示词铁律**：每镜含飞刀/剑/刃器时，必须至少一处明确刃尖方向——"刀尖指向""刀刃朝""刃口对着"。

### 武器旋转规则

- **飞行中**：利刃不应自旋（除非是回旋镖类法宝）。飞刀=刺向目标，不是旋转飞舞
- **被弹飞后**：可以旋转、翻滚、失稳——此时物理规律允许乱飞
- **回返掉头**：刀尖沿弧线划出U形轨迹重新对准目标——不是刀身原地180°翻转

## 修仙战斗视觉语法

### 真元/灵力可视化标准

修真者的灵力（真元）不是"特效"——它是角色体内能量系统的外部投射：

| 灵力阶段 | 视觉表现 | 错误 |
|----------|---------|------|
| 丹田蓄力 | 体内微光透衣袍隐约可见（自发光） | 凭空出现光效 |
| 经脉传导 | 光沿手臂/脊椎路径流动（非随意飘散） | 光效无路径无来源 |
| 掌心/指尖释放 | 光从掌心/指尖向外扩散/喷射 | 光从身体其他部位随机发出 |
| 法宝充能 | 灵力从持宝手沿接触点注入法宝 | 法宝自己发光无能量来源 |

### 法宝行为规范

| 法宝 | 行为特征 | 错误 |
|------|---------|------|
| 飞刀（御物） | 修士神识操控，刀随诀走，有明确飞行轨迹 | 飞刀无规律乱飞 |
| 真元盾（护体） | 以施术者为中心，随身体移动，盾破需重聚 | 盾独立于施术者存在 |
| 黑镯（法宝） | 脱手后自主攻击/防御，有独立行为逻辑 | 法宝行为无规律 |
| 凝聚大刀（化形） | 多把飞刀合并→灵力气场塑形→大刀成型 | 突然出现大刀无过程 |

### 修仙战斗距离感

| 距离 | 战斗方式 | 视觉特征 |
|------|---------|----------|
| 远距（30尺+） | 法宝对轰/御物攻击 | 飞刀轨迹清晰可见，有飞行时间 |
| 中距（10-30尺） | 法宝+术法混合 | 攻防节奏加快 |
| 近距（5尺内） | 近身兵刃+掌法 | 贴身肉搏感，兵器碰撞火花 |
| 零距（接触） | 擒拿/点穴/真元渗透 | 极特写+身体接触点 |

## 修仙角色表演规范

### 情绪表达的仙侠化

仙侠角色的情绪表达应在"真人表演"基础上增加**仙侠风格的克制感**——修真者不会像普通人那样外放：

| 情绪 | 普通人 | 修真者 |
|------|--------|--------|
| 愤怒 | 大吼大叫/脸红脖子粗 | 眼神骤冷+真元外溢暴走+衣袍无风自动 |
| 疼痛 | 惨叫/捂伤口 | 咬牙闷哼+灵力在伤口处汇聚自愈 |
| 恐惧 | 颤抖/后退 | 瞳孔收缩+真元波动紊乱+后退半步即止 |
| 得意 | 大笑/炫耀 | 冷笑/嘴角微扬+灵力光芒更盛示威 |
| 惋惜 | 叹气/摇头 | 眼神一黯+握兵器的手微松+瞬间恢复战意 |

### 流汗与流泪的严格区分（致死级）

修真战斗中的**汗**是战斗消耗的物理证据，**泪**是情感崩溃的信号。二者不可混淆：

| 正确（流汗） | 错误（疑似流泪） |
|------------|---------------|
| 汗水从鬓角滑落 | ~~眼角滑落液体~~ |
| 额头汗水沿脸颊流下 | ~~眼眶湿润~~ |
| 汗珠从下颌滴落 | ~~眼中水光闪动~~ |
| 鬓角渗出细密汗珠 | ~~眼角有泪光~~ |

**铁律**：描写汗时，起点必须是鬓角/额头/脸颊/下颌/鼻尖——绝不可从眼角/眼眶出发。眼眶红是血管充血（愤怒），不是流泪。

## 修仙战斗节奏铁律

1. **利刃永远刃尖朝敌**：飞刀/剑/刀在攻击飞行时，刃尖必须指向目标
2. **灵力有源头**：所有真元特效必须有体内→体外的传导路径
3. **法宝有逻辑**：每个法宝有独立的行为规则，不可随机运动
4. **修真者不失控**：仙侠角色保持一定的克制感——愤怒时"真元暴走"而非"撒泼打滚"
5. **汗≠泪**：战斗后的汗从鬓角/额头出发，绝不从眼出发
