# Seedance 2.0 提示词工程规范

即梦Seedance 2.0 AI视频生成专用提示词规范。Seedance 2.0不是画笔——它是摄影机+演员+灯光师+后期团队。用导演思维写提示词，描述画面、动作和情绪，不堆砌技术参数。

## 五要素导演法

每个AI视频提示词遵循此结构：

```
[主体] + [动作] + [环境] + [镜头运动] + [风格/约束]
```

| 要素 | 核心问题 | 要求 |
|------|----------|------|
| 主体 | 谁/什么在画面里？ | 具体外观特征，不抽象 |
| 动作 | 发生什么？ | 物理化工序描述——"做什么→怎么做→物理结果" |
| 环境 | 在哪里？ | 含光线、氛围、空间尺度 |
| 镜头 | 摄影机怎么动？ | 只写1个主镜头指令 |
| 风格/约束 | 什么感觉？不要什么？ | 风格参考词+负面排除词 |

## 种子铁律

### 1. 单一镜头指令
每个AI视频提示词只能有**1个**镜头运动指令。推/拉/摇/移/跟/升/降/手持/固定——选一个。多个镜头指令=画面必然抖动。

### 2. 禁止参数堆砌
- 错误：`8K超高清 HDR 120fps 电影级画质 杜比视界 IMAX 奥斯卡最佳摄影`
- 正确：`2D手绘赛璐璐画风，参考国产动画《一念永恒》，2D多平面摄影，浅景深`

参数堆砌是Seedance画面质量下降的头号原因。用风格参考词代替技术参数。

### 3. 情绪必须动作化
Seedance不理解抽象的"悲伤""恐惧""美丽"。它只理解物理事实：
- 不是"她悲伤"→是"她低头咬牙，眼眶泛红，手指颤抖攥紧衣角"
- 不是"恐怖氛围"→是"阴影中一只眼睛缓缓睁开，瞳孔收缩，呼吸在冷空气中凝成白雾"
- 不是"美丽浪漫"→是"侧逆光穿过飘起的发丝，碎金般的光斑落在锁骨，裙摆被微风掀起一角"

### 4. 动作必须拆分工序
- 错误："她在做饭"
- 正确："她拿起菜刀，刀刃在灯光下闪过寒光，手腕轻压，刀刃切入番茄，汁液渗出沿砧板流淌"

### 5. 光线是第一杠杆
光线决定画面质感的70%：
- 黄金词：侧逆光、轮廓光、丁达尔效应、体积光柱、霓虹光、烛光摇曳、月光漫射、窗光百叶分割、暖黄顶灯、冷蓝夜光、金色余晖、阴影分割脸
- 错误："光线很好""有氛围的光"
- 正确："黄昏侧逆光穿过百叶窗，在她脸上投下条纹阴影，轮廓光勾勒发丝边缘"

## 8类镜头运动对照表

| 类型 | 中文指令 | 速度控制词 | 适用场景 |
|------|----------|------------|----------|
| 推镜 | 镜头慢推/缓推/快推 | slow/gentle/gradual/smooth | 情感聚焦、细节强调 |
| 拉镜 | 镜头拉远/慢拉 | gentle/gradual | 环境揭示、空间展示 |
| 横移 | 镜头右摇/左摇/横移 | smooth/controlled | 跟踪主体、场景扫描 |
| 跟拍 | 正面跟拍/侧面跟拍/贴地跟拍 | controlled/steady | 人物行走、运动场景 |
| 环绕 | 环绕镜头/半环绕/低角度环绕 | slow/smooth | 产品展示、人物肖像 |
| 航拍 | 高空俯拍/无人机视角/鸟瞰 | steady/gentle | 城市大场面、自然风光 |
| 手持 | 手持晃动/轻微手持感 | slight/natural | 纪录片风格、紧张感 |
| 固定 | 固定镜头/静止机位 | locked | 主体动作为焦点 |

速度控制：极慢用barely/imperceptible，慢用slow/gentle，中用smooth/controlled。`fast`是画质杀手——除非动作戏快切，否则不用。

## 时间轴分段写法

在单个AI视频提示词中，用时间戳分段描述可大幅提升Seedance的理解精度：

```
[0-1.2s] 极特写，手指按下金属按键，指尖泛白，暖黄侧光，浅景深
[1.2-2.5s] 慢推镜，近景，主角瞪大眼盯着按键，瞳孔中倒映冷蓝夜光
```

## 负面提示词

每个即梦AI视频提示词末尾必须附加（或在段首统一声明）。全片统一使用以下负面排除：

```
无字幕，无背景音乐，仅保留环境音效和对白；如有台词、对话、旁白，使用默认字体，不透明度为0。
```

## 群体人物预算（致死级）

即梦Seedance 2.0 将每个不同的 `@引用` 计为1个独立角色。集体泛称（"四名老者""几个修士"）不产生 @引用计数——AI 从泛称中无法确定具体人数。

| 错误（集体泛称→人数随机） | 正确（逐人列举→人数精确） |
|---|---|
| `四名老者呈扇形追击` | `@老者A/@老者B/@老者C/@老者D四人呈扇形追击` |
| `几个修士围住他` | `@修士A/@修士B/@修士C三人围住@洛玉祁` |
| `四道人影高速掠过` | `@老者A/@老者B/@老者C/@老者D四道人影高速掠过` |
| `一群弟子在广场练功` | `十二个弟子在广场练功`（无需@引用的路人用精确数字） |

**铁律**：N个确定角色 = N个独立 @引用，一个都不能少。

## 常见陷阱与纠正

**陷阱一：参数堆砌症**
- 错误："8K超高清 HDR 120fps 电影级画质 杜比视界"
- 正确："2D手绘赛璐璐画风，参考国产动画《一念永恒》，2D多平面摄影，浅景深"

**陷阱二：动作模糊症**
- 错误："一个女孩在跳舞然后转身然后微笑然后跑向远方"
- 正确："固定机位：前3秒慢推镜捕捉舞者旋转裙摆动态；后2秒定格回眸瞬间，发丝逆光飞舞"

**陷阱三：角色消失残留症（致死级——角色遁逃/消失/传送场景必读）**
即梦逐镜独立生成，不理解镜头间的叙事连续性。角色消失后，后续镜头若出现 `@角色名` 引用，AI 必然在画面中生成该角色——即使提示词写的是"已消失""空无一物"。

- 错误（镜头A→镜头B→镜头C，角色应在B中消失）：
  - 镜头B："@角色被光茧包裹塌缩成奇点" → AI 不理解"奇点"=消失，角色仍可见
  - 镜头C："黑雾扩散，@角色的身影已完全消失" → AI 看到 `@角色` 就生成角色
  
- 正确写法：
  - 镜头B（消失瞬间）：**用物理化工序描述消解过程**——"茧内人形轮廓急速模糊消解为暗色粒子流，光茧坍缩至针尖大的光点，光点彻底消失在虚空中"。用看得见的物理变化替代抽象概念（奇点/坍缩/维度消失）
  - 镜头C（消失后）：**删除已消失角色的 @引用**，描述纯环境——"黑雾内部纯粹无一物，无任何人形轮廓或身影残留，只有雾与电弧"
  - 镜头D（留白/余韵）：**不提已消失角色**，用空镜+环境叙事

**核心原则**：已消失的角色 = 从该镜起彻底删除其 @引用。用明确的否定物理描述（"无任何人形轮廓""纯粹无一物""空无一物"）替代"已消失""不见踪影"等状态描述。

**陷阱四：手指畸变症（致死级——手部特写/手势/结印/掐诀场景必读）**
即梦对手部解剖结构的理解极弱，含手部动作的镜头极易出现多指/并指/断指/手指数量异常。此问题无法仅靠负面排除完全解决——必须配合正向的物理化手部描述。

- 错误（抽象或裸露的手部描述）：
  - "右手握拳" → AI 不知拳头的解剖结构，随机生成
  - "双手结印高举" → AI 不理解结印手势，手指缠绕畸变
  - "手指按下按键" → AI 随机生成手指数量

- 正确（用道具/遮挡/明确轮廓辅助手部可辨识度）：
  - "右手五指紧握成拳，指节因用力而泛白，拳面青筋微凸" —— 描述整个手部轮廓而非单个手指
  - "双手十指交叉结印高举过头，掌心金色灵力喷涌而出，衣袖在灵力风暴中向上翻飞覆盖手背" —— 用特效和衣袍遮挡部分手指
  - "食指第二指节按下金属按键，其余四指自然蜷曲，指甲盖因用力而泛白" —— 明确手指数量和动作

**手部铁律（3条）：**
1. **每镜含手部动作必须写出精确手指数量**：不写"手指"，写"五指""食指""拇指+食指""双手十指"
2. **能用遮挡就遮挡**：衣袍/袖口/灵力特效/道具/阴影——任何能部分遮挡手指的元素，降低AI暴露手指畸变的机会
3. **手部特写前加缓冲镜**：巨掌/结印/掐诀等无法回避手部特写的镜头，前1镜必须给手部一个中景或全景的预备动作，让AI有空间锚定手部位置和比例

**陷阱五：风格漂移症**
- 错误："赛博朋克风格的中国古代宫殿有龙和机器人又要粉色又要金属质感"
- 正确："蒸汽朋克版紫禁城，黄铜+深红+煤黑色板，赛璐璐画风+国漫场景美术风格"

**陷阱六：道具穿帮症（致死级——道具转移/收回/销毁场景必读）**

即梦逐镜独立生成，不理解镜头间的道具状态变化。道具被收走/转移/销毁后，下一镜若未明确声明道具的消失，AI 可能在画面中残留该道具——即使提示词写的是"刚收走""已消失"。

- 错误（镜A→镜B，道具在A中被收走）：
  - 镜A："@凌无策将小钟收回袖袍之内"
  - 镜B："小钟刚收起，@赵玉林的症状瞬间就好了" → AI 不理解"刚收起"=道具已消失，可能在@赵玉林手中或地上残留小钟

- 正确写法：
  - 镜A（收走瞬间）：**物理化描述收走动作**——"@凌无策五指一翻将小钟收入袖袍，袖口合拢，小钟彻底没入袖内不见踪影"
  - 镜B（收走后）：**删除道具引用，用物理否定描述其缺席**——"@赵玉林双手空空五指张开撑在地面上，画面中无小钟，@凌无策袖袍已合拢，@赵玉林苍白着脸在地上一阵喘息"
  - 镜C（后续镜头）：不再提该道具

**道具连续性铁律（3条）：**
1. **道具转移后，下一镜必须明确新持有者**：不写"小钟被收走"，写"小钟在@凌无策袖袍内，@赵玉林手中空无一物"
2. **道具消失后，用物理否定描述**：不写"小钟已消失""小钟不见"，写"画面中无小钟""双手空空""地面空无一物，仅青石地砖"
3. **关键道具首次出现时绑定角色**：写"@凌无策掌中的小钟"，不写"一口小钟"——明确道具归属，避免AI将道具随机分配给其他角色

**陷阱七：物理障碍穿透症（致死级——角色需跨过/绕过家具/障碍物时必读）**

即梦不理解场景中的物理障碍物（案桌/屏风/栏杆/台阶等）。角色需要绕过或跨过障碍物时，若提示词不描述绕行动作，AI 会让角色直接穿透障碍物。

- 错误（段首无家具声明，镜中无绕行描述）：
  - 段首场景："府衙正厅" 人物位置："@凌无策端坐主位" → 未声明案桌位置
  - 镜中："@凌无策神色一变连忙俯身过去按住@赵玉林" → AI 让@凌无策直接穿透案桌

- 正确写法：
  - **段首场景声明**加入家具布局：`**场景：** 府衙正厅，@凌无策面前横着一张紫檀案桌（宽约三尺长约六尺），案桌后方为主位座椅，案桌前方为厅中青石空地`
  - **段首人物位置**加入相对于家具的空间关系：`**人物位置：** @凌无策坐于案桌后方主位；@赵玉林倒地在案桌前方厅中地面；案桌横亘在@凌无策与@赵玉林之间`
  - **需要跨过障碍的镜头**加入绕行动作：`@凌无策神色一变，从案桌后猛然起身绕至案桌前，俯身过去按住倒在地上的@赵玉林`

**物理障碍铁律（3条）：**
1. **段首必声明关键家具**：场景中阻挡角色移动路径的家具/建筑元素（案桌/屏风/门槛/台阶/围栏），必须在段首场景声明中写出位置和尺寸
2. **人物位置必相对于家具**：不写"@凌无策在主位"，写"@凌无策坐于案桌后方主位"
3. **穿越障碍必写绕行动作**：角色需要到达障碍物另一侧时，必须写"绕过/跨过/从…后起身绕至…前"——不写绕行=AI默认穿透

---

**陷阱八：同框能量穿帮症（致死级——火焰/雷电/真元等远程能量攻击必读）**

即梦在同框画面中无法理解"火焰向某人飞去但尚未到达"的中间态——只要攻击方和被攻击方出现在同一镜头，AI 会将两者之间的火焰/能量直接"连接"到被攻击方身上，造成烧身/击中/穿帮。

- 错误（攻击方和被攻击方同框，火焰在两者之间蔓延）：
  - `固定，全景，黑袍挥袖写实火焰喷涌而出朝四旬男人席卷而去，火舌前端距四旬男人尚有数丈距离` → AI 让火焰直接烧到四旬男人身上

- 正确写法（强制分镜——攻击方释放和被攻击方反应必须分属两个独立镜头）：
  - 镜A：`固定，中景，黑袍正面朝向镜头凌空挥袖写实火焰从双袖喷涌而出向镜头方向猛烈膨胀放射状扩散，黑袍身影在火焰核心化为黑色剪影`（攻击方正面释放，火焰向镜头/画外扩散，被攻击方不出现在此镜）
  - 镜B：`切至，特写，四旬男人面部被突如其来的火光逐层照亮——此刻仅是火焰扩散的反射光尚未有火焰近身，瞳孔猛然收缩...`（被攻击方反应特写，只有火光映面没有火焰本体）
  - 关键：镜A和镜B之间用`切至`明确断开——切断 Seedance 在同框中建立的空间连续性，防止火焰"跟随"到下一镜

**同框能量攻击铁律（2条）：**
1. **攻击方释放时被攻击方不得同框**：火焰/雷电/真元冲击/灵力波动等远程能量攻击的释放镜头中，只保留攻击方一人。被攻击方在画外，下一镜再切入。
2. **被攻击方反应镜只写光不写火**：被攻击方的反应镜头只描写"火光照亮面部/瞳孔反射火焰"等光照效果，禁止出现火焰本体——即使写"火焰在远处"也会被 AI 拉到角色身上。
