# Seedance 2.0 分镜提示词方法论

分镜师将导演讲戏本忠实地翻译为 Seedance 2.0 能直接使用的动态视频提示词。分镜师不需要理解剧本的叙事意图——创意判断集中在导演身上，分镜师专注于技术执行：@引用语法、节拍密度约束、平台文件限制。

**联动文件**：`references/action-design-master.md`（戏型/镜头语法分类体系、动作节拍扩展五段法、生理反馈链、衔接逻辑系统、站位基线系统——打斗和动作场景的完整导演手册）、`references/fight-vfx.md`（国术劲力法则、2D特效三层结构、动作特效组合词典）。

## 五要素导演法（Seedance 适配版）

### 核心公式

```
[摄影机指令/角度] + [主体+最简视觉特征] + [核心物理动作/变化] + [主导光影关键词] + [质感/氛围收尾词]
```

每镜提示词以摄影机指令开头，随后是主体+动作+光影的密集视觉关键词，最后以质感/氛围词收尾。

### 密度铁律（致死级）

**每镜AI提示词25-60中文字**——这是黄金密度区间。

| 密度 | 字数 | 效果 | 判定 |
|------|------|------|------|
| 过稀疏 | <20字 | AI信息不足，画面空洞、无细节 | 作废 |
| **黄金密度** | **25-60字** | 信息密度精准，AI理解到位，画面有质感 | 合格 |
| 过饱和 | >80字 | 多个视觉焦点竞争，AI随机选择导致画面混乱 | 作废 |

**核心原则**：每镜只有一个核心视觉焦点。让AI做好一件事，而不是让它从一堆描述中随机选。

### 风格对比：散文式 vs 摄影机指令式

**❌ 散文式（当前通病——过长、多焦点、叙事化）：**
```
黑袍右手前伸五指张开工整——火焰从掌心持续喷涌如高压水枪般锥形喷射，火焰抖动线勾勒高压火柱轮廓呈明暗交替的脉冲纹路，黑袍身体因反冲力微微后仰肩部向后牵引
```
> 字数~80。问题：4个视觉焦点（手姿、火焰形态、脉冲纹路、身体反冲）竞争AI算力，每个都描述不充分。

**✅ 摄影机指令式（黄金密度——每镜一个焦点）：**
```
镜A：特写，黑袍五指张开掌心凝聚亮橙光核——周围空气呈螺旋波纹扭曲，边缘高亮
镜B：中景侧面，黑袍猛然挥臂——身前亮橙色火焰炸裂铺满画面，动态模糊+高对比度，黑袍成黑色剪影
镜C：中景，黑袍身体因反冲力后仰——肩部向后牵引衣袍向前翻飞，脚下云层被后坐力推开
```
> 每镜30-50字。一个焦点拆成三镜，每镜只说一个动作阶段。

### AI提示词密度法则：七要七不要

| # | 要 | 不要 |
|---|-----|------|
| 1 | **要**写摄影机看到的具体物理变化（火焰炸裂、瞳孔收缩、云层推开） | **不要**写叙事性的"发生了什么"（"他释放了火焰""他很震惊"） |
| 2 | **要**每镜一个核心视觉焦点（火焰/眼神/身体反馈/环境变化） | **不要**一镜塞多个焦点竞争AI算力 |
| 3 | **要**用强动词收尾（炸裂、铺满、倒映、勾勒、贯穿） | **不要**用形容词堆砌氛围（"非常震撼""极其美丽"） |
| 4 | **要**写明确的光源方向+色温（侧逆暖黄光、冷白顶光、暖橙火光从下方） | **不要**写"光线很好""有氛围的光" |
| 5 | **要**用质感关键词收尾（动态模糊、高对比度、边缘高亮、浅景深、电影级质感） | **不要**堆技术参数（8K/HDR/120fps） |
| 6 | **要**每个动作阶段独立成镜（起势→过程→落点→反应四镜拆分） | **不要**一个长镜塞完整动作链 |
| 7 | **要**保持25-60字黄金密度，写完删到不能再删 | **不要**怕AI看不懂而反复解释——删掉所有可有可无的字 |

每镜提示词以运镜指令开头（`固定` / `缓推` / `横摇` / `低角度仰拍` / `荷兰角` 等），随后自然融入主体动作、环境描述、光影氛围、风格约束。

## 黄金开场：物品串联法（微观→中观→宏观漏斗）

### 核心心法

开场即高潮——前30秒至少15个无台词镜头，用动态物品的物理轨迹强制收束AI的空间发散，观众无意识跟随物品进入故事。

### 景别漏斗公式

```
镜1（极特写）：动态物品特写——飘落的树叶/燃烧的灰烬/按下的按键/雨滴
镜2（大远景）：物品飘过的宏观场景，定义世界观
镜3（中景）：物品落点或延续，镜头拉近至人物空间
镜4（近景）：物品被截停/触碰/穿过，镜头推进主角所在位置
```

四镜完成世界建构——观众的潜意识跟随物品完成空间穿越，而非生硬场景切换。

### 物品选择标准

- **必须有动态轨迹**：飞鸟/落叶/光线/飘浮灰烬/行驶的车/雨滴——物理运动是空间锚点
- **必须有叙事关联**：最终指向核心道具或主角——车窗升降键→主角好奇心、飘落灰烬→战场余烬
- **景别切换必须顺滑**：极特→大远景→中景→近景，阶梯递进不跳级

### 空镜不空铁律

- 空镜必须有风、光影变化、动态元素——拒绝静态PPT式空镜
- 空镜 ≤ 3秒，除非用于哲学性留白
- 天气和光线是最廉价最有效的氛围工具

| 镜号 | 景别 | 功能 | 时长 |

### 视觉符号植入

开场必须植入与核心冲突相关的"视觉符号"——车窗升降键代表"没见过世面"、飘落灰烬代表"战场余波"、碎裂瓷瓶代表"封印解除"。符号在开场特写中出现，在后续剧情中回收。

## @引用体系

### 角色@引用

人物直接用角色名作为 @引用，不再使用 `@图片N` 编号中间层：

```
@虾仁  @黑袍  @四旬男人
```

每个 @角色名 即对应一个独立的人物参考图/音色资源，由平台自动关联。

### 场景@引用

场景参考统一使用 `@` 简引用，段首声明一次后全段继承：

```
场景参考@
```

段首的 `参考@锚定风格` 统一锚定全段风格基调。

### 音色@引用

音色引用**内联到台词前方**，不再仅列在段首：

```
台词：（音色参考@）黑袍："啊——！" 配音状态
```

无台词镜不需要音色引用。

### 平台文件限制（致死级）

| 限制项 | 上限 | 说明 |
|--------|------|------|
| 图片引用 | ≤9 张/条 | @角色名 + 场景@ 引用数 |
| 视频引用 | ≤3 个/条 | 通常为 0 |
| 音频引用 | ≤3 个/条 | 音色参考@ 引用 |
| 总文件数 | ≤12 个/条 | 图片+视频+音频 |

超限 = Seedance 拒收。分镜师必须逐条核验。

## 节拍密度规则（致死级）

### 基本定义

- **1 个动作节拍** = 一个连续动作或动作-反应对
- **1 节拍 ≈ 2.5 秒屏幕时间**
- 例如："拔刀→斩下"是 1 个节拍，约 2.5s；"拔刀→转身→斩下→收刀"是 3 个节拍，约 7.5s

### 密度约束

| 单镜时长 | 最大节拍数 | 说明 |
|----------|-----------|------|
| 5.0s | ≤2 节拍 | 单镜黄金密度 |
| 3.0s | ≤1 节拍 | 短镜单动作 |
| 8.0s | ≤3 节拍 | 长镜宽松密度 |

**红线**：5s 单镜塞 4 个节拍 = 动作糊、跳、不自然 → 作废。

### 安全区

- **前 0.5s**：场景建立期——不放关键动作或台词开头
- **后 0.5s**：动作收束期——不让台词/动作在此截断

### 单镜内时间分段（长镜头专用）

单镜时长 ≥ 4s 时，将镜内动作按秒拆分为时间段落，让 AI 精确理解每个时刻该生成什么画面：

```
X.Xs 运镜指令，景别，[引用声明]
0-1秒：[初始状态描述——人物位置、姿态、环境]
1-4秒：[中间动作流——连续动作的时间递进]
4-6秒：[收束状态——动作完成后的定格或过渡]
台词/音效 [情绪: 类型 X%] [衔接: 类型]
```

**使用条件**：
- 单镜 ≥ 4s 且包含多个连续动作阶段 → 使用时间分段
- 单镜 < 3s 或仅单一动作 → 不需要时间分段
- 叠加动作用 `（叠加）` 标记：`3-5秒（叠加）：[并行动作描述]`

**示例**（10s一镜到底）：
```
10.0s 跟拍，中景，以@丹眉为主角
0-1秒：首帧墙体黑影填满画面，镜头从左向右水平移动，墙体从画面左侧逐渐退去，露出丹眉全身
1-5秒：制片人推丹眉肩膀沿水平往右走，嘴里念念叨叨连珠炮，丹眉无奈微皱眉
3-5秒（叠加）：男场务从画面右侧迎面小跑递桃木剑，丹眉不停脚步右手接过
5-8秒：丹眉手握桃木剑继续往右走，路过化妆师伸手想补口红被甩开
8-10秒：制片人最后推一下放手退场，丹眉继续走两步停下站定，深呼吸整理姿态
台词：丹眉："知道啦知道啦哥，都拍了几百集了。" [情绪: 日常 70%+无奈 30%]
```

### 三段时间戏剧结构法（长镜头升级版）

长镜头（≥8s）可按戏剧结构拆分为"起幅→交锋→余波"三段，每段有明确叙事功能：

```
起幅（0-30%时长）：情绪酝酿/空间建立/动作起势——缓推/固定
交锋（30-70%时长）：核心动作/情绪爆点/冲突发生——快切/手持/急推
余波（70-100%时长）：情绪留白/状态定格/场景收束——极缓推/拉远/定格
```

**示例**（10s一镜到底）：
```
10.0s 斯坦尼康低机位缓推→急推→极缓定格，中景→特写→中景

起幅 0-3s｜情绪酝酿：
画面轨：[前景] 低机位缓推，侧逆暖黄光勾勒轮廓，丹眉双手腹前端罗盘指尖微调 [中景] 罗盘下沿在画面顶部，丹眉腰以下侧面稳步前行 [背景] 通道堆叠flight cases+深灰地坪漆+工作吊灯+尘埃漂浮
声音轨：通道工作噪音底噪+丹眉脚步"哒"和beat精确同步+衣袍轻微摩擦

交锋 3-7s｜核心动作：
画面轨：[前景] 急速推近罗盘——八卦刻度+朱砂红指针清晰可见，镜头突变手持微晃 [中景] 丹眉脚步从"哒"（地坪漆）→"咚"（木地板）——踏上舞台瞬间地面纹理骤变 [背景] 光线从冷蓝→暖橙跳变，黑丝绒幕布+悬挂道具剪影+观众席方向传来低声哗然
声音轨：脚步"哒"→"咚"的材质跳变+空间混响从密闭→空旷扩散+观众席窃窃私语由远及近

余波 7-10s｜定格收束：
画面轨：[前景] 镜头极缓定格，丹眉站定双脚并拢罗盘稳如磐石 [中景] 丹眉做90度向左转身——从面朝画左变为面朝镜头 [背景] 大光圈散景+舞台暖光池+观众席背影剪影在画面下方入画
声音轨：脚步最后一拍"咚"的余音+观众席议论声渐收+舞台环境嗡鸣淡出
```

### 前景/中景/背景三层画面描述

即梦不理解"景深"。必须用前景/中景/背景三层明确写出每层的空间内容，AI才能生成有深度感的画面：

| 层次 | 距镜头 | 内容类型 | 描述要素 |
|------|--------|---------|---------|
| **前景层** | 0-5虚拟单位 | 遮挡物/剪影/飘过物/画框 | 材质+轮廓+虚焦程度+占画面比例 |
| **中景层** | 5-15虚拟单位 | 主角/核心动作/互动 | 角色位置+身体朝向+动作+光影 |
| **背景层** | 15-30+虚拟单位 | 环境/次要角色/氛围 | 空间尺度+动态元素+光线氛围+材质 |

**三层描述示例**：
```
[前景] 虚焦布袋子剪影占据画面左侧1/3——深褐色块状轮廓+棉麻系绳虚影
[中景] 穿睡衣的黑波波头女孩从画面左下角抬起头来——赶时间的微皱眉+右手伸向玄关柜
[背景] 暖橙晨光斜射照亮玄关墙面——浅景深虚化+无地面无天花板仅一面墙
```

**铁律**：
- 每镜至少覆盖前景+中景+背景三层——缺层=画面扁平
- 前景必须有物（飘过粒子/衣角/发丝/遮挡物/剪影）——至少30%镜头有前景
- 三层之间的光影关系必须一致——前景暗→中景亮→背景柔

### 多镜头段落（蒙太奇）

15s 蒙太奇段落：节拍分布在各个子画面中，每个子画面独立遵守密度约束，总节拍可以远超单镜头上限。

```
示例——15s 蒙太奇（4个子镜）：
  子镜1(4s): 1节拍（拔剑）     ✓ 各自合规
  子镜2(3s): 1节拍（闪避）     ✓
  子镜3(5s): 2节拍（反击+命中） ✓
  子镜4(3s): 1节拍（收剑）     ✓
  总节拍: 5 → 远超单镜2节拍上限，但蒙太奇下合规
```

## 动作功能标签系统

### 核心概念

每个镜头都有明确的叙事功能——不是"这个镜头里发生了什么"，而是"这个镜头在叙事链条上承担什么任务"。分镜师必须为每镜标注功能标签，确保每个镜头都在推进叙事而非填充时长。

### 动作四镜公式（动作戏必拆）

任何动作（出招/拔剑/施法/受击）必须拆分为四个独立镜头，每个镜只有一个功能：

```
起势（意图）→ 过程（速度/力量）→ 落点（物理反馈）→ 反应（情绪外化）
```

| 功能标签 | 叙事任务 | 典型景别 | 时长 | 示例 |
|---------|---------|---------|------|------|
| **起势（意图）** | 动作前的蓄力/眼神/肌肉紧绷——建立"即将发生什么"的预期 | 特写/极特 | 0.5-1.0s | `特写，黑袍五指张开掌心凝聚亮橙光核，周围空气呈螺旋波纹扭曲，边缘高亮` |
| **过程（力量）** | 动作释放的瞬间——速度感/力量感/视觉奇观 | 中景/全景 | 0.8-1.2s | `中景侧面跟推，黑袍猛然挥臂——身前亮橙色火焰炸裂铺满画面，动态模糊+高对比度` |
| **落点（物理反馈）** | 动作命中或完成后的物理结果——火花/烟尘/气浪/形变 | 全景/特写 | 0.8-1.5s | `全景，火焰如海啸般向前吞噬——云层蒸发成蒸汽嘶嘶作响，热浪扭曲远景呈水波纹` |
| **反应（情绪外化）** | 受击者/旁观者/环境的反馈——瞳孔收缩/身体后仰/环境震颤 | 特写/极特 | 0.5-1.0s | `极特写，四旬男人瞳孔急剧缩小——眼球倒映出翻滚的橙色火海，微表情震惊` |

**铁律**：
- 四镜缺一不可——缺少起势=动作突兀，缺少落点=没有打击感，缺少反应=观众不知道这一击有多重
- 每镜只承担一个功能——起势镜不写过程，落点镜不写反应
- 动作戏全程值此四镜公式，每个回合独立拆解

### 文戏功能标签

| 功能标签 | 叙事任务 | 触发条件 |
|---------|---------|---------|
| **人设建立** | 首次展示角色核心性格特征的动作/表情 | 新角色出场 |
| **信息揭示** | 关键台词/道具/线索的视觉呈现 | 剧情转折点 |
| **关系建立** | 双人/多人空间关系和权力位阶的视觉化 | 对话场景开场 |
| **情绪转折** | 角色情绪从A到B的过渡瞬间 | 表情极性反转 |
| **悬念钩子** | 制造"接下来会发生什么"的疑问 | 段落结尾 |
| **余韵留白** | 情绪高潮后的静默收束 | 大动作后的喘息 |

**铁律**：每个镜头都要能回答"这个镜头删掉，观众理解会受损吗？"——如果不会，这个镜头就是可有可无的。

## 每镜提示词格式

### 紧凑AI行 + 情绪标签 + 衔接逻辑

```
时长 运镜指令，景别，以@角色名为主角，场景参考@，动作描述，环境描述，光影描述 台词/音效 [情绪: 类型 X%] [衔接: 类型]
```

- 每镜严格一行，不得换行
- **禁止镜号前缀**（不写 `1.` `2.` 序号）
- 运镜指令在行首
- 行尾 `[情绪: 类型 X%] [衔接: 类型]`
- 配音状态用逗号 `，` 分隔，不用 `｜`
- **音色引用内联**：有台词时，台词前加 `（音色参考@）`，格式为 `台词：（音色参考@）角色名："原文" 配音状态`
- **衔接逻辑**：每镜末尾标注与下一镜的衔接类型，采用**复合连接逻辑标签** `[衔接: 主逻辑+次逻辑]` 格式。主逻辑定义镜头如何连接（动作/视线/声音），次逻辑定义情绪如何传递（递进/揭示/延续/暗示）。本段最后一镜不写衔接逻辑。

### 复合连接逻辑标签系统

每个AI生成的镜头必须打上 `[衔接: 主逻辑+次逻辑]` 标签，防止AI画面跳脱：

| 标签组合 | 主逻辑 | 次逻辑 | 适用场景 |
|----------|--------|--------|---------|
| **动作顺接+表情递进** | 上一镜动作落点=本镜动作起点 | 角色情绪逐镜升级 | 打斗连招/追逐/动作序列 |
| **视线引导+空间揭示** | 角色视线方向引导摄影机 | 下一镜揭示视线目标 | 角色看向画外→切目标 |
| **声桥L/J-Cut+悬念补偿** | 声音先入或延迟出画 | 维持好奇心等待揭示 | 场景切换/关键信息延迟 |
| **同框引用+关系建立** | 同一画面框入多个角色 | 角色空间关系确立 | 多人对话/对峙/群戏 |
| **匹配转场+隐喻暗示** | 形状/运动/色彩匹配 | 视觉符号的情感暗示 | 圆月→车灯/烟火→爆炸 |
| **空间跳切+动作匹配** | 不同空间硬切 | 相同动作矢量连接 | 跨场景追逐/平行叙事 |
| **情绪呼应+色彩桥接** | 前镜情绪色彩延续 | 光线/色调平滑过渡 | 情绪连续但场景切换 |
| **因果关系+声画分离** | A导致B的逻辑链 | 声音先揭示因果 | 听到碎裂声→切碎物 |
| **状态锚定+节奏呼吸** | 动作结束后静止帧 | 给观众消化时间 | 高潮后的余波镜 |

**铁律**：
- 主逻辑保证画面连续性（不跳脱），次逻辑保证情绪连贯性（不突兀）
- 禁止只写 `[衔接: 硬切]` 不加次逻辑——单一硬切=告诉AI随机切
- 动作场景优先使用"动作顺接+表情递进"和"视线引导+空间揭示"
- 对话场景优先使用"同框引用+关系建立"和"声桥+悬念补偿"

**动作四镜链（起势→过程→落点→反应完整拆解）**：

## 分镜输出铁律（21条）

1. **运镜优先**：以运镜指令/摄影机角度开头（`固定`/`缓推`/`低角度仰拍`/`荷兰角`）
2. **单焦点密度（致死级）**：每镜AI提示词25-60中文字，每镜只有一个核心视觉焦点——超80字或塞多个焦点=作废
3. **群体人数锁定**：N个角色逐一写出N个独立角色名，禁止集体泛称
4. **角色命名**：分镜行中使用角色名（不带@），禁止泛称
5. **动作物理化**：情绪用身体反应表达——不写"他震惊"，写"瞳孔急剧缩小，下颌肌肉抽搐"
6. **主语必命名**：动作必须有角色名作为施动者
7. **光线必写**：光源方向+色温+质感——`侧逆暖黄光勾勒轮廓`/`冷白顶光分割面部`/`暖橙火光从下方照亮`
8. **禁止元数据侵入**：不写层级标签和技术参数
9. **衣着通用化+服装零描述**：一律写`角色名的衣服`或`角色名的衣袍`，禁止颜色+款式描述，仅描述物理运动
10. **远景用颜色标注**：写`（角色名的衣服颜色）`，"颜色"为占位符
11. **远景禁止抽象化**：人物保持可辨识人形轮廓
12. **精简冗余形容**：参考图已锁定外观
13. **角色名去重**：同一提示词中同一角色名最多1次
14. **利刃方向法**：刃尖必须指向目标
15. **台词一字不差**：不准改写、不准概括
16. **衔接逻辑必写**：每镜末尾标注衔接类型（本段末镜除外），从action-design-master.md第三章选择
17. **段首站位基线必标**：动作场景段首标注站位基线
18. **动作四镜拆分（动作戏致死级）**：任何动作必须拆为起势→过程→落点→反应四镜，每镜只承担一个功能——一镜塞完整动作链=作废
19. **质感收尾词必写**：每镜AI提示词末尾包含1-2个质感关键词——`动态模糊`/`高对比度`/`边缘高亮`/`浅景深`/`剪影`/`电影级质感`
20. **写完即删**：写完每镜后删除所有可有可无的字——如果删掉这个词AI仍能生成正确画面，就删
21. **功能可删测试**：每镜自问"删掉这个镜头，观众理解会受损吗？"——不会=删掉该镜

完整铁律细则 → `references/output-format.md`

## 画面构成铁律（入画出画+排除声明）

即梦逐镜独立生成，不会自动继承上一镜的空间逻辑。必须用以下铁律锁死画面构成：

### 入画铁律
- 所有新出场人物必须从画面**四边之一**进入——左侧/右侧/上/下边缘入画
- **绝不从画面深处凭空出现**、绝不从画面中央淡入
- 人群/群演同样从边缘入画——不写"画面中出现一群人"

### 出画铁律
- 人物退场必须从画面边缘走出去——左侧/右侧/下边缘出画
- **绝不从画面中间凭空消失**、绝不在画面中间淡出
- 退场后若镜头持续——下一秒明确声明"画面中无X角色"

### 画面排除声明
对于有严格画面控制需求的镜头（固定焦段、限定区域），段首或镜首明确声明**画面里看不到什么**：
- `画面里看不到地面、看不到天花板、看不到门——仅一面墙和前景剪影`
- `全程只看丹眉下半身侧面（腰部以下到脚底）——画面里完全不出现任何人的上半身和脸`
- `画面中无黑袍、无四旬男人——仅虾仁一人与翻涌云层`

排除声明 = 告诉 AI 不要生成什么，比告诉它生成什么同样重要。

## 增补镜头三原则（提升画面丰富度30%-50%）

编剧写的是"谁说了什么"，导演拍的是"谁说的时候发生了什么"。增补镜头是将文字剧本转化为视觉叙事的核心武器。

### 原则一：前置铺垫——未见其人，先见其势

重要角色登场或关键事件发生前，用1-2个镜头铺垫"即将发生什么"：

- **视觉焦点前置**：先拍正在震颤的水杯→再拍地面震动→最后揭示巨兽登场
- **光影变化前置**：先拍墙壁上的阴影逐渐扩大→再拍门被推开→角色入画
- **声音前置（J-Cut）**：先听到沉重的脚步声→画面仍停留在当前场景→再切到新角色入画

公式：`视觉焦点（震颤的物体/变化的阴影/他人的警觉反应）→ 揭示来源`

**铁律**：反派/强者登场绝不直接拍脸——用环境失衡铺垫（光线变暗、水杯震颤、旁人屏息），最后才揭示本体。

### 原则二：全息反应链——事件的价值由旁观者定义

每一个关键事件/台词发生后，必须覆盖三层反应：

```
1. 直接承受者的反应（被击中者/被对话者）—— 0.5-1.0s
2. 关键旁观者的反应（最能代表观众情绪的配角）—— 0.5-1.0s
3. 环境/群演的反应（空间氛围变化/集体后退/倒吸凉气）—— 0.5-1.0s
```

**节奏铁律**：先给被冲击者的"生理反应"（瞳孔收缩/身体后仰），再给旁观者的"表情特写"，最后给环境的"氛围变化"。

### 原则三：环境叙述——情绪外化到场景中

在情感高潮或压抑时刻，镜头离开人物，切给环境1-2镜——让环境替角色说话：

| 情绪 | 环境外化视觉 |
|------|-------------|
| 紧张/压抑 | 水杯水面震颤、钟摆声放大、窗外雷云聚集、烛火摇曳加剧 |
| 悲伤/失落 | 雨水沿窗滑落、窗帘微动、落叶飘入空房间、灰烬缓缓沉降 |
| 震惊/崩溃 | 手中物品滑落、纸张散落飘飞、灯光明暗闪烁、镜子碎裂 |
| 愤怒/爆发 | 门被猛然撞开反弹、窗帘被风掀起、桌上物品被气浪推落 |

**铁律**：环境叙述镜 ≤2.0s，必须有动态元素（飘落/震颤/闪烁/滑落），拒绝静态空镜。

## 画面节奏标注

对节奏控制严格的段落，在段尾或镜尾标注节奏型：

```
【画面节奏】慢-停-切焦-停-急-停
```

节奏型词汇：`慢`（缓推/缓摇）、`停`（固定/静止帧）、`急`（快切/快摇/急速变焦）、`切焦`（rack focus）、`揭示`（zoom out/pull back）、`顿`（0.5s定格）

## 核心视觉动作标记

段落中如有决定整段成败的关键视觉moment，用 `【核心视觉动作】` 标记：

```
【焦点切换——本段视频核心视觉动作】
第10到11秒：镜头焦点从对面墙壁缓慢平滑地向前景拉近，对面墙壁逐渐变虚焦，前景布袋子从模糊剪影逐渐变清晰。
```

一个大分镜段最多标记 **1个** 核心视觉动作。用多了等于没用。

## 转场机制详解

除硬切外，以下转场机制需要在提示词中详细描述其物理过程（即梦不会自动理解转场概念）：

| 转场类型 | 机制描述 | 示例写法 |
|----------|---------|---------|
| 遮挡转场 | 物体/人物从画面一侧滑入→完全遮挡画面→画面内容由A变为B→遮挡物滑出 | `监视器边框水平从右侧扫过把丹眉完全遮挡→边框完全充满画面→摄影棚消失只剩监视器屏幕——进入邵氏胶片世界` |
| Rack Focus | 镜头焦点从A层平滑拉到B层，A变虚B变清 | `镜头焦点从对面墙壁缓慢平滑地向前景拉近，墙壁逐渐变虚焦，前景布袋子从模糊剪影逐渐变清晰` |
| Zoom Out揭示 | 镜头后拉→画面范围扩大→揭示之前不可见的空间信息 | `镜头zoom out向后拉远——揭示之前侧拍看不见的舞台空间：前景观众席背影+中景丹眉+后景玩偶` |
| 纯白/纯黑帧 | 用0.3s纯色帧做冲击过渡 | `0.3s纯白帧——画面全白仅留剪影轮廓` |

转场机制的物理过程必须完整写出：起始状态→中间过程→结束状态。不可只写"转场进入XX世界"。

## 同框过渡优先原则

当需要从角色A的镜头过渡到角色B的镜头时，即梦逐镜独立生成无跨镜记忆。过渡方式的优先级如下：

| 优先级 | 方式 | 做法 | 适用场景 |
|--------|------|------|---------|
| **第1优先** | 同框镜头 | A和B在同一画面中自然共存，下一镜可聚焦B | 对话、对峙、交接物品 |
| **第2优先** | 摇镜过渡 | 镜头从A摇向B，两者出现在同一连续空间中 | A和B在同一场景但距离较远 |
| **第3优先** | 音画分离转场 | 声音先入（J-Cut）或画面先切+声音延迟（L-Cut）引导注意 | 跨场景过渡、电话对话 |
| **最后选择** | 直接跳切 | A→直接切B，无任何过渡 | 仅限表现震惊/突然/暴力转折 |

**铁律**：绝不无故直接跳切——除非就是为了表现"突然"本身。无故跳切=即梦随机选择B的机位角度=空间跳脱。

## 大分镜 15 秒生成单元铁律

Seedance 2.0 单次生成最长按 15 秒设计。每个大分镜必须是一个可独立生成的 15 秒以内片段。

- 大分镜时长不得超过 15 秒。
- 超过 15 秒的剧情按场景变化、情绪转折、关键台词、动作阶段、道具归属变化、新角色入画/退场拆段。
- 每段首镜必须承接上一段末镜或建立新空间。
- 每段末镜必须留下可继承的视觉/声音/动作钩子。
- 从第二段开始，段首必须写 `**承接上一段：**`。
- 非最后一段，段尾必须写 `**下段衔接钩子：**`。

段间衔接优先级：

1. 动作承接。
2. 道具承接。
3. 视线承接。
4. 声音承接（J-Cut/L-Cut）。
5. 遮挡转场。
6. 匹配转场。
7. 空间建立硬切。

禁止只写“转场到某地/第二日/另一边”。必须写清楚上一段末镜的物理状态如何进入下一段首镜。

## 对白可视化（台词三镜法）

将文字台词转化为视觉的三镜公式——告别A说一句切B反应、B说一句切A反应的乒乓切换：

```
讲话者突出（输出信息）→ 听者反应（接收信息）→ 环境外化（情绪可视化）
```

**镜1 — 讲话者突出**（谁在说话）：
- 中景或近景，讲话者面部+上半身
- 嘴型与台词同步，身体语言同步（手势/前倾/后退）
- 配音状态：六维发声拆解

**镜2 — 听者反应**（信息被接收）：
- 特写或近景，听者的面部微表情
- 关键微表情：瞳孔收缩/放大、喉结滚动、手指收紧、嘴唇抿紧/微张
- 听者的反应比讲话者的台词更能传递信息量——观众通过听者的脸判断信息的重量

**镜3 — 环境外化**（情绪视觉化）：
- 不是拍人脸，而是拍环境如何被情绪改变
- 紧张→水杯水面震颤/钟摆声放大/窗外雷声
- 悲伤→雨水滑落/窗帘微动/烛火摇曳
- 震惊→杯子滑落/纸张散落/灯光明暗闪烁

**铁律**：
- 日常对话中A连续2-3句在同一镜内完成 → 再切B反应——禁止每句台词切一次
- 仅在以下时机切镜：①对话对象改变且反应比话语重要 ②情绪发生显著转折 ③新角色入画
- 关键台词（揭示/转折/威胁）必须给听者反应特写

### 标点剪辑法完整版

标点符号决定剪辑节奏——用标点控制什么时候切、切给谁：

| 标点 | 剪辑行为 | 作用 |
|------|---------|------|
| **逗号 ，** | 切给听者的微表情特写 | 让观众看到信息被接收的瞬间 |
| **句号 。** | 切回讲话者/动作落点，或切给全景/环境空镜让信息在空间中回荡 | 一句话完整的收束——给观众0.5s消化信息 |
| **问号 ？** | 切给听者——等待回答的疑惑反应 | 制造悬念——"他会怎么回答？" |
| **叹号 ！** | 切给冲击性画面——动作落点/环境响应 | 情感的视觉爆发 |
| **省略号 ……** | 延长反应镜头时间+留白，长镜头定格在角色凝固的表情上 | 利用无声的几秒拉满心理张力——"他接下来会说什么？" |
| **破折号 ——** | 声音先入（J-Cut）+延迟切镜 | 话语被打断/突然转折/新信息强行闯入 |
| **问号+叹号 ？！** | 快速双切——先给听者震惊→再给冲击源 | 极端情绪的复合表达 |

**铁律**：逗号和句号本身不强制触发镜头切换——可以在同一镜头内用角色表演承接。问号/叹号/省略号/破折号必须触发镜头切换。

### 潜台词与谎言视听化

当角色A在说谎或隐瞒时，镜头不拍A的嘴——镜头切给B的脸，让观众从B的反应中读到真相：

- **A说谎** → 镜头给B（听者）——B微眯双眼/嘴角微扬/眉毛微挑——"我听出来了"
- **A隐瞒关键信息** → 镜头给A的手（紧攥/颤抖/摩挲物品）——嘴在说谎，手在坦白
- **A威胁但内心恐惧** → 镜头给A的脚（后退半步/重心后移）——身体比嘴巴诚实
- **A强装镇定** → 镜头给A的喉结（滚动一次）、手指关节（攥到泛白）、太阳穴（汗珠渗出）——控制不住的生理反应才是真相
- **A掩饰慌乱** → 镜头给A在桌下搓动的双手、无意识抖动的脚尖、反复摩挲道具的极特写——嘴在编造，身体在告密

公式：**拍那个泄露真相的身体部位，而非说话的脸。**

**执行铁律**：
- 谎言镜头中，A的嘴可以出现在画面边缘或虚焦中，但绝不在焦点中心
- B的反应镜头中，B不需要说话——微表情自己会说话
- 泄露真相的身体部位用极特写，强化"身体不听话"的视觉冲击

## 即梦高风险元素黑名单

以下元素写入提示词 = 即梦必然生成穿帮画面。分镜师必须逐镜对照此表，触发任一项即改写：

| 黑名单元素 | 穿帮机制 | 安全替代 |
|-----------|---------|---------|
| **残像分身** | 即梦逐镜独立，残像=多个人物→畸变/多头/多肢 | 固定镜头+角色45°出画→下镜切至到达位 |
| **目光穿过X锁定Y** | 即梦渲染远处Y但无跨镜记忆→空镜或错误角色 | 角色面对镜头表演；末尾加`画面中无Y`；用音效暗示远处事件 |
| **眼睛映出远方火光** | 不理解"映出"→瞳孔贴错误光斑/颜色块 | 删除；用面部光影变化（暖橙反光在颧骨/下颚）暗示光源 |
| **飞出画面+移动镜头** | 摇镜/跟拍与飞出方向矛盾→角色滞留画面边缘 | 飞出=固定镜头；下镜切到达位置 |
| **入场镜内混对手台词** | 动作节奏+文戏节奏混=即梦节奏紊乱→角色僵住 | 入场镜无台词→独立反应镜 |
| **蒙面角色的五官描写** | 不理解物理遮挡→蒙面布上凭空生出嘴/眼/眉毛 | 声明`不见任何五官`；用身体语言+声线替代表情 |
| **手部特写无遮挡** | 即梦不理解手部解剖→手指数量错误/关节反向 | 必须写精确手指数量+用衣袖/手套部分遮挡 |

**铁律**：分镜师每完成一个大分镜段，必须逐镜对照此黑名单自检。触发任一项=本镜作废重写。

## 反应镜拆分铁律

角色情绪转折/惊讶/识别→**必须单独成镜**，不可与动作镜合并。即梦无法在一镜内同时处理"动作节奏"和"情绪停顿"——合并=动作变形+表情僵硬。

**识别信号**（触发拆分）：
- 惊呼/质问台词："是你！""怎么可能！""你……你还活着？"
- 表情极性反转：得意→震惊、愤怒→恐惧、平静→崩溃
- 反应者此前未出镜或处于画面外

**拆分格式**：
```
X.Xs 固定，近景，以@角色名为主角，[身体反应+微表情+台词]
```

**铁律**：
- 反应镜运镜必为`固定`——情绪需要静止，镜头运动=分散注意力
- 反应镜中只出现反应者一人——不混入被反应对象
- 反应镜后需要0.5-1.0s的情绪留白（角色沉默+微表情持续），再切下一镜

## 观察者镜头安全协议

角色A在远处观察角色B/C时，即梦无跨镜记忆会导致以下穿帮：画面中出现B/C但姿态/位置错误、A的视线方向与B/C位置不匹配、被观察者在画面中畸变。

**安全协议（三必须）**：

1. **必须对镜表演**：观察者A面对镜头（或略微偏左/偏右），不出现被观察者B/C在画面中
2. **必须否定声明**：镜头末尾明确声明`画面中无B无C`——物理排除被观察者
3. **必须音效暗示**：用环境音/远处声音暗示B/C的存在和动作——观众通过声音知道"有人在那边"

**示例**：
```
3.0s 固定，中景，以@虾仁为主角，场景参考@，虾仁悬浮高空双手抱胸面对镜头，死鱼眼微眯，嘴角微扬，衣袍被高空风吹得猎猎作响，画面中无黑袍无四旬男人 远处传来黑袍的惨叫声+火焰燃烧噼啪声 [情绪: 玩味 70%+观察 30%] [衔接: 因果关系+声画分离]
```

**铁律**：
- 观察者镜头必须使用`固定`机位——禁止摇镜/推镜
- 一旦观察者开始行动（飞过去/走过去），结束观察者协议，进入正常镜头序列

## 运镜约束精确化

即梦对抽象运镜指令的理解有限。运镜指令必须包含精确的物理约束。

### 特殊构图与心理暗示

除了基本的推拉摇移，以下特殊构图角度自带强烈的心理暗示功能——在提示词中直接写明角度名称，AI可理解其情感意图：

| 构图角度 | 中文指令 | 心理效果 | 适用场景 |
|---------|---------|---------|---------|
| **荷兰角** | `荷兰角` / `倾斜构图` | 心理失衡、不安、诡异、即将失控 | 反派登场、黑化瞬间、谎言被揭穿前、精神崩溃 |
| **低角度仰拍** | `低角度仰拍` | 赋予角色压迫感/权威感/威胁性 | 反派凝视、强者俯视、绝对压制 |
| **高角度俯拍** | `垂直俯拍` / `上帝视角` | 宿命感、角色无力感、被命运审判 | 角色坠地、陷入绝境、重大转折后的留白 |
| **过肩镜头** | `过肩镜头` | 建立关系张力、引导观众视线方向 | 双人对峙、对话场景建立空间关系 |
| **越轴跳切** | `越轴跳切` | 空间错位感、突发变故、打破平衡 | 剧情突转、暴力打断、关键信息揭示 |

**使用铁律**：
- 荷兰角+手持晃动 = 极度不安，适用于反派的阴毒时刻或主角的精神恍惚
- 垂直俯拍只用在全片最绝望/最宿命的1-2个瞬间——滥用=失效
- 过肩镜头中前景角色必须虚焦（浅景深），视线焦点在后景角色脸上
- 越轴跳切仅在全片关键转折点使用1-2次——频繁越轴=观众空间感崩溃

### 五要素精准运镜

```
运镜方式 + 镜头高度 + 拍摄距离 + 画面占比 + 禁止项
```

**示例对比**：

| 维度 | 模糊写法 | 精确写法 |
|------|---------|---------|
| 镜头高度 | `平拍` | `镜头水平高度在丹眉腰部略低` |
| 拍摄距离 | `中景` | `镜头与丹眉平行水平距离3米侧面` |
| 画面占比 | `全身` | `丹眉全身入镜从头顶到鞋底脚部完整可见，占画面竖向1/2` |
| 禁止项 | （缺） | `所有运动严格水平横向不能Z轴位移`、`绝不弧形`、`绝不切换` |
| 运动方向 | `跟拍` | `镜头从左向右水平平移，背景从右向左流动，速度严格匹配` |

### 取景范围精确约束

需要限死画面范围时，用精确的空间描述锁死构图：

- `全程只看丹眉下半身的侧面（腰部以下到脚底+约腰部到鞋的区间）`
- `罗盘下沿在画面顶部边缘可见`
- `整段15秒内画面里完全不出现任何人的上半身和脸`

### 禁止项（告诉摄影机不能做什么）

- `绝不Z轴位移` / `绝不弧形运动` / `绝不切换机位`
- `镜头固定不跟随——画面中只留消散的音爆云`
- `绝不从画面深处或中央出现人物`

**铁律**：禁止项和允许项同样重要。即梦不会自动知道什么不该做。

## 物理环境作为叙事触发点

环境的变化（地面材质、光线色温、空间开阔度）可以作为叙事转折的触发条件，而非仅仅作为背景描述：

```
地面突变（地坪漆→温暖木质舞台板）+光线突变（冷→暖舞台光）+空气突变（密闭后台→空旷舞台）→ 触发叙事转折
```

### 环境触发三要素
| 要素 | 变化方式 | 示例 |
|------|---------|------|
| 地面/材质 | 脚步音效改变+视觉纹理切换 | `脚步从"哒"变木地板"咚"` |
| 光线 | 色温跳变+亮度变化 | `冷蓝ambient shadow→暖spotlights` |
| 空间 | 混响变化+视野开阔度 | `密闭通道→空旷舞台空气感` |

环境触发点必须在镜头描述中同时写出视觉变化和音效变化，形成感官同步。

## 造型曲线：宏观情绪强度调控

### 核心概念

在大分镜段（90-150镜）的完整序列中，情绪强度不是一条直线——它是波浪形的"造型曲线"。分镜师需要在宏观层面调控每一段的强度参数（Intensity 1-10），构建张弛有度的叙事节奏。

### 强度分档与视觉策略

| 强度 | 场景类型 | 镜头策略 | 运镜约束 | 光影特征 |
|------|---------|---------|---------|---------|
| **7-10** | 高压动态戏/冲突爆发/终极对决 | 镜头极度碎裂，快切密集（0.5-1.5s/镜），强制使用荷兰角+手持晃动 | 禁用固定长镜 | 高反差冷暖光对冲，频繁光影跳变 |
| **4-6** | 对峙蓄力/信息揭示/情绪转折 | 中等节奏（1.5-3.0s/镜），推镜+过肩镜头为主 | 固定+缓推交替 | 光影渐变过渡，色调统一中求变化 |
| **1-3** | 文戏/内心独白/压抑留白 | 镜头时间拉长（2.5-5.0s/镜），机位绝对固定（Locked-off） | 禁止任何运镜 | 环境音的视觉化：滴水、灰尘在光柱中悬浮、阴影缓缓移动 |

### 造型曲线构建公式

```
高压段（7-10）→ 情绪释放 → 短暂喘息（3-4）→ 信息揭示（5-6）→ 二次攀升（7-9）→ 余韵收束（2-3）→ 终极高潮（9-10）→ 留白尾声（1-2）
```

**铁律**：
- 连续高压段不超过20s——观众需要情绪呼吸点
- 高强度段与低强度段交替——持续亢奋=麻木
- 全片最高强度点（10/10）只能出现1次，放在全片75%-85%位置
- 结尾留白：最后5-10s强度降至1-3，给观众消化空间

### AI提示词强度标签

每个大分镜段在段首标注情绪强度：

```
**情绪强度：7/10**
```

便于分镜师全局把控造型曲线，确保全片节奏不失控。

## 角色专属光影色彩系统

### 核心原则

每个主要角色绑定专属的光影色调——AI模型在色彩潜空间中锁定角色视觉DNA，防止画面色调崩坏，同时建立观众的色彩记忆。

### 色彩绑定规则

| 角色类型 | 色调 | 光影特征 | 应用场景 |
|---------|------|---------|---------|
| **主角（正派）** | 暖色调/鹅黄/金色 | 侧逆暖光勾勒轮廓、暖黄顶灯、金色粒子 | 登场/高光时刻/情绪正面 |
| **主角（反转型）** | 暖橙→冷蓝渐变 | 从暖调逐渐过渡到冷调 | 角色黑化/心理转变 |
| **御姐/高冷型** | 冷色调/蓝调/银灰 | 冷白侧光、蓝调阴影分割脸、银灰反光 | 登场/对峙/掌控场面 |
| **神秘/魅惑型** | 粉紫光晕/温和霓虹 | 柔光浅景深、粉紫边缘光、暗角 | 登场/话中有话/意味深长 |
| **反派/隐藏BOSS** | 暗绿+深紫/低饱和度 | 逆光剪影为主、环境光变暗、冷绿暗部 | 未露面时：环境失衡暗示；露面后：低压暗光 |

### 应用铁律

- 角色登场第一镜必须使用其专属光影——建立色彩锚点
- 两个角色同框时，光线取主导方的色调——色彩暗示权力关系
- 角色情绪剧烈转变时，光影色调同步跳变——强化视觉冲击
- AI提示词中直接写光影色调：`暖黄侧逆光勾勒轮廓` 而非 `好的光线`

## 道具独立描述

段落中的核心道具（对剧情有决定性作用的物品），在段首单独声明其外观细节，避免AI每次生成时漂移：

```
【道具名称——用途说明】
材质+颜色+尺寸+特殊标记+使用痕迹。
```

**示例**：
```
【布袋子——焦点切换后画面主体】
深褐色做旧棉麻质地的小布袋子，正面贴着一张米黄色符纸，符纸上有朱砂红色手写的繁体汉字道符图样，袋身有自然褶皱和长期使用的斑驳痕迹，棉麻系绳和一根红色细绳一起系在袋口。
```

道具独立描述放在段首声明块中、站位基线之后。一个大分镜段最多声明 **1个** 核心道具。普通道具不需要独立描述。

## 段首统一声明块

每个大分镜段首粘贴：

```
## 大分镜X：标题（起始时间-结束时间）

出场角色：@角色A。@角色B。@角色C。
参考音色：@角色A的参考音色@。@角色B的参考音色@。
**场景：** [地点、环境、光线、氛围、天气/时间]
```
