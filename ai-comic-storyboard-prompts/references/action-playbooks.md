# 徒手徒脚肉搏专项案例页

## 1. 使用范围

给 `拳 / 掌 / 肘 / 肩 / 腿 / 膝 / 摔 / 擒拿 / 借力反制 / 身法换位 / 群像徒手围攻` 提供稳定写法。使用时先判断肉搏子类，再写动作 DNA、接触点、衣料/脚步/场景反馈和结果变化，避免所有徒手戏都变成同一种“拳打脚踢”。

## 2. 徒手徒脚肉搏总分类

| 子类 | 核心距离 | 动作 DNA | 必写反馈 | 易错 |
|---|---|---|---|---|
| 拳掌短打 | 近距 / 贴身 | 中线、短促、直进、连消带打 | 衣料凹陷、肩背短震、脚跟离地 | 变成大摆拳、手臂乱甩 |
| 重拳重掌 | 中近距 | 蓄力、沉肩、腰胯带动、爆发 | 地面水花、尘土、木门震动 | 只有气波，没有身体接触 |
| 肘肩靠打 | 贴身 | 小距离闯入、贴身一震、肩背发力 | 对方侧步失衡、墙面落灰 | 写成大幅冲撞或无接触推飞 |
| 腿法脚法 | 中距 / 近距 | 脚掌支撑、髋部带动、低位截线或直线蹬出 | 下盘路线被截、地面水花、衣摆被带起 | 高踢乱飞、脚底滑行 |
| 膝法贴身 | 贴身 | 控距、提膝、短促上顶、身体压近 | 对方上身一滞、衣料震动、呼吸中断 | 精确攻击危险部位 |

| 倒地翻滚起身 | 地面 / 低位 | 受力落地、翻滚卸力、撑地起身 | 水花、尘土、衣料摩擦 | 无重力漂浮 |
| 群像徒手围攻 | 多方向 | 波次、主线、二级碰撞、外围反应 | 直接受力者最大，外围只停步遮挡 | 一拳全员飞退 |

## 3. 徒手徒脚通用动作公式

```text
起势：脚掌抓地 / 重心压低 / 肩肘收住 / 髋部预转
抢位：贴近 / 侧移 / 截步 / 走圈 / 堵住退路
出招：拳 / 掌 / 肘 / 肩 / 膝 / 腿 / 摔 / 擒拿中的一个主动作
闭合距离：先进到贴身 / 半臂距离，拳掌与目标衣料之间不能留明显空隙
接触：落在衣料、护臂、上身广义区域、肩背、下盘路线或场景物件
短停：0.1-0.18 秒，观众看清接触点
HIT特写：快速连击后的最后一击切近景 / 特写，画面只看拳掌压进衣料的一瞬间，拳掌与衣料有真实重叠、遮挡、压扁
反震：衣料凹陷、肌肉短震、脚跟离地、呼吸一顿、脸颊肌肉短促挤压且五官稳定
力道视觉化：身上水滴、灰尘、雨丝从接触点炸开，水波状力道纹和空气振动短促外扩
传导：水滴、灰尘、衣摆、发丝、灯笼、桌椅、门板、地面裂纹、碎石
结果：退、飞退、撞物、撞碎明确物件、撑地、包围圈重排、出口出现
```

### 3.1 三帧接触锁定

徒手徒脚肉搏的打击感优先级高于招式复杂度。只要用户反馈“没有物理接触 / 像隔空打人”，先按三帧锁定写，不急着加群像、飞退和大场景破坏。

```text
入距帧：角色先进到贴身 / 半臂距离，拳掌离目标衣料只剩几厘米，接触前不出现气波。
接触帧：拳面 / 掌根 / 肘肩压进衣料或护具，出现真实重叠、遮挡、压扁、受力褶皱，短停 0.12-0.2 秒。
反弹帧：衣料凹陷后弹回，肌肉短震，水滴灰尘从接触点弹开，对手身体先滞住再移动。
```

写 Seedance 肉搏时，所有水波力道纹、空气振动、飞退、撞物、碎石和场景破坏，都必须排在接触帧之后；不能用它们替代接触帧。

### 3.2 动漫国漫肉搏 / 真人肉搏表现分流

同样是“拳拳到肉”，动漫国漫版和真人版不能混用一套特效词。

| 表现模式 | 可以使用 | 默认避免 | 核心判断 |
|---|---|---|---|
| 动漫国漫肉搏 | 炁流、红青气浪、墨色游龙、水墨粒子、残影叠影、速度线、夸张冲击波、地面夸张崩裂 | 真人实拍质感、纯摄影机写实、过弱物理反馈 | 先让打斗爽感成立，再用接触点和攻守互换兜住动作 |
| 真人肉搏 | 风压、雨丝偏移、衣摆掀动、水滴炸开、灰尘、肌肉短震、衣料凹陷、摄影机微震、动态高斯模糊 | 炁流、水墨游龙、全屏粒子、二次元脸、动画速度线 | 先让真实接触成立，再用环境反馈放大力量 |

动漫国漫双人肉搏适合用户明确要求“一人之下级、国漫画风、中式武侠写实风、水墨粒子、炁流、动画级运镜、势均力敌、各有千秋”。它不是低强度写实测试，而是高燃对招模式：

```text
开场即交锋 -> 双方风格对照 -> 高速攻防互换 -> 肢体相抵 / 命中 hit-stop -> 炁流 / 速度线跟随力道 -> 近身拳脚搏杀 -> 双方脱离重组
```

动漫国漫肉搏可以写红墨相交气流、青墨相交气流、墨色游龙、残影叠影和夸张地面裂纹，但必须保留肉搏底层逻辑：

- “拳脚对撞”不是武器对撞，接触点应写拳锋相撞、掌根架住、前臂格挡、膝肘相抵、肩靠短震、脚掌截线。
- “劈、砍、刺、挑、砸”不是冷武器专属，而是力线方向 / 发力形态；但在徒手提示词里不能裸写，必须补身体部位和接触方式，写成劈掌、斜劈拳、刺拳、挑肘、挑掌、砸拳、下砸肘、肩砸、膝撞等，避免模型误召刀剑斧棍。
- 红色气浪 / 青色气浪分别绑定两名角色，表现两种风格，不随机换色。
- 一人刚猛爆裂，一人灵动迅捷，但每个镜头都要攻守互换，不能一方持续碾压。
- 残影和速度线不能遮住拳脚接触点、脸部表情、手臂格挡线和脚步。
- 冲击波、碎石、烟尘、炁流必须在接触帧之后出现，不能替代物理接触。

徒手版方向动词映射：

| 裸动词 | 肉搏可用写法 | 冷武器可用写法 | Seedance 注意 |
|---|---|---|---|
| 劈 | 劈掌、斜劈拳、下劈腿 | 劈刀、劈斧、劈棍 | 肉搏写手掌 / 拳臂 / 腿线接触点 |
| 砍 | 砍掌、手刀横砍、前臂横砍感 | 刀砍、斧砍、短兵横砍 | 肉搏慎用，优先写劈掌 / 手刀，避免召唤刀具 |
| 刺 | 刺拳、直拳点入、指节点打 | 剑刺、枪刺、匕首刺 | 肉搏写拳面 / 指节 / 掌根，不写危险部位 |
| 挑 | 挑肘、挑掌、膝线挑起 | 剑挑、枪挑、刀挑 | 肉搏写挑开手臂 / 挑起防线，不写挑伤身体 |
| 砸 | 砸拳、下砸肘、肩砸、膝撞 | 锤砸、斧砸、棍砸 | 肉搏写衣料凹陷、肩背短震和脚步承压 |

可直接复用句式：

```text
这场是动漫国漫高燃双人肉搏，不是真人实拍版。两名侠客站在城楼之上正面激斗，势均力敌、各有千秋：一人刚猛爆裂，拳脚发力带出红墨相交气浪；一人灵动迅捷，身法换线带出青墨相交气浪。开场无铺垫，拳锋 / 前臂 / 掌根真实相撞，接触点先出现，再爆发红青气浪、墨色气浪、速度线和短促冲击波，0.12-0.2 秒 hit-stop。随后高速攻防互换：一方强攻，另一方格挡 / 让线 / 反击，再被对方化解；每 2-3 招必须有一次清楚的肢体相抵、反震和脱离重组。残影、速度线和动态模糊只服务高速动作，不遮住拳脚接触点。
```

## 4. 拳掌短打

适合：咏春、形意、拳击短连击、街巷近战。

写法重点：

- 拳掌走直线，目标明确。
- 手肘收住，不做大开大合摆臂。
- 命中点小，反馈深。
- 必须先贴近到半臂距离，再出短拳；拳掌与衣料之间不能有明显空隙。
- HIT 特写必须看见拳面 / 掌根压住衣料，形成轻微遮挡、压扁和褶皱。
- 连击不能糊，每一击都要有一个接触点和最小物理反馈；轻击反馈小，重击反馈大。

Seedance 句式：

```text
角色先压住重心，肩肘收住，前半拍几乎不动，下一瞬先贴近到半臂距离，拳掌从中线短促打入。快速连击 2-3 下，每一击都必须拍到拳面 / 掌根压到对手衣料或护具上，拳掌与衣料之间不能留明显空隙；轻击也要出现衣料小凹陷、水珠弹开、肌肉短震、脚步停顿或格挡受力。最后一击切 hit 画面特写，命中点只落在对手上身衣料区域，拳掌压进衣料，形成真实遮挡、压扁和受力褶皱，0.15-0.2 秒短停；衣料向内凹陷再弹回，肩背和手臂肌肉短震，身上水滴和灰尘从接触点炸开。接触点出现一圈短促水波状力道纹，空气轻微振动，衣摆和雨丝向外掀开。对手先滞住，再被重击飞退，撞碎身后一个明确物件；碎屑、水滴、灰尘沿人物飞退方向或撞击反弹方向运动，地面只在脚掌蹬地、身体落地或撞击传导处出现短裂纹，不凭空出坑。
```

资产要求：

- 服装必须有衣料褶皱、湿尘状态和凹陷弹回状态。
- 场景必须有可撞物：木门、桌椅、摊位、灯笼、墙面，并指定一个可被直接受力者撞碎或撞裂的明确物件；破坏方向必须跟随人物飞出方向和物理惯性。

## 5. 重拳重掌

适合：八极、洪拳、南拳、降龙类高武掌法、boss 压迫。

写法重点：

- 先写蓄力，不要一上来就飞。
- 发力从脚底、腰胯、肩背传到拳掌。
- 命中后场景要接住力量。

Seedance 句式：

```text
角色脚掌重重踩实，水花向外一震，腰胯先沉，肩背随后压住力量。下一拍重拳 / 重掌打入，对手上身衣料瞬间凹陷，身体先停住，再被整股力量推得飞退撞上木门；门板剧震，门缝落灰，地面短裂纹和碎石向外跳起。
```

禁止：

- 不写生理损伤。
- 不写具体易伤部位。
- 不用“脸部变形”，改写为“受力表情被挤压，五官保持可识别”。

## 6. 肘肩靠打

适合：八极贴山靠、贴身制服、狭窄空间、电梯 / 走廊 / 门框边。

写法重点：

- 距离必须贴身。
- 肩 / 肘不是远距离攻击，而是闯入后的短震。
- 结果常常是侧步失衡、撞墙、路线被挤开。

Seedance 句式：

```text
角色先用小碎步闯入贴身距离，肩肘收紧，身体贴近对方动作线。下一瞬肩背短促一靠，接触点落在对方上盘衣料和肩侧广义区域；衣料震动，墙面灰尘落下，对方侧步失衡撞到门框，身后的同伴因路线被堵停住。
```

## 7. 腿法脚法

适合：北腿、戳脚、跆拳道、泰拳、街斗截步。

写法重点：

- 先写支撑脚，腿法才有重量。
- 优先写低位截线、正蹬、侧踹、扫腿改变路线。
- 高踢只在角色服装、空间和风格允许时使用。
- 命中反馈不写精确危险部位，用“下盘路线 / 身体重心 / 上身广义区域”。

腿法类型：

| 类型 | 核心动作 | 结果 |
|---|---|---|
| 低位截步 | 脚掌卡住来路 | 对方步伐被截，重心断开 |
| 正蹬 / 侧踹 | 支撑脚抓地，髋部推出 | 对方被蹬退，衣料和水滴反震 |
| 扫腿 | 身体压低，腿线横切 | 对方下盘失衡，脚步乱掉 |
| 连环腿 | 一脚逼停，一脚改线 | 不追求花，保两个清楚落点 |
| 回身腿 | 先转身，再短促踢出 | 需要中景交代身体轴线 |

Seedance 句式：

```text
角色支撑脚先踩实地面，膝盖微弯，髋部带动腿线短促蹬出。接触点只写在对手下盘路线或上身广义区域，对方步伐被截住，脚下积水炸开，身体向侧后失衡。其他人只因路线被挡或水花碎石飞散停步，不同步后退。
```

禁止：

- 旗袍、长袍、狭窄空间里乱用高踢。
- 一字马、长时间腾空、滑行踢。
- 连续五六脚没有接触点。

## 8. 膝法贴身

适合：泰拳、近身压制、狭窄空间短促爆发。

写法重点：

- 必须先有贴身距离或控距。
- 膝法是短促上顶，不是远距离飞膝乱撞。
- 反馈落在对手上身广义区域、衣料和重心，不写危险部位。

Seedance 句式：

```text
角色先用双手或肩线压住近身距离，脚步贴住地面，下一瞬短促提膝。接触点只表现对手上身衣料一震，呼吸一顿，身体被迫后撤半步；衣料雨水甩开，旁边木箱被他撞得轻响。
```

## 9. 擒拿制服

适合：现代制服、走廊突围、护送、女性角色以巧制力。

写法重点：

- 目标不是伤害，而是让对方动作线停止。
- 写“手腕、衣袖、肩线、手臂动作线”，不写关节损伤。
- 结果是对方停步、转向、被带偏、无法继续推进。

Seedance 句式：

```text
角色没有追打，而是贴近后扣住对方手腕和衣袖，另一只手压住肩线，把对方原本向前的动作线带向侧面。对方脚步被迫错开，身体失衡撞到墙边，动作停止，主角借这个空档穿过门口。
```

安全替代：

- “反折、脱臼、锁喉”一律改成“压住动作线、带偏重心、停止推进”。

## 10. 摔投失衡

适合：柔道、摔跤、街斗倒地、太极借力。

写法重点：

- 先写重心和脚步，不要凭空摔飞。
- 写别步、转胯、带落点。
- 落点要安全可读：地面、草垫、木板、门边、摊位，不写头颈具体落点。

Seedance 句式：

```text
对手冲近时，角色先侧身让线，一脚别住对方前进路线，转胯带动对方重心离开原位。对方身体短暂离地，衣摆和水滴被甩起，随后横向落到木板地面，地面灰尘震开一圈；他没有失去意识，只是撑地喘息，动作线被打断。
```

## 11. 借力反制

适合：太极、合气道、柔术化动作、强弱反转。

写法重点：

- 先粘住来力，再让对方力量落空。
- 反击不一定大，但结果要明显。
- 重点是“被自己的力量带出去”。

Seedance 句式：

```text
对手重拳压来，角色手臂先贴住来力，身体没有硬顶，而是转腰半圈把力线引到侧面。对手拳势落空，脚步追不上身体，下一拍被角色短促一带，整个人向自己原本发力的方向冲出去，撞乱后方桌椅。
```

## 12. 身法换位

适合：八卦掌、醉拳、闪避反击、1v2 / 1v3。

写法重点：

- 身法不是装饰，必须改变攻击角度。
- 绕圈、错步、假倒、低身之后要接一个明确结果。

Seedance 句式：

```text
角色没有正面硬接，而是脚步斜切半圈，身体从对手攻击线外滑到侧后方。对手扑空半步，主角立刻从侧线短掌打入衣料区域；命中点小而清楚，水滴和灰尘向侧后炸开，对手被迫转身补位。
```

## 13. 倒地翻滚起身

适合：受击后不崩、救援后起身、追逐中跌落、强动作连续性。

写法重点：

- 倒地不是失败，而是卸力。
- 必须写接触地面的顺序：肩背、手掌、侧身、脚掌重新找地。
- 起身必须有支点。

Seedance 句式：

```text
角色被力量带倒后没有平滑滑走，先侧肩触地，手掌拍在湿地面上卸力，水花和灰尘同时溅起。身体顺势翻滚半圈，脚掌重新踩住地面，单手撑地起身，眼神仍锁住对手。
```

## 14. 群像徒手围攻

适合：1v多、院落突围、夜市群殴、帮派围攻。

写法重点：

- 一次最多 1-2 个有效威胁。
- 直接受力者反应最大。
- 二级碰撞者因挡线、撞物、让路反应。
- 外围群演只承担密度、遮挡和停步。

Seedance 句式：

```text
包围圈不是所有人同时冲，第一波只有正前方和左侧形成有效威胁。主角先处理正前方直接受力者，命中后他飞退撞到侧后方同伴，侧后方同伴被挡线停步，外围只抬臂遮挡和让路。气波只带起衣摆、水滴和尘土，不直接打飞全场。
```

## 15. 拳种 / 武术流派路由

说明：用户口语里说“招式”时，经常指的是咏春拳、八极拳、太极拳这类拳种 / 武术流派。本页按这个口径处理：先识别拳种 / 流派，再选身体武器和代表动作。单个动作名如日字冲拳、贴山靠、崩拳、扫腿，属于下一层“代表动作词”。

| 用户口语中的“招式” | 正式层级 | 核心动作 DNA | 代表动作词 | Seedance 写法重点 |
|---|---|---|---|---|
| 咏春拳 | 南拳近战短桥 | 短桥、中线、连消带打、手肘内收 | 问路手、摊手、膀手、伏手、日字冲拳、寸劲 | 守中线，手部变化贴身清楚；一拍化开，一拍打入，一拍命中短停 |
| 八极拳 | 刚猛贴身拳种 | 震脚、闯入、肩肘靠打、整劲爆发 | 贴山靠、顶肘、撑捶、震脚 | 脚掌震地，肩背压入，命中后对方撞物或飞退，场景接住力量 |
| 太极拳 | 内家借力拳种 | 粘连、化力、转腰、借力发放 | 掤、捋、挤、按、云手、野马分鬃、白鹤亮翅 | 先贴住来力，不硬顶；转腰化开后短促发出，让对方被自己的力带出去 |
| 形意拳 | 内家直进拳种 | 三体式、直行直进、五行拳、硬打硬进 | 劈拳、崩拳、钻拳、炮拳、横拳 | 起势稳，直线进入，命中点干脆；少绕圈，结果要重 |
| 八卦掌 | 走圈换位拳种 | 走圈、扣摆步、绕侧后、换掌 | 穿掌、拍掌、扣步、摆步、游身掌 | 脚步改变攻击角，从侧线进入；绕不是装饰，绕后必须接命中或破位 |
| 洪拳 / 南拳 | 南派沉桥重拳 | 扎马、沉桥、桥手压制、拳重而稳 | 工字伏虎、虎鹤双形、沉桥、冲拳 | 低重心、桥手先压住对方来路，重拳落点稳，衣料和地面反馈重 |
| 醉拳 | 假失衡拳种 | 假倒、假醉、真控重心、就势反击 | 醉步、跌扑、滚翻、就地反击 | 看似踉跄但支点清楚；倒地、翻滚、起身都要有手脚支撑 |
| 螳螂拳 | 象形快手拳种 | 勾搂采挂、短快连取、手部节奏密 | 勾、搂、采、挂、刁、进 | 手部小动作要清楚，不糊成乱甩；每次牵引都改变对方动作线 |
| 鹰爪 / 擒拿系 | 控制制服体系 | 抓、扣、封、压、带偏重心 | 刁、拿、扣、封、压、携腕、牵羊 | 不写关节伤害；写压住动作线、带偏重心、停止推进 |
| 北腿 / 戳脚 | 腿法体系 | 支撑脚抓地、髋部带动、低位截线 | 弹腿、侧踹、正蹬、扫堂腿、勾挂腿、玉环步 | 先写支撑脚；腿法改变下盘路线，不让脚底滑行 |
| 泰拳 | 现代八体打击体系 | 拳、肘、膝、腿，贴身压迫 | 直拳、肘击、顶膝、扫腿、蹬踹 | 控距后短促肘膝腿；反馈落在衣料、重心、脚步，不写危险部位 |
| 拳击 | 现代拳法体系 | 步伐、直摆勾、防反、头肩闪避 | 刺拳、直拳、摆拳、勾拳、闪躲 | 前后脚发力，拳点清楚；快拳必须有命中短停和肩背反震 |
| 散打 / MMA | 现代综合徒手体系 | 踢打摔转换、抢位、压制后收束 | 低扫、侧踹、抱摔、压制、脱身 | 先写距离转换，再写单个结果；不要同镜头把拳腿摔全堆满 |
| 柔道 / 摔跤 | 摔投控制体系 | 重心、别步、转胯、落点 | 一本背负、大外刈、小内刈、抱摔、绊摔 | 先写别步和转胯，落点安全可读，地面水花或灰尘震开 |

选择规则：

- 如果用户只给拳种 / 流派，例如“咏春拳”，先套该流派核心 DNA，再选 1-2 个代表动作。
- 如果用户同时给人数，例如“咏春 1v2”，先按 1v2 小组夹击安排 A/B 分工，再让咏春动作只处理一个直接威胁。
- 如果用户给年代 / 服装，例如“民国旗袍女侠用咏春”，动作必须受服装和场地限制，少高踢，多短桥、转马、截步。
- 如果用户给强打击要求，拳种 / 流派不变，只提高命中反馈等级：更清晰的接触点、更短的 hit-stop、更明显的衣料/水花/灰尘/场景传导。

## 16. 徒手徒脚资产要求

角色资产必须补：

- 肩颈、肘、膝、髋、脚踝结构清楚。
- 肌肉量级明确：纤细、匀称、肌肉、壮硕等。
- 服装活动度明确：旗袍、长袍、风衣、西装、布衣分别限制不同腿法。
- 衣料状态：湿雨、尘土、凹陷、震动、飞水。
- 受力表情：脸颊肌肉短促挤压，五官保持可识别。

场景资产必须补：

- 地面摩擦：青石板、木地板、泥地、积水、雪地、沙地。
- 可撞物：门板、桌椅、摊位、墙角、木箱、竹筐、屏风。
- 可反馈层：水花、灰尘、碎石、木屑、灯火、纸窗。

## 17. 安全边界

高危写法要改：

- 精确易伤部位 -> 上身广义区域、肩侧、下盘路线、衣料区域。
- 关节损伤 -> 动作线被压住、重心被带偏、停止推进。
- 昏迷 / 死亡 -> 撑地喘息、靠墙坐下、不再继续动作。
- 血腥 -> 雨水、水滴、灰尘、擦痕、衣料震动。

## 18. 一句话结论

徒手徒脚肉搏不是“拳打脚踢”四个字，而是：先确定拳种 / 武术流派，再确定身体哪一段发力，最后写接触点、反震、场景传导和结果。每一种拳种都要有自己的动作 DNA。


# 法术 / 异能专项案例页

适用范围：给 `法术 / 异能 / 气劲 / 剑气 / 掌风 / 护体 / 结界 / 符咒 / 阵纹 / 幻术 / 精神压迫 / 空间位移 / 法相真身 / 召唤物` 提供稳定写法。使用时先选法术动作母体，再写来源成形、释放轨迹、相抵/命中、人物受力和环境余波。

## 1. 核心判断

法术戏不是“光效更多”，而是“力量来源、相抵点、人物受力和环境余波”是否成立。

```text
前摇 -> 来源成形 -> 释放轨迹 -> 相抵 / 命中 -> 人物受力 -> 环境余波 -> 收束 / 末帧
```

法术 / 异能的打击感优先级：

```text
人物动作 > 能量来源 > 轨迹方向 > 相抵点 > 受力反馈 > 余波特效
```

只要用户反馈“只有特效、没有打起来、像随机光爆”，先降级为 `S1 来源可读 -> S2 相抵锁定` 测试，不急着写大范围毁灭。

## 2. 法术 / 异能总分类

| 子类 | 核心距离 | 动作 DNA | 必写反馈 | 易错 |
|---|---|---|---|---|
| 元素法术 | 中远距 / 范围 | 掌心、法印、符箓、阵纹聚能后释放 | 护体面凹陷、衣袍鼓起、灯火偏移、雨水被推开 | 只有满屏火光 / 雷光，没有人 |
| 气劲 / 掌风 / 剑气 | 中距 / 中近距 | 武学动作带出脱体劲力，轨迹短、窄、方向清楚 | 空气波纹、衣摆掀动、脚步后滑、尘线外扩 | 把气劲写成无来源法术炮 |
| 护体 / 结界对冲 | 贴身到远距 | 防御面立起、受压、裂纹、回弹或卸力 | 光面震荡、裂纹光痕、肩背承压、脚步后退 | 护盾像静态贴图，没有受力 |
| 符咒 / 阵纹 | 固定区域 | 符纸、墙面、地面、门框、法器作锚点 | 阵线亮起、边缘抖动、灰尘沿线扩散 | 地面突然发光，没有锚点 |
| 精神 / 幻术 | 视线 / 声音 / 气味 / 回声 | 眼神、铃声、香烟、低语触发感知错位 | 呼吸停顿、眼神一滞、背景虚焦、声音拉远 | 写成血腥或直接身体伤害 |
| 空间位移 | 短距 / 立体空间 | 起点压缩、路径折痕、落点风声 | 衣摆延迟、尘土滞后、灯笼晃动、脚步重新落地 | 满屏瞬移，位置关系丢失 |
| 法相 / 真身 | 中远距 / 终局 | 背后虚影、体表光纹、威压下压 | 袍袖后掀、石阶落灰、低境界者退线 | 巨影遮住主角动作 |
| 召唤 / 御物 | 中远距 / 多目标 | 召唤物从阵纹、法器或阴影中出现，遵循主人手势 | 落地重量、风压、阵纹震动、路线改变 | 召唤物凭空乱入无因果 |

## 3. 法术接触锁定测试

### 3.1 S1-S6 测试阶梯

| 测试级 | 目标 | 通过标准 |
|---|---|---|
| S1 来源可读 | 观众知道法术从哪里来 | 掌心 / 法印 / 剑身 / 符纸 / 阵纹 / 眼神 / 法器等来源清楚 |
| S2 相抵锁定 | 证明力量真的撞上 | 两股能量、护体面或法器在同一点顶住，短停 0.12-0.2 秒 |
| S3 人物受力 | 证明不是纯特效 | 肩背一震、衣袍鼓起、脚步后滑、手指收紧、呼吸一顿 |
| S4 环境余波 | 证明力量传导到场景 | 雨水、灰尘、灯火、纸窗、石阶、阵纹按受力方向变化 |
| S5 攻防互换 | 证明不是单方面光炮 | 一方释放，一方护体 / 格挡 / 让线，随后反压或脱离 |
| S6 多人法术 | 证明群像有层级 | 只有主相抵点最强，外围通过风压、遮挡、阵线传导二级反应 |

### 3.2 三帧法术锁定

```text
来源帧：人物先做可读前摇，掌心 / 法印 / 剑身 / 符纸 / 阵纹出现能量源，脸和手不能被特效遮住。
相抵帧：能量撞上护体面 / 另一股能量 / 法器表面 / 地面阵线，短停 0.12-0.2 秒，接触点凹陷、震荡或产生裂纹。
受力帧：受力者肩背短震、衣袍鼓起、脚步后滑半步；余波再推开尘土、雨水、灯火和纸窗。
```

所有光爆、冲击波、地裂、云层翻动、群演后退，都必须排在相抵帧之后；不能用特效替代相抵点。

## 4. 表现模式分流

同一段法术，真人写实、国漫高燃、仙侠奇观不能混用一套词。

| 表现模式 | 可以使用 | 默认避免 | 核心判断 |
|---|---|---|---|
| 真人写实异能 | 空气折痕、灯光闪烁、风压、雨丝偏移、尘土反冲、衣摆掀动 | 满屏符文、炁流游龙、巨型法阵铺满画面 | 特效像现实空间异常，人物受力必须先成立 |
| 国漫高燃法术 | 红青炁流、墨色粒子、速度线、夸张冲击波、能量对冲 | 完全写实摄影逻辑、过弱视觉反馈 | 先让爽感成立，但相抵点和攻防互换不能丢 |
| 仙侠玄幻法术 | 法印、阵纹、剑气、灵光护体、结界裂纹、法相虚影 | 无来源随机光爆、全屏粒子雾 | 能量体系清楚，特效贴来源、轨迹和受力方向 |
| 暗黑 / 精神幻术 | 低频声、眼神压迫、背景虚焦、空间轻微错位、回声拉远 | 血腥身体伤害、不可识别脸部扭曲 | 伤害表现落在感知错位和动作中断 |

## 4.1 宏大特效尺度升级规则

用户反馈“特效不够宏大 / 小家子气”时，不是简单把光效加亮，也不是让粒子铺满全屏。宏大感来自五层同时成立：

```text
人物前摇清楚 -> 能量源扩大 -> 相抵点放大 -> 中层冲击展开 -> 外层环境连锁 -> 大景重组
```

| 宏大层级 | 写法 | 避免 |
|---|---|---|
| 空间尺度 | 从半身近景升级到山门、广场、石阶、殿门、巨柱、云层等可比较尺度 | 只在掌心一小团发光 |
| 能量体量 | 火浪、水幕、雷柱、风墙、阵纹覆盖一段石阶或半个广场，但来源仍从人物动作开始 | 无来源全屏光爆 |
| 相抵核心 | 相抵点最亮、最密、最有压缩感，0.12-0.2 秒短停 | 到处都一样亮，看不出爆点 |
| 中层冲击 | 圆形气浪、水汽墙、火星雨、阵纹裂光从相抵点扩展到数米外 | 粒子随机乱飞 |
| 外层环境 | 巨柱落灰、宫灯齐晃、雨幕被推成弧面、石阶积水整片后退、云层开口 | 地面凭空炸坑 |
| 大景运镜 | 近景看来源，中景看相抵，远景 / 高俯拍看能量规模和空间被改写 | 全程近景导致小气 |

宏大法术必须保留“人物小，能量大”的对比：人物动作仍然真实，衣袍、肩背、脚步仍有受力；能量和环境反馈则扩大到场景层级。宏大不是失控，所有大范围变化都必须从相抵点向外传导。

可直接复用句式：

```text
这版法术特效要更宏大，但仍是真人电影级写实。能量不是掌心小光团，而是从掌心 / 法印 / 阵纹扩展成覆盖整段石阶的火浪 / 水幕 / 雷光 / 风墙。相抵点保持最清楚，先短停 0.12-0.2 秒，再让冲击向外扩展：雨幕被推成弧面，宫灯同时摇晃，巨柱落灰，石阶积水整片向两侧退开，远处云层被余波照亮。所有宏大特效都从相抵点发散，不凭空全屏爆光。
```

## 4.2 仙侠 / 玄幻剧法术参考母体

> 参考方向：`长月烬明`、`斗罗大陆`、`斗罗大陆之燃魂战`、`青云志`、`花千骨`。  
> 使用原则：只学习“法术视觉语法”，不照搬 IP 角色、具体招式名、明星脸或可识别名场面。

这些参考共同解决一个问题：法术不能像“演员摆手 + 后期贴光”。它必须让观众先看懂力量属于谁、从哪里来、为什么这么大、撞到哪里、谁承受了它。

### A. 五类法术视觉母体

| 参考母体 | 学习重点 | Seedance 写法 | 易错 |
|---|---|---|---|
| 神魔光源型 | 神性 / 魔性通过光源、色谱、纹样、空间压迫区分 | 先写角色体内或法印出现主光源，再让光照改变脸、衣袍、地面和云层；神性偏清透上升，魔性偏低位压迫、暗红 / 黑金 / 紫黑翻涌 | 只写“红光”“黑气”，没有光源和空间压迫 |
| 魂技图腾型 | 武魂 / 魂环 / 魂技层级清楚，技能身份可读 | 先出现武魂虚影或法器轮廓，再亮起一圈 / 多圈能量环，最后释放对应技能；每个角色只保一种主色和一个主技能结果 | 环随便转、兽影乱入、技能没来源 |
| 法宝门派型 | 法器、剑诀、门派法印是法术锚点 | 法术先从剑身、铃、镜、幡、珠、符纸、阵眼发出，角色手诀只是启动器；法器表面受力、震鸣、回弹必须可见 | 人物空手随机放光，法器只当装饰 |
| 仙门符阵型 | 阵纹、结界、封印、门派空间共同承担力量 | 地面、柱面、门框、石阶、剑阵先亮起线条，再形成区域锁定、护体或反压 | 地面突然铺满光，不知道阵眼在哪里 |
| 情劫失控型 | 情绪变化驱动法力失控或爆发 | 先写眼神、呼吸、手指、泪光、衣袍微动，再让法力从身体近处失控外溢，最后影响场景；特效颜色和强度跟情绪变化同步 | 一上来全屏爆炸，没有情绪前摇 |
| 妖力情缘型 | 妖力、红线、誓约、树灵 / 根脉把人物关系可视化 | 先写两人之间的情绪牵引，再让红线、花瓣、狐火、树影根脉从两人之间生长，形成护阵、牵引、封印或反噬 | 红线随机飘，和人物关系无关 |
| 水木竹影型 | 淮水、竹亭、竹叶、水纹、风声构成清雅法术空间 | 法术从竹叶、水面、纸伞、袖口或竹亭梁柱生长，先细密轻柔，再在相抵时变成水幕、竹影阵、叶刃风线 | 只写绿色粒子，没有水木空间 |
| 剑心剑意型 | 剑意来自本心、誓念、剑身与剑阵，不只是武器发光 | 剑身先微震，剑心光线从胸口 / 手腕 / 剑脊传到剑尖，再分化成剑气窄线或万剑阵；剑意强弱跟人物意志同步 | 满屏飞剑乱飞，没有主剑和心念来源 |

### B. 反生硬三层生长

法术生硬，通常是因为只写了“释放结果”，缺了生长过程。必须把特效拆成三层：

```text
内层：人物身体前摇和法力源头
中层：法术成形、纹样 / 图腾 / 魂环 / 法器响应
外层：相抵点之后的空间余波和场景连锁
```

稳定句式：

```text
法术不能凭空出现。先拍 {人物} 的呼吸、眼神、手诀、脚步或法器微震，能量从 {掌心/眉心神印/剑身/魂环/符纸/阵眼/法器} 一层层生长出来；第一层贴近身体，第二层扩展成 {光纹/魂环/法阵/图腾虚影/护体面}，第三层才沿 {直线/弧线/下压/环形/贴地} 释放。相抵后再让雨水、尘土、衣袍、宫灯、石阶、云层按受力方向变化。
```

### C. 体系可读：先定“法术身份”

如果提示词只写“火焰、雷电、蓝光、红光”，模型会生成廉价光效。写之前必须先选择法术身份：

| 法术身份 | 必写来源 | 可视化符号 | 攻防结果 |
|---|---|---|---|
| 神力 / 神印 | 额间神印、胸口光纹、背后神相 | 金白、琉璃、飞天纹样、上升光柱 | 护体、净化、镇压、封印 |
| 魔气 / 魔神 | 低位黑红光、指尖暗纹、背后魔相 | 暗红、黑金、紫黑、地面裂光、低位雾压 | 侵蚀、压制、吞噬、破阵 |
| 魂技 / 武魂 | 武魂虚影、魂环、兽形 / 器形轮廓 | 环形能量、兽影、器武魂、技能色带 | 控制、强攻、辅助、敏攻、护体 |
| 法宝 / 法器 | 剑、铃、镜、幡、珠、伞、印 | 法器纹路亮起、悬浮、震鸣、回弹 | 封路、反弹、镇压、牵引 |
| 符阵 / 结界 | 地面阵眼、符纸、门框、石柱 | 阵线、符文、圆阵、裂纹光痕 | 困住、护住、反压、破眼 |
| 情绪失控力 | 呼吸紊乱、眼神失焦、手指颤抖 | 体表光裂、衣袍无风鼓起、周围物件悬浮 | 爆发、失控、误伤、被压回 |

### D. 五部参考的可迁移规则

- `长月烬明`方向：使用“神魔色谱 + 敦煌 / 上古纹样 + 大空间光源”。适合神魔大战、法相、终局法术、神印觉醒。写法重点是人物小、神域大、主光源贯穿世界观。
- `斗罗大陆 / 燃魂战`方向：使用“武魂显形 + 魂环启动 + 技能释放 + 团队分工”。适合玄幻团队战、等级体系、魂技或异能技能。写法重点是每个角色的技能身份和颜色不能混。
- `青云志`方向：使用“法宝 / 仙剑 / 门派法诀 / 结界阵法”。适合仙门斗法、剑阵、法宝对冲、灵兽或召唤物登场。写法重点是法器锚点和门派空间。
- `花千骨`方向：使用“情绪前摇 + 仙力失控 / 封印松动 / 洪荒力量外溢”。适合情劫、黑化、爆发、牺牲前法术。写法重点是先情绪再特效。
- `狐妖小红娘·月红篇`方向：使用“妖力情缘 + 红线契约 + 树灵根脉 + 狐火护阵”。适合情缘缔结、妖力护人、黑气侵蚀、以情为阵。写法重点是法术必须绑定人物关系，红线 / 树影 / 狐火从两人之间或情绪节点生长。
- `狐妖小红娘·淮水竹亭`方向：使用“水木竹影 + 家族法术 + 竹亭空间 + 清雅攻防”。适合竹林水榭、纸伞风线、水纹护阵、竹叶封路。写法重点是法术不一定巨大，但要有水、竹、风、袖、伞、亭之间的空间联动。
- `天地剑心 / 王权篇`方向：使用“剑心本源 + 剑阵秩序 + 万剑压迫 + 意志破局”。适合剑修对决、剑阵破阵、万剑穿压、以剑护人。写法重点是剑意先从人物心念和主剑启动，再扩展成剑气 / 剑阵，不能直接满屏飞剑。

### E. 生硬判废项

- 演员只伸手，特效直接从画面中央出现。
- 只有光团变大，没有神印、魂环、法器、阵眼或身体动作来源。
- 特效颜色随机切换，和角色、阵营、技能身份不绑定。
- 法术爆开后人物没有肩背、脚步、衣袍、呼吸的受力反馈。
- 没有相抵点，只有两团光互相覆盖。
- 魂环、阵纹、法相、召唤物遮住脸、手和身体动作。
- 情绪爆发型法术没有眼神、呼吸、手指、停顿等前摇。

## 4.3 降魔 / 驱邪 / 镇煞参考母体

> 参考方向：钟馗、黑悟空系神怪动作、燕云系写实武侠奇术、马小玲、英叔。  
> 使用原则：学习“驱邪降魔的仪式动作、法器锚点、邪祟受制反馈”，不复刻具体人物脸、造型、台词或可识别名场面。

降魔戏和普通法术不一样。普通法术常常是“能量对冲”，降魔更像“识邪 -> 定场 -> 镇压 -> 封收”。如果只写一道光打过去，会显得生硬，也没有驱邪仪式感。

### A. 降魔五类母体

| 母体 | 学习重点 | Seedance 写法 | 易错 |
|---|---|---|---|
| 判官镇煞型 | 钟馗式威压、判官印、符箓、锁链、门神金红光 | 先识破邪气来源，再以判官印 / 符箓 / 金红法链定住邪祟影子，镇压从头顶或地面阵线向内收紧 | 只写大红光，没有“定住”和“收紧” |
| 神怪斗法型 | 黑悟空系：神怪实体、棍势、法相、定身、化身、妖气反扑 | 武学动作和法术同时成立，先有身法 / 兵器 / 法相起势，再让妖气、尘土、石像、山风跟着受力 | 只有特效，没有动作和妖物重量 |
| 江湖奇术型 | 燕云系写实武侠奇术、符纸、机关、身法、兵器压制 | 降魔动作更接地，靠贴符、踏阵、兵器封线、身法避邪气完成，不做全屏仙光 | 太仙侠化，失去江湖写实感 |
| 都市驱魔型 | 马小玲式现代驱魔、手印、念珠、符咒、龙形能量、城市夜景 | 现代环境中先建立阴气和人群危险，再用符咒 / 手印 / 法器拉出清楚能量路径，护人和封邪同时进行 | 现代场景里突然古装大法阵 |
| 僵尸道法型 | 英叔式桃木剑、黄符、铜钱剑、八卦镜、铃铛、红绳、糯米 | 先布阵和试探，桃木剑 / 符纸 / 铜钱剑接触邪祟时才爆符火；邪祟受制表现为后退、僵住、黑气回缩 | 直接把僵尸打飞，全无道法步骤 |

### B. 降魔动作链

```text
识邪：发现邪气来源 / 影子异常 / 怨气附体 / 妖物真形
定场：符纸、法器、阵线、红绳、铜钱、剑尖、脚步落位形成边界
交锋：邪气反扑，人物避让 / 架住 / 贴符 / 法器相抵 / 兵器封线
镇压：法印、符火、剑气、法链、金光、佛道纹样从相抵点向内收紧
封收：邪气被压回器物 / 符箓 / 阵眼 / 影子 / 门缝，场景恢复但留下余灰和符纸残光
```

### C. 降魔视觉反馈

| 反馈层 | 稳定写法 |
|---|---|
| 邪祟反馈 | 影子被钉住、黑气向内回缩、附身者脚步停住、衣袍被反风压住、眼神短暂清醒 |
| 法器反馈 | 符纸燃边、桃木剑震鸣、铜钱剑细响、八卦镜反光、铃铛急响后忽然停住 |
| 场景反馈 | 门窗抖动、烛火倒伏、灰尘沿阵线扩散、地面符线一圈圈收紧、雨水绕开阵眼 |
| 人物反馈 | 手指结印发紧、肩背承压、脚步扎地、呼吸变短、衣袖被邪风反推 |

可直接复用句式：

```text
这段是降魔镇煞，不是普通光炮。先让 {人物} 识破邪气来源：影子异常 / 门缝黑气 / 附身者动作停顿；随后用 {桃木剑/符纸/铜钱剑/八卦镜/判官印/红绳/法器} 定住边界。邪气先反扑，人物侧身避开主邪风，再把法器压到 {阵眼/影子/门框/地面符线/邪祟身前半臂外}，相抵点 0.12-0.2 秒短停。符火、金红法链、镜光或剑气从相抵点向内收紧，邪气不是爆开，而是被压回 {符箓/影子/门缝/器物/阵眼}；周围烛火、纸窗、雨水和灰尘按收束方向变化。末帧留下符纸余灰、法器微震和人物短促呼吸。
```

### D. 降魔判废项

- 没有邪气来源，人物直接对空气放光。
- 没有法器 / 手诀 / 阵眼锚点，光效凭空出现。
- 邪祟被打中后没有受制反馈，只是随机飞走。
- 降魔没有“收束”，只有爆炸。
- 符纸、桃木剑、铜钱剑、八卦镜、铃铛只当装饰，不改变邪祟路线。
- 现代驱魔、僵尸道法、神怪斗法、仙侠法术混成一团，年代和场景不统一。

## 5. 子类写法

### 5.1 元素法术

元素法术必须写“元素来源”和“路径”，不能只写元素名。

| 元素 | 来源建议 | 轨迹 | 相抵反馈 | 环境余波 |
|---|---|---|---|---|
| 火 | 掌心火纹、符纸燃边、剑身火线 | 直冲、卷压、贴地蔓延 | 护体光面变红、热浪回弹 | 木屑焦黑、灯火被压低 |
| 水 | 袖口水珠、地面积水、法印水纹 | 水幕、弧线、水鞭、涌浪 | 护体面凹陷成水纹 | 雨水逆卷、衣袍湿痕加重 |
| 冰 | 指尖寒霜、地面白雾、剑刃霜线 | 冻线、冰棱、寒雾扩散 | 护体面结霜后裂开 | 石阶霜纹、呼吸白雾 |
| 雷 | 掌心电弧、符咒紫纹、云层低压 | 闪断、链状、直线劈落 | 光面短促抽搐、手臂震麻感 | 灯火闪烁、雨丝被电弧点亮 |
| 风 | 袍袖旋风、剑身风线、脚下尘圈 | 风刃、压风、旋流 | 护体面被压出凹形 | 灰尘成线、纸窗鼓动 |
| 土 / 岩 | 地面阵纹、脚掌震地、石阶裂线 | 岩刺、土墙、地脉压上 | 护体面被硬推后移 | 石屑跳起、地面短裂纹 |
| 金 | 法器金纹、剑芒、指尖金线 | 金刃、金光束、利线切过 | 光面发出尖锐震鸣 | 金属饰物同频震动 |
| 木 / 藤 | 地面根线、符纸青纹、树影伸长 | 藤蔓封线、枝叶旋绕 | 护体面被缠住后压紧 | 落叶逆卷、木梁轻响 |

句式：

```text
{人物} 先用 {掌心/剑身/符纸/地面阵纹} 聚起 {元素}，能量从 {来源} 成形，沿 {直线/弧线/压下/贴地} 轨迹释放。{受力者} 用 {护体/兵器/法器} 顶住，接触点短停 0.12-0.2 秒，护体面出现 {凹陷/裂纹/震荡}，肩背一震，脚步后滑半步。余波只沿受力方向推开 {雨水/尘土/灯火/纸窗/衣袍}。
```

### 5.2 气劲 / 掌风 / 剑气

气劲适合武侠和高武，不一定发光；写实版本应优先用空气和环境表现。

```text
起势：沉肩 / 转腰 / 剑身压线 / 掌心收劲
释放：短促掌风 / 剑气窄线 / 透明气劲波纹
相抵：护体面、兵器、门板或地面尘线被压住
结果：对方退半步、衣袍后掀、尘土按方向外扩
```

禁止把气劲写成无来源远程炮；必须有身体动作或兵器动作作为发力源。

### 5.3 护体 / 结界对冲

护体不是静态盾牌，必须有受压变化。

| 阶段 | 写法 |
|---|---|
| 立起 | 护体光面从胸前、剑身、法印或地面阵纹展开 |
| 受压 | 接触点向内凹陷，光面出现裂纹、波纹或层叠震荡 |
| 承重 | 防守者肩背压低、脚步后滑、手指收紧 |
| 卸力 | 光面斜向导开来势，余波擦过地面或柱子 |
| 结果 | 防线仍在但被压退，或裂开一角露出破绽 |

句式：

```text
{防守者} 的护体光面不是平面贴图，而是从 {来源} 立起。{攻击法术} 压上时，光面接触点向内凹陷并出现裂纹光痕，0.12-0.2 秒相持；防守者肩背一沉，脚步后滑半步。下一拍护体把来势斜向卸开，余波擦过 {石阶/柱子/雨水/尘土}，末帧留下防线裂纹或短暂破口。
```

### 5.4 符咒 / 阵纹

符咒阵纹必须有锚点：符纸、地面、门框、墙面、法器、掌心。没有锚点的发光地面容易像随机贴图。

稳定结构：

```text
锚点出现 -> 线条亮起 -> 区域锁定 -> 相抵 / 封路 -> 阵线抖动 -> 结果
```

常用动作：

- 符纸封路：符纸先贴住门框 / 柱子，再拉出光线封住路线。
- 地阵锁步：脚下阵纹先亮一圈，角色脚步短暂停住，灰尘沿阵线外扩。
- 反阵破眼：主角先找到阵眼锚点，再用法器 / 掌风压住一点，阵纹从该点断裂。
- 双阵对冲：两片阵纹边缘顶住，接触线最亮，外围光线弱一层。

### 5.5 精神 / 幻术

精神幻术的“打击感”来自感知被压住，不来自血腥身体伤害。

```text
触发源：眼神 / 铃声 / 香烟 / 回声 / 指尖轻敲
感知变化：背景虚焦、声音拉远、灯火延迟、空间轻微错位
人物反馈：呼吸停顿、眼神一滞、手中动作被打断、脚步迟疑
结果：对方错过半拍、路线判断错误、护体或招式前摇被打断
```

禁止写七窍流血、脸部扭曲、不可识别表情、真实精神伤害细节。可写“眼神短暂失焦、额角冷汗、呼吸乱掉、手中法诀被打断”。

### 5.6 空间位移

空间位移必须保留起点、路径和落点，否则模型容易乱跳。

```text
起点压缩 -> 空气折痕 / 残影边缘 -> 短距错位 -> 落点风声 -> 衣摆和尘土延迟追上
```

写法原则：

- 位移距离不宜过大，优先半步、一臂、一步半、绕到侧后方。
- 起点必须留痕：脚下水花、衣摆残影、空气折痕。
- 落点必须承重：脚掌重新落地，尘土延迟落下，灯笼或衣摆被带动。
- 如果用于闪避，必须写“避开什么路线”；如果用于进攻，必须写“落点后如何相抵或逼退”。

### 5.7 法相 / 真身 / 召唤

法相和召唤适合高潮，但容易遮住主体。规则是“虚影服务人物动作”。

```text
人物先起势 -> 背后法相 / 召唤物成形 -> 法相跟随人物动作放大力线 -> 相抵点落在护体 / 阵眼 / 地面 / 兵器上 -> 人物和环境共同受力
```

禁止：

- 巨大虚影把主角脸、手、身体动作全部遮住。
- 召唤物凭空出现，没有阵纹、法器、阴影或地面裂光作为来源。
- 全场单位同步后退；大范围压迫也要有前排、后排、侧翼分层传导。

## 6. 一来一回法术攻防

法术对打不能是两个人各自放光。必须有攻防互换。

```text
一方前摇释放 -> 另一方护体 / 让线 / 反阵 -> 相抵短停 -> 受压后退 -> 反压或改线 -> 双方收势重组
```

句式：

```text
这段法术必须是一来一回。先让 {攻击者A} 用 {来源} 聚起 {法术A}，轨迹从 {方向} 压向 {防守者B}；{防守者B} 不原地发光，先用 {护体/符咒/法器/位移} 处理来势。两股力量在 {相抵点} 顶住并短停 0.12-0.2 秒，防守者肩背一震、脚步后滑。下一拍，防守者借相抵结果把来势卸向 {地面/柱子/侧墙/天空}，再用 {反击法术/阵纹/气劲} 抢回主动权。末帧双方都收势重组，下一轮攻防从新的距离开始。
```

## 7. 多人法术 / 群像法术

多人法术必须分层，不是所有人一起发光。

默认分层：

- 第一层：主释放者和主受力者。
- 第二层：护阵、补符、维持结界的人。
- 第三层：被余波压退、遮挡、停步的人。
- 第四层：只承担场面密度的背景群演。

规则：

- 一次只保一个主相抵点。
- 群体反应从主相抵点向外传导，不能全场同步倒退。
- 阵法多人需要分工：有人维持阵眼，有人补线，有人承压后退。
- 大范围奇观也要落回局部细节：阵眼裂纹、护体光面凹陷、旗帜和衣袍同方向后掀。

## 8. 禁止与降级

禁止：

- 只有光爆，没有人物动作。
- 特效遮住脸、手、脚步和相抵点。
- 法术无来源、无轨迹、无相抵、无人物受力。
- 一放法术全场无因飞退。
- 精神幻术写成血腥身体伤害。
- 空间位移没有起点和落点。
- 法相和召唤物遮住主体动作。
- 真人写实异能使用全屏水墨游龙、二次元速度线或不可识别脸部变形。

降级策略：

```text
光效失控 -> 降级为 S1 来源可读
看不出对冲 -> 降级为 S2 相抵锁定
人物没反应 -> 降级为 S3 人物受力
全场乱飞 -> 降级为 S4 环境余波 + 单一直接受力者
位置乱跳 -> 降级为空间位移起点 / 路径 / 落点三帧
```

## 9. Seedance 可复用测试句

```text
这次先测试法术 / 异能的相抵和受力，不追求满屏大爆炸。{攻击者A} 先有清楚前摇：{掐诀/剑指/掌心收光/符纸亮起/阵纹浮现}，能量从 {来源} 成形，沿 {直线/弧线/压下/贴地/折线} 轨迹压向 {防守者B}。{防守者B} 用 {护体光面/法器/符咒/气劲} 顶住，两个力量在 {相抵点} 真实相抵并短停 0.12-0.2 秒；接触点出现凹陷、裂纹或水波震纹。受力瞬间，防守者肩背一震，衣袍鼓起，脚步后滑半步，手指收紧；余波再沿受力方向推开雨水、尘土、灯火和纸窗。其他人只因余波遮挡、停步或衣摆被掀产生二级反应，不同步后退。末帧停在双方收势重组，下一轮攻防即将开始。
```


# 防守 / 躲避 / 化劲招式库

用途：给肉搏、冷武器、柔兵、盾牌、远程投掷、法术气波和空中动作补“守式”。防守不是一句“挡住”或“躲开”，而是一条完整动作链：看见来势，守住危险线，承受或让开力量，再借这一下回到反击。

## 1. 总原则

防守动作必须先回答四件事：

1. 攻击线从哪里来：上劈、横扫、直刺、低扫、贴身冲拳、气波、飞剑、火线、坠落压迫。
2. 防守者怎么避开危险线：侧身、下腰、伏低、撤步、错步、架桥、盾顶、兵器格挡、护体承压。
3. 力量落在哪里：前臂、拳面、棍身、剑脊、刀背、盾面、护体光面、地面阵纹、柱子或掩体。
4. 防守后局势怎么变：借触换手、抢侧位、逼退半步、脱离重组、反击、保护他人或开出缺口。

通用链条：

```text
识别来势 -> 起势守线 -> 守式 / 避式 / 化式 -> 相抵 / 擦身 -> 受力反馈 -> 反式 -> 收式
```

### 五类防守

| 类型 | 作用 | 常用动词 | 适用 |
|---|---|---|---|
| 守式 | 直接接住或封住来势 | 格、挡、架、封、护、顶、抱门 | 肉搏、兵器、盾、护体 |
| 避式 | 让开危险线，不硬接 | 闪、侧、伏、蹲、下腰、撤、错步、翻身 | 重兵、气波、飞剑、长兵、怪物扑击 |
| 化式 | 接触后卸力改线 | 拨、带、压、缠、引、卸、滑开、借力 | 太极、咏春、剑刀对拆、柔兵、法术引流 |
| 反式 | 防守后立刻抢回主动 | 还击、反刺、反掌、回身、抢侧、顶回、推开 | 1v1 对招、1v2 夹击、兵器攻守互换 |
| 收式 | 重组距离和姿态 | 回防、收剑、拉开、沉肩、落地、重新起势 | 任何对招结尾 |

硬规则：

- 防守不要提前出现。攻击线发展到 70%-90% 时再守，才有压迫感。
- 守式必须有受力反馈：手腕一沉、肩背短震、衣料抖动、脚步后滑、水花或尘土被震开。
- 避式必须说明避开哪条线：拳线、刀锋线、斧头落线、棍身横扫线、气波中心线、飞剑轨迹。
- 化式必须说明力量被卸到哪里：外侧、地面、柱子、盾面、阵纹、空气弧线。
- 反式必须接一个结果：逼退、抢侧位、打开缺口、救人、脱离、再次相抵。
- 真人写实禁止血腥、具体损伤和明星脸；用失衡、后退、护具 / 衣料受力、光效碎裂替代伤害。

## 2. 肉搏防守招式

肉搏防守必须写身体结构，不只写“挡”。重点看中线、肩肘、腰胯、脚步和距离。

| 来势 | 守式 / 避式 | 化式 / 反式 | Seedance 写法重点 |
|---|---|---|---|
| 直拳 / 冲拳 | 抱门、护中线、摊手、拍手 | 伏手压腕、日字冲拳还击 | 前臂相触，肩肘内收，拳线被压偏 |
| 摆拳 / 横扫掌 | 架桥、侧身让线、低头钻桥 | 贴身进马、肘肩短靠 | 头肩避开拳线，不让脸穿过拳路 |
| 重拳 / 重掌 | 双手架住、撤半步、沉肩卸力 | 借力转腰、推开重心 | 守住后脚掌后滑，衣料和手臂震动 |
| 高位踢 / 横踢 | 提臂格挡、侧身退线、伏低 | 截步抢入、低位反压 | 支撑脚和髋线清楚，不要凭空被踢飞 |
| 低扫腿 | 提膝护线、撤步、跳步避扫 | 截步卡住距离、短拳反击 | 地面水花和脚步重排表现避扫 |
| 膝撞 / 贴身顶撞 | 双肘夹线、下沉重心 | 转胯带偏、肩靠反顶 | 距离压到贴身，反馈短促不夸张 |
| 擒拿抓腕 | 抽腕、沉肘、转肩 | 反缠腕、封肘、带偏重心 | 不写关节损伤，只写动作线被压住 |
| 摔投来势 | 降重心、踩稳、绕背 | 别步反带、推肩脱离 | 落点安全可读，结果是失衡或退步 |
| 气劲 / 掌风 | 下腰避波、侧身让线、伏低穿过 | 贴地进步、掌根反推 | 气波掀衣摆和雨丝，不遮住身体避线 |

肉搏防守招式名速查：

- 守式：抱门守中、架桥护线、膀手封桥、摊手开门、伏手压腕、双臂横架、提膝护线。
- 避式：侧身让拳、低头钻桥、下腰避波、撤步引空、错步换背、伏身穿线、旋身避掌。
- 化式：云手化劲、四两拨千斤、缠腕卸力、沉肩卸劲、借力转腰、贴桥改线。
- 反式：借触换手、贴身进马、短拳还中、肩靠逼退、截步封路、转身推开。

肉搏模板：

```text
{攻击者} 的 {拳 / 掌 / 肘 / 腿} 从 {方向} 快速压近，攻击线发展到身前时，{防守者} 才用 {守式 / 避式} 处理：{前臂 / 肩线 / 脚步 / 腰胯} 清楚避开危险线。相触帧 0.1-0.15 秒短停，能看到衣料、手臂肌肉、水滴和脚步受力。下一拍 {防守者} 借触 {化开 / 抢侧位 / 短促反击 / 脱离重组}，不是两个人隔空挥拳。
```

## 3. 冷武器防守招式

冷武器防守先看兵器长度和危险平面。身体、脸、辅助手不能穿过刀锋、斧刃、枪尖、链刃主弧线后还完好无损。

### 3.1 刀 / 剑 / 短兵

| 兵器 | 防守招式 | 适用来势 | 写法重点 |
|---|---|---|---|
| 剑 | 格剑、挂剑、截剑、压剑、拨挡还刺 | 直刺、斜劈、轻兵变线 | 用剑脊 / 剑身侧面架住，不用手臂穿锋线 |
| 细剑 | 拨挡还刺、退步反刺、侧身让刺 | 窄线突刺、快速点刺 | 先让线，再用细剑偏转，不硬接重器 |
| 软剑 | 抖剑卸力、缠腕带偏、绕身藏锋 | 轻兵、手腕线、侧线攻 | 软刃弹性改线，不能硬接斧锤 |
| 刀 | 刀背横架、压刀卸力、挂刀带开 | 重劈、横砍、短兵压进 | 刀背相抵，脚步承重，辅助手护胸 |
| 匕首 / 短刀 | 藏身让线、贴臂封手、短格抢侧 | 贴身短刺、擒拿混战 | 站位近，动作小，不远距离生效 |
| 双兵 | 左架右切、交叉双架、轮替格挡 | 双线压迫、连击 | 左右手分工清楚，不能残影糊成一团 |

模板：

```text
{攻击者} 的 {武器A} 从 {上劈 / 横扫 / 直刺} 压来，{防守者} 先 {侧身 / 撤半步 / 低头} 避开危险平面，再用 {剑脊 / 刀背 / 短兵侧面} 在 {相抵点} 架住。相抵短停 0.12-0.2 秒，金属摩擦、少量火星或雨水从接触点弹开，手腕一沉，脚步后撤半步。下一拍借相抵点 {滑开 / 压住 / 拨开 / 反刺 / 抢侧位}。
```

### 3.2 长兵 / 棍棒 / 重兵

| 兵器 | 防守招式 | 适用来势 | 写法重点 |
|---|---|---|---|
| 枪 / 矛 | 拦枪、拿枪、拨枪、退步封线 | 直扎、横扫、上挑 | 保持两到三步距离，先封线再反扎 |
| 戟 / 长柄刀 | 侧身让刃、杆身架挡、下压卸力 | 斜劈、钩拉、横扫 | 先避刃，再架杆，不让身体穿过刃区 |
| 棍 | 立棍一挡、横棍封门、拨棍带偏 | 横扫、下劈、群攻 | 木棍回弹、木屑、水滴震开 |
| 斧 / 锤 | 撤步让落点、盾顶、兵器架柄 | 重劈、重砸、横摆 | 不硬接正面重心，防守者脚步必须后滑或下沉 |
| 盾 | 盾面顶守、盾缘带偏、错身推线 | 直冲、重砸、投掷物 | 盾面受压凹感、肩背承压、脚掌蹬地 |
| 道具兵 | 雨伞架挡、折凳封门、木箱盖顶守 | 突围、都市 / 民国道具战 | 结果是开路和逼退，不写高危伤害 |

长兵重兵模板：

```text
{重兵 / 长兵} 的主攻击线先压出来，{防守者} 不站着硬顶，先 {撤步 / 侧身 / 伏低} 让开主力线，再用 {枪杆 / 棍身 / 盾面 / 道具面} 承接余势。接触点短停，{木屑 / 金属闷响 / 盾面震动 / 水花} 沿受力方向弹开，防守者脚掌后滑或重心下沉。下一拍把来势带到外侧，打开一条反击线或脱离距离。
```

### 3.3 柔兵防守

柔兵包括软剑、软鞭、链镖、绳镖、流星锤、长绸类奇门兵。柔兵不能像盾牌一样硬接，要写“绕、缠、带、收”。

| 柔兵 | 防守招式 | 写法重点 |
|---|---|---|
| 软剑 | 抖剑弹开、缠腕带偏、绕身藏锋 | 腕力清楚，主弧线只有一条，剑身弹回护身 |
| 软鞭 | 盘鞭封路、缠兵带偏、回身收鞭 | 先收再放，鞭梢和回收路线可读 |
| 链镖 | 绕兵牵偏、回环收镖、贴地扫链避进线 | 镖头位置清楚，链身不能满屏乱绕 |
| 绳镖 | 挂线牵开、收绳换位、绕柱卸力 | 可借柱、梁、栏杆改变来势 |
| 流星锤 | 绕臂回护、外圈封路、带偏重兵 | 重量在外圈，防守者身体保持中心稳定 |

柔兵模板：

```text
{防守者} 不用柔兵硬挡重击，而是先收势护身，再让 {软剑 / 软鞭 / 链镖} 沿一条清楚主弧线绕到外侧，缠住或带偏 {对方兵器 / 手腕路线 / 进攻线}。链节 / 鞭梢 / 软刃抖动有方向，下一拍立即收回护身或抢侧位，禁止满屏光带和连续乱绕。
```

## 4. 远程投掷 / 暗器 / 飞剑防守

远程防守重点是“轨迹可读”。先写发出点，再写飞行线，再写躲避、格挡或落点。

| 来势 | 防守方式 | 写法重点 |
|---|---|---|
| 飞剑 / 飞刃 | 侧身让线、剑脊格开、盾面弹开 | 飞行线清楚，不从身体穿过 |
| 暗器 / 飞牌 | 伏低、翻身、袖摆遮挡、道具挡开 | 只保 1-3 枚主轨迹，不写满屏乱飞 |
| 箭 / 弩 | 掩体转移、盾面顶守、滚身避线 | 不特写身体命中，写箭落在木柱 / 盾面 |
| 投枪 / 标枪 | 侧身闪避、滚地避开、兵器挑偏 | 速度快但落点必须清楚 |
| 法器抛射物 | 护体相抵、法器引流、地面阵纹卸力 | 来源、轨迹、相抵点、余波方向完整 |

模板：

```text
{远程物} 从 {发出点} 沿 {直线 / 弧线 / 贴地线} 飞来，镜头先看清轨迹。{防守者} 在它到身前一瞬 {侧身 / 下腰 / 伏低 / 用盾面 / 用剑脊 / 用掩体} 处理，轨迹擦过衣摆或撞上 {盾面 / 柱子 / 地面 / 兵器}，发出明确反馈。下一拍人物借避线结果继续前冲、换位或回击。
```

## 5. 法术 / 气波 / 异能防守

法术防守要写前摇、来源、轨迹、相抵、人物受力和余波。防守不是让角色站着发光。

| 来势 | 防守方式 | 写法重点 |
|---|---|---|
| 直线气波 / 掌风 | 下腰避波、侧身让线、护体半弧承压 | 雨丝、尘土、衣摆被同方向压开 |
| 火焰 / 雷光 | 护体光面、符纸引偏、法器承接 | 光源照脸和衣料，不能满屏白光 |
| 剑气 / 光刃 | 横剑相抵、侧翻让线、阵纹卸力 | 相抵点窄，余波像切开水雾 |
| 巨掌 / 法相压迫 | 后撤蓄力、双手顶守、阵纹承压 | 人物脚下裂纹从承压点展开，不凭空炸坑 |
| 精神压迫 / 幻术 | 定神、闭眼、咬住呼吸、法印护心 | 表情和呼吸承压，不只加粒子 |
| 空间位移攻击 | 预判落点、转身架住、错步避开 | 位移前后有残留光 / 风压，落点清楚 |

法术防守招式名速查：

- 护体：半弧护体、罡气护面、符阵护身、剑意护线、金钟罩影、灵盾承压。
- 避式：下腰避波、侧翻避刃、伏地穿风、踏步错位、空中折身、绕柱避光。
- 化式：法器引流、阵眼卸力、符纸带偏、剑气分流、掌风反卷、借地散劲。
- 反式：护体反震、结界回推、符阵反压、借波回掌、错位反咒、法相抬爪。

模板：

```text
{攻击者} 的 {法术 / 气波} 从 {掌心 / 法器 / 阵纹 / 剑身 / 法相} 生长后压来，轨迹清楚。{防守者} 不站着发光，先用 {下腰 / 侧身 / 撤步 / 双手起印 / 法器横架} 处理来势；相抵或擦身帧 0.12-0.2 秒短停，护体面向内凹、衣摆和发丝被压开，脚下水面 / 尘土 / 阵纹沿受力方向震动。下一拍力量被 {卸到地面 / 带偏到柱侧 / 反推回去 / 收进法器}，末帧双方重新拉开或快速接近进入下一招。
```

气波下腰避让模板：

```text
气波贴着上半身高度横压过来，{防守者} 在气波到身前一瞬后仰下腰，肩线和脸部避开中心线，衣摆、发丝和雨丝被气波从胸前掀过；气波擦过背后 {石柱 / 木门 / 水面}，留下清楚受力反馈。下一拍 {防守者} 借腰腹回弹起身，脚掌蹬地反冲，不是原地摆姿势。
```

## 6. 空中 / 飞身防守

人物可以飞，但飞行动作也要有力源、轨迹、避线和落点。

| 来势 | 防守方式 | 写法重点 |
|---|---|---|
| 空中下劈 | 侧翻避线、横剑架落、落地滑退 | 起跳点、空中转体、落地水花清楚 |
| 地面气波上冲 | 腾身避开、空中折腰、落地反压 | 气流把衣袍向上掀，不随机漂浮 |
| 飞身扑击 | 错步让开、肩背滚身、抓梁换位 | 不让两人空中穿模 |
| 高处坠压 | 撤步让落点、盾面顶守、滚地脱离 | 落点有尘土 / 水花 / 地面短裂纹 |

模板：

```text
{攻击线} 从 {上方 / 下方 / 侧面} 压来，{防守者} 的飞身不是漂浮，而是从 {脚掌蹬地 / 借柱 / 借墙 / 法术托举} 起势。空中用 {侧翻 / 折腰 / 旋身 / 横架} 避开危险线，衣袍和发丝按气流方向拖动。落地帧脚掌踩实，水花 / 灰尘 / 石粉从落点散开，末帧回到可继续对招的距离。
```

## 7. 多人防守与保护

多人防守不能全员一起挡。每一拍只保一个主威胁和一个主防守点。

| 结构 | 防守写法 |
|---|---|
| 1v2 | A 先攻，B 后半拍补线；主角先架 A，再侧身让 B，借 A 的位置挡住 B |
| 1v3 | 三点围压里只处理第一威胁，第二人因挡线停步，第三人补位到下一拍 |
| 1v多 | 防守目标是开缺口，不是每个人都打倒 |
| 护人 | 先写被保护者位置，再写主角身体 / 盾 / 法术挡在危险来源和被保护者之间 |
| 突围 | 用格挡、推线、道具遮挡制造一条通路，末帧停在缺口出现 |

多人模板：

```text
这段多人防守只保一条主威胁线。{A} 从 {方向} 先攻，{防守者} 用 {守式 / 避式 / 化式} 处理 A 的来势；{B} 和 {C} 只因 A 被带偏、路线被挡或余波遮挡产生停步 / 抬臂 / 让线的二级反应。下一拍才轮到 B 或 C 补位，不允许全员同步后退或一起挥动。
```

## 8. 防守提示词审核清单

输出前逐条检查：

- 是否写清攻击线来自哪里。
- 是否写清防守发生在来势到身前，而不是提前摆好姿势。
- 守式是否有相抵点，避式是否有避开的危险线，化式是否有卸力方向。
- 肉搏是否看见前臂、肩肘、腰胯、脚步和衣料反馈。
- 冷武器是否避开刀锋 / 斧刃 / 枪尖 / 链刃危险平面，单手武器是否有辅助手。
- 柔兵是否只有一条主弧线，并有收势。
- 法术是否有来源、轨迹、护体 / 相抵点、人物受力和余波方向。
- 空中动作是否有起跳点、空中避线和落地点。
- 防守后是否有反击、脱离、抢位、救人或开缺口的结果。
- 是否避免血腥、具体损伤、明星脸、穿模、肢体畸形、满屏白光和随机粒子。

## 9. 一句话结论

防守招式的核心不是“挡住了”，而是“危险线怎么来、人物怎么避或承受、力量落在哪里、下一拍怎么反回去”。写清这四件事，武戏才会有真正的一来一往。


# 起式 / 亮架 / 开打前姿态专项库

用途：给打斗开始前补“起式”。起式不是第一招，也不是静态摆拍，而是开打前的身体、心理、距离和节奏准备：谁先动，谁在等；谁压迫，谁诱敌；这场从文戏慢慢转武戏，还是情绪炸开后一拍入战。

资料归纳来源：综合武术 stance / kamae / guard 资料、舞台打斗 rhythm / intention 资料、影视动作写作 fight scene pacing 资料。共同结论是：姿态决定平衡、攻击、防守、距离和心理意图；打斗节奏必须服务角色情绪，而不是平均快。

外部资料速记：

- 武术 stance：姿态关乎脚位、重心、躯干和攻防转换；开合、长短、宽窄会改变稳定性、机动性和暴露面。
- 日本剑术 kamae：构え不只是架势，还包含心理准备和战略姿态；中段、上段、下段、八相、胁构等分别表达守中线、强攻、诱敌、蓄势或藏锋。
- 拳击 / 泰拳 guard：护架同时管理平衡、视野、距离、攻防转换；手位、肘线、下颌、脚步决定能不能立刻动。
- 舞台 / 影视打斗：每个动作来自冲动并产生后果；节奏从呼吸开始，情绪升高时加速，犹豫、停顿和沉默也能制造张力。
- 对招开局：双方通常不会“直接打中”，而是先通过架势、诱敌、封线或逼迫制造 opening，再进入第一触点。
- 写作层：好打戏要让动作清楚、冲突升级、暴露角色心理；起式正是把“心理”翻译成“身体”的第一步。

## 1. 起式的功能

起式必须至少完成下面 2 项，重要打戏最好 4 项全有：

1. 亮身份：门派、流派、修为、兵器习惯、人物性格。
2. 抢距离：远距、中距、近距、贴身、上下层、出口和危险线。
3. 定情绪：怒、惧、忍、疑、冷、悲、疯、护人、诱敌。
4. 给第一触点：拳线、剑线、刀线、枪尖、气波、阵纹、扑击路线。

通用链条：

```text
交代空间 -> 情绪定速 -> 起式亮架 -> 试探破静 -> 第一触点 -> 对招交换
```

禁止：

- 只写“摆好架势”，没有手、脚、眼、呼吸、距离。
- 角色怒到失控，却慢慢摆一个仪式化长起式。
- 文戏刚转武戏，却突然爆招，不给呼吸、停顿和眼神变冷。
- 起式和兵种不匹配：拳脚用剑架，重兵用轻兵飘架，法术不写前摇。

## 2. 按情绪决定起式节奏

起式不是固定慢动作。情绪决定节奏，节奏决定镜头。

| 情绪状态 | 起式节奏 | 身体表现 | 镜头 / 剧本重点 | 易错 |
|---|---|---|---|---|
| 冷静 / 高手压场 | 慢起式 | 呼吸稳，重心低，手脚少动，视线锁定 | 静到有压迫，慢慢抢中线 | 慢但空，没有威压 |
| 激怒 / 被挑衅 | 快起式 | 肩线前压，步子抢半拍，手先到护架 | 一拍亮架，二拍进入第一触点 | 直接乱冲，失去招法 |
| 恐惧 / 防御 | 碎起式 | 后撤半步，手先护中线，眼神找出口 | 护架不稳，呼吸乱，但还要可读 | 写成完全无能 |
| 忍怒 / 压住杀意 | 慢到快 | 先静止，手指收紧，忽然短步入线 | 停顿很值钱，破静那一下快 | 全程匀速 |
| 文戏转武戏 | 慢起式 | 话停，眼神变冷，衣袖 / 手指 / 兵器先动 | 从台词余音接身体前摇 | 话还没落就爆炸 |
| 护人 / 救人 | 快且低 | 身体先挡到危险来源和被护者之间 | 起式目标是挡线，不是炫技 | 忘了被保护者位置 |
| 诱敌 / 试探 | 慢且松 | 故意露空门，脚步轻移，眼神不急 | 让对手先动，自己等反式 | 漏洞不清楚 |
| 绝望 / 决死 | 慢后重 | 呼吸沉，重心扎地，动作少但结果重 | 起式像“下决定”，不是发疯 | 只写大喊 |
| 心虚 / 伪装 | 断续起式 | 手架慢半拍，脚尖退，视线躲闪 | 起式暴露人物心理 | 只写台词不写身体 |
| 疯狂 / 失控 | 快且破 | 架势不完整，重心前扑，动作有危险感 | 可以乱，但第一攻击线必须清楚 | 满屏乱动 |

快捷规则：

- S2 文戏余波 / 不满：慢起式，重点是气氛变化。
- S3 紧张 / 兴奋：中速起式，试探明显。
- S4 愤怒 / 恐惧 / 绝望：快起式或慢到快，破静点要硬。
- S5 决死 / 崩溃 / 狂怒：极短起式或仪式化重起式，取决于角色是否还能控制自己。

## 3. 徒手 / 肉搏起式

徒手起式看五件事：下颌、肩肘、中线、腰胯、脚步。

| 起式类型 | 适用情绪 / 场景 | 身体写法 | 第一触点 |
|---|---|---|---|
| 抱门守中 | 冷静、防守、高手试探 | 下颌微收，双手护住中线，肘不外飘 | 前臂相触、拍开来拳 |
| 低重心短桥 | 贴身肉搏、走廊、电梯 | 膝微屈，肩沉，前脚试探半步 | 肩靠、短拳、膝线压进 |
| 虚步引手 | 诱敌、门派切磋 | 前脚虚点，前手轻探，后手藏力 | 对方抢入，自己借触换手 |
| 开肩压步 | 激怒、主动压迫 | 肩线前压，步子抢线，手先架后打 | 直拳 / 掌根进入上盘 |
| 散打护架 | 现代制服、硬派突围 | 手高，肘收，脚步小跳，重心可前可后 | jab / 侧踹 / 抱摔前接触 |
| 少林低架 | 少年高手、门派感 | 手型清楚，低重心，小跳步，眼神亮 | 低位截腿、快拳破线 |
| 太极云手起势 | 化劲、借力、长辈高手 | 手臂圆转，脚跟轻换，腰胯带手 | 拨开来势、带偏重心 |

模板：

```text
{角色} 没有立刻出拳，而是先把 {下颌 / 肩 / 肘 / 腰胯 / 前脚} 调到 {起式类型}。情绪是 {情绪词+S级}，所以起式节奏为 {慢 / 中 / 快 / 慢到快}；{呼吸 / 眼神 / 手指 / 衣料} 先出现变化。下一拍，{第一触点} 在 {中线 / 侧线 / 下盘 / 贴身距离} 发生。
```

例：

```text
【情绪】忍怒 S4，慢到快
【起式】他话音落下后没有追问，只把下颌压低，左脚虚点半步，双肘收进中线。右手指节慢慢收紧，袖口停住不晃。下一拍，他忽然抢进半步，前臂先与对方手腕相触，短拳才贴着中线压出。
```

## 4. 武器起式

武器起式先问三件事：兵器在哪里，锋线朝哪里，距离够不够。

| 兵器 | 起式 | 节奏 | 写法重点 | 第一触点 |
|---|---|---|---|---|
| 剑 | 起剑 / 横剑守中 / 剑尖低垂 | 慢、中、快都可 | 剑尖高度决定攻守，腕线要清 | 剑脊相抵、点刺试探 |
| 刀 | 横刀压线 / 刀背护身 / 拖刀蓄势 | 中到快 | 刀更重，起式要有肩背和脚步 | 刀背相架、斜压逼退 |
| 枪 / 矛 | 枪尖压线 / 后手蓄劲 / 半步探枪 | 慢到快 | 枪尖先控制距离，不急扎 | 枪尖点线、杆身相拦 |
| 棍 | 棍尾点地 / 横棍封门 / 立棍护身 | 中 | 棍要有两端和回弹 | 棍身相撞、封住横扫 |
| 扇 | 扇骨半开 / 袖中藏扇 / 开扇遮线 | 慢 | 扇适合文戏转武戏和藏锋 | 扇骨点腕、扇面遮视线 |
| 双兵 | 左右错架 / 一守一攻 | 快 | 双手分工清楚，不能糊成残影 | 左架右点、轮替进线 |
| 重兵 | 肩扛蓄势 / 拖地起势 / 双手沉握 | 慢后重 | 重量、落点和脚步必须清楚 | 对手先让落点，不能硬顶 |
| 柔兵 | 收势护身 / 盘绕成圈 / 一条主弧线 | 慢到快 | 先收再放，主弧线只保一条 | 缠住兵器或带偏手腕 |

剑式参考：

- 中线守剑：剑尖指向对方胸口或眼线，冷静、克制、高手。
- 高位起剑：剑在肩上或头顶，进攻性强，适合激怒或压场。
- 低位藏剑：剑尖下垂，像让出上盘，适合诱敌或后发。
- 侧身藏锋：剑身收在身侧，适合文戏转武戏、突然亮刃。

模板：

```text
{角色} 的 {兵器} 先从 {鞘中 / 身侧 / 地面 / 袖中 / 背后} 进入起式：{锋线 / 枪尖 / 棍端 / 扇骨 / 链身} 对准 {中线 / 下盘 / 出口 / 对方兵器}。情绪是 {情绪词+S级}，因此节奏 {慢 / 快 / 慢到快}；第一触点不是大招，而是 {相抵 / 试探 / 封线 / 逼退 / 引诱}。
```

## 5. 法术 / 异能起式

法术起式必须写前摇，不能凭空爆。前摇包括：来源、手势、介质、环境先兆、对手感知。

| 起式类型 | 适用 | 写法重点 | 第一触点 |
|---|---|---|---|
| 掐诀聚气 | 仙侠、符修、道法 | 指节、腕线、袖风，灵气先聚不先炸 | 掌心 / 指尖出现窄线 |
| 阵纹起亮 | 阵法、结界、群战 | 脚下或四角先亮，边界清楚 | 阵边与对方攻击相抵 |
| 法器前摇 | 宝印、镜、铃、幡、飞剑 | 法器先动，人物后动 | 法器挡线、引流、放出第一束光 |
| 元素先兆 | 雷、火、冰、风、水 | 环境先变：湿气、温度、光影、尘土 | 元素沿固定轨迹生长 |
| 精神压迫 | 幻术、神识、心魔 | 表情、呼吸、耳鸣、视线迟滞 | 对方动作慢半拍或失焦 |
| 空间位移 | 闪现、缩地、换位 | 原地风压 / 残影 / 落点预告 | 出现在侧线，不随机穿模 |
| 护体起式 | 防守、保护、硬接 | 护体面先内凹或贴身成弧 | 与气波 / 剑气相抵 |

节奏规则：

- 冷静法师：前摇慢，动作少，边界清楚。
- 激怒法师：前摇短，手势快，但来源和轨迹不能省。
- 文戏转法术：先让环境轻变，再让手势成立。
- 失控爆发：可以快，但必须有一个能看见的引爆点。

模板：

```text
{角色} 的法术起式先从 {手指 / 法器 / 阵纹 / 眉心 / 脚下} 出现，{环境先兆} 比攻击更早出现。情绪 {情绪词+S级} 决定前摇 {短 / 中 / 长 / 断续}；下一拍，第一束力量沿 {直线 / 弧线 / 地面阵线 / 空中落线} 与 {对手护架 / 兵器 / 地面 / 结界} 形成第一触点。
```

## 6. 对峙起式

对峙起式是“谁先破静”。重点不是动得多，而是观众能看出局势变化。

| 对峙结构 | 起式写法 | 适用 |
|---|---|---|
| 高手对高手 | 双方都慢，只有脚尖、剑尖、呼吸变化 | 切磋、宿敌、师徒 |
| 强者压弱者 | 强者少动，弱者碎动后退 | boss 压场、恐惧 |
| 弱者反杀 | 弱者先防守低架，眼神忽然定住 | 翻盘、觉醒 |
| 恋人 / 师徒反目 | 文戏停顿长，起式慢且克制 | 情感重场 |
| 群战开局 | 先给包围圈，再给主角低重心或兵器封门 | 1v多 |
| 偷袭前一秒 | 被动方起式不完整，只能临时护线 | 突然转武 |

模板：

```text
【对峙】双方先不出招。镜头交代 {距离 / 出口 / 阻隔 / 旁人位置}。
【情绪定速】{A情绪+S级} 对 {B情绪+S级}，决定起式节奏为 {慢压慢 / 快破静 / 慢到快 / 碎乱防守}。
【起式亮架】A 用 {起式A} 控制 {中线 / 下盘 / 侧线 / 空间}; B 用 {起式B} 回应。
【第一触点】第一下只做 {试探 / 相抵 / 让线 / 封路 / 护人 / 逼退}，再进入正式对招。
```

## 7. 全局对招接入规则

凡写“对招交锋 / 双方过招 / 1v1 boss战 / 文戏转武戏”，必须在出招前补一行起式：

```text
【情绪定速】A：{情绪+S级}；B：{情绪+S级}；起式节奏：{慢 / 中 / 快 / 慢到快 / 碎乱}
【起式亮架】A：{徒手 / 武器 / 法术 / 对峙起式}；B：{回应起式}
```

对招链条升级为：

```text
交代空间 -> 情绪定速 -> 起式亮架 -> 试探破静 -> 第一触点 -> 攻防交换 -> 命中反馈 -> 收势重组
```

常用映射：

| 对招状态 | 推荐起式 | 节奏 |
|---|---|---|
| 冷静切磋 | 中线守架、虚步引手、剑尖低垂 | 慢 |
| 激怒开打 | 开肩压步、高位起剑、短前摇法术 | 快 |
| 文戏转武戏 | 手指停住、袖中藏锋、扇骨半开、眼神变冷 | 慢到快 |
| boss 压迫 | 少动压线、重兵拖地、法相半显 | 慢且重 |
| 弱者防守 | 抱门守中、后撤碎步、护体半弧 | 碎乱 |
| 护人开战 | 身体先挡线、盾 / 手 / 法器先到被护者前 | 快且低 |
| 诱敌反打 | 低位藏剑、虚步露空、松架等对手进线 | 慢 |
| 绝望决死 | 扎地、沉肩、双手握紧、阵纹从脚下爬开 | 慢后重 |

## 8. 剧本与分镜模板

### 剧本模板

```text
【情绪】{角色A 情绪+S级} vs {角色B 情绪+S级}
【起式】{角色A} 因为 {情绪 / 目的}，以 {快 / 慢 / 慢到快 / 碎乱} 的节奏进入 {起式类型}；{角色B} 用 {回应起式} 抢 / 守 / 让 {距离或中线}。
【第一触点】双方先在 {拳线 / 兵器相抵点 / 法术边界 / 侧线 / 下盘} 短促接触。
【正式对招】再接 {招式名 / 攻防交换 / 命中反馈 / 收势重组}。
```

### 分镜模板

```text
【镜头】先交代 {空间 / 距离 / 出口 / 危险来源}，再用 {缓推 / 短跟 / 固定停顿 / 急推} 捕捉起式。
【情绪定速】{情绪+S级} -> 起式节奏 {慢 / 中 / 快 / 慢到快 / 碎乱}
【起式亮架】{手脚 / 兵器 / 法术 / 护架 / 眼神 / 呼吸} 的变化清楚可读。
【第一触点】{相抵 / 让线 / 试探 / 封路 / 护人}，停 0.1-0.2 秒。
【进入对招】再写位移、攻击、环境震荡和音效。
```

## 9. 安全和 AI 生成注意

- 起式可以有“杀意、压迫、决死感”，但分镜 prompt 不写具体杀伤部位和血腥结果。
- 现代写实题材尽量用“护架、制服、逼退、放倒、打开缺口”，少用真实武器名。
- 武器起式写“锋线、相抵、封路”，不要写具体刺入或破坏身体。
- 法术起式写“前摇、来源、轨迹、相抵”，不要满屏白光和随机粒子。
- 起式越复杂，正式对招越要减法；一镜头不要同时塞多种手型、脚步、武器和法术。

## 10. 一句话结论

起式的核心是“这场打斗怎么从情绪变成身体”。先定情绪，再定节奏；先亮架，再第一触点；最后才进入招式和命中反馈。
