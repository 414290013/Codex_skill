# Xianxia Industrial Choreography Knowledge

Use this knowledge base for玄幻、修真、修仙、神通、元神出窍、法天象地、破碎虚空、渐变变身、御剑、高速近战、境界威压、3A动漫质感、东方神漫三渲二动作戏。

This file is a functional classification of reusable xianxia storyboard rules. It is not a transcript and does not preserve original source names, platforms, paths, or fixed external style-copy requirements.

## Core Use Rule

Do not apply this file to pure文戏. Use it only for rows classified as `修仙打戏`, `玄幻修真动作`, or action rows inside `混合转场`.

Default goal: short-drama compactness + high-speed xianxia readability.

- Keep the main plot event moving forward.
- Make supernatural rules visible through position, action, VFX source, and environmental feedback.
- Avoid slow scenic padding unless the user asks for full cinematic detail.
- Never turn修仙打戏 into ordinary punches with decorative glow.

## High-Speed Xianxia 15-Second Module

When the user asks for fast xianxia combat, anime impact, short-video pacing, or high-density action, a 15-second action unit may use a **9-beat fast-cut module**. This is optional and only for pure action/high-speed action blocks, not dialogue drama.

Recommended 9-beat structure:

1. `0.00-1.67s 黄金钩子`: immediate attack, sudden block, or supernatural reveal. Use extreme low angle, close impact anchor, or wide pressure reveal.
2. `1.67-3.34s 闪避`: side-step, blink movement, sword-flight dodge, sleeve turn, or terrain-assisted evasion.
3. `3.34-5.01s 反击`: slash, palm seal, sword-finger command, talisman throw, or weapon counter.
4. `5.01-6.68s 格挡/破招`: weapon clash, spell shield, qi deflection, sleeve guard, formation interruption.
5. `6.68-8.35s 碰撞反馈`: sparks, ink particles, shock ring, ground cracks, cloth/hair pull, weapon recoil.
6. `8.35-10.02s 受击/失衡`: safe impact feedback, body displacement, stance collapse, weapon tremor, defensive retreat.
7. `10.02-11.69s 追击`: forward chase, airborne step, sword-flight pursuit, clone lane advance, camera chase.
8. `11.69-13.36s 小高潮`: sword qi release, giant palm pressure, spirit strike, formation burst, space-rip tremor.
9. `13.36-15.00s 衔接中段`: end inside an unfinished action, spell, cloth motion, particle flow, or camera movement if the next unit continues.

Use this module only when it improves rhythm. If dialogue, emotional reaction, or plot clarity needs more breathing room, use the normal 2-3 second shot grammar or a 5-row compact short-drama action block.

## Camera Variation Rule

Action units must avoid repeated shot shape.

- Do not use the same `景别/构图` and same camera route in consecutive action rows.
- Good contrast pairs: wide pressure reveal -> hand/weapon close-up; close impact -> aerial wide; fixed lock -> fast chase; push-in -> orbit; low-angle close -> overhead drop.
- For a continuing action across big storyboards, carry speed, direction, light, cloth, particles, and body state forward, but change the camera type or angle enough that the next unit does not feel like a frozen repeat.
- Every fast action row should state the camera's start, route, and end, not only `快速跟拍`.

## Action Continuity Chain Lock

Xianxia action rows must form one continuous cause-and-effect chain. A row is invalid if it looks exciting by itself but resets the action timeline.

Hard rule:

- The `起始帧` of row N must visibly inherit the `终止帧` of row N-1.
- Do not let row N begin earlier than row N-1 ended. No temporal rewind such as `row 1 ends at weapon collision` then `row 2 starts with the sword raising to strike`.
- If a new attack begins, write the transitional recovery or redirection beat first: recoil, foot slide, wrist roll, body turn, blink-step, weapon rebound, or spell deflection.
- Every row must leave a `下一镜继承状态`: weapon position, body momentum, distance, energy pressure, debris direction, cloth/hair motion, camera motion, and active supernatural state.
- The next row must use that inherited state as its first visual anchor.

Continuity chain template:

`承接上一镜[终止状态] -> 本镜[动作变化] -> 接触/能量变化 -> 反馈传导 -> 本镜终止状态/下一镜继承点`

Examples of valid inheritance:

- Previous end: `双刃接触点压缩成红黑能量核`. Next start: `微距锁住同一个能量核，剑脊与刀刃仍咬合，双方手腕被反震压低`.
- Previous end: `环状冲击波贴地扩散`. Next start: `低机位跟随同一圈冲击波扫过第一根石柱，石柱表面先裂出放射纹`.
- Previous end: `石柱群裂纹贯通`. Next start: `高角度俯视裂纹从石柱根部传到地脉，碎石先失重悬浮，再被反向气流托起`.

Invalid jumps:

- `交错碰撞` -> `重新劈下`.
- `接触点压缩` -> `巨大风暴已经形成` without expansion.
- `冲击波扫过` -> `所有石柱瞬间化粉` without crack, vibration, dust, and force route.
- `重力颠覆` appears without prior pressure field, ground lift, debris suspension, or air-direction change.

## Professional Fight Director Lock

VFX cannot replace martial choreography. A修仙打戏 row must first read as a designed attack-defense beat, then receive supernatural enhancement.

Required beat fields for every weapon/combat row:

- `继承状态`: what exact body/weapon/energy state is inherited from the previous row.
- `战斗距离`: distance before and after the beat; explain any sudden close/far change.
- `攻线`: attack line, edge direction, target line, body lead, foot/air step route.
- `守线`: guard, parry, deflection, dodge, bind, shield, sleeve guard, or spell interception.
- `接触帧`: clear physical or energy contact anchor. The audience must know what met what.
- `模糊帧`: where blur/smear appears and why: acceleration smear, weapon-edge smear, contact-rim blur, afterimage, or feedback trail.
- `反震/反馈`: recoil, weapon vibration, wrist drop, body rotation, stance slide, target displacement, cloth/hair/debris response.
- `退场状态`: what position, momentum, distance, and energy state the next row inherits.

Ban vague montage substitutes in action rows:

- Do not write `上百次折返`, `疯狂交错`, `无数次碰撞`, `肉眼难辨`, `满屏残影` as the main action design.
- If many exchanges occur, select **3-5 readable representative beats** and make the rest background afterimages or sound texture.
- Do not summarize an entire combo as one row unless the row also names the visible attack line, defense, contact frame, smear frame, and exit state.

## Single-Shot Action Density And VFX Stacking Cap (单镜动作节拍密度与特效叠加上限)

修仙打戏单镜常因"塞太多"而崩坏：AI 生成时会糊成抽象残影、特效相互冲淡、攻防关系不可读。此规则按"一拍一意"的武术动作设计原则锁死。

**A. 动作节拍上限（每镜 ≤ 1 个清晰回合）：**

- 单镜独立动作事件 ≤ 1 个清晰回合 (一次劈砍 + 一次格挡 = 1 个回合 / 一次突刺 + 一次错身 = 1 个回合)。
- 严禁单镜出现 `残影连续折返三次`、`连续闪避五招`、`一镜内攻防转换三次`、`双方在半空连续折返N次` 等高密度节拍堆叠。
- 多回合连续打斗必须拆成多个连续 row，每 row 一个回合，按 `承接上一镜终止状态 → 本镜单回合 → 终止状态供下一镜继承` 的链条推进。
- 违规示例：`快速侧移跟拍；起始帧红黑两道残影同时拔地而起；关键帧@A大开大合挥出重斩，@B诡异侧身以黑刀毒蛇般突刺，双刃交错产生局部模糊与空气爆痕；终止帧残影在半空连续折返三次`（一镜塞了 起势+劈砍+突刺+错身+三次折返 = 5 个回合，AI 必糊）。
- 合规改写：拆成两 row：①起势+交错错身一次；②咬合后续。

**B. 特效叠加上限（每镜核心视觉锚点唯一）：**

- 单镜核心视觉冲击 = 1 个唯一锚点（参见 `Professional Fight Director Lock` 中的 `本镜视觉冲击核心`）。
- 严禁同镜叠加两个以上同等强度特效，例如同时写 `真空波纹球 + 刺目极光爆发 + 空间裂缝撕裂 + 环状冲击波` 四连发。
- 必须保留视觉峰值的递进留白：全场打斗的最高峰特效（终极绝杀光爆 / 刃尖刺目极光 / 重力颠覆）必须保留至最后 1-2 row 引爆，**禁止在中段镜头提前用尽**。
- 中段镜头使用 1 个锚点 (如真空波纹球 OR 火星迸溅 OR 局部能量压缩光核)，最终镜头才允许叠加 2 个锚点形成爆点。

**C. 兵器力学清晰度优先（慎用抽象描述）：**

- 打斗镜头优先保留可清晰渲染的物理元素：`金属对撞火星`、`兵刃压缩灵力`、`兵刃弯曲形变`、`手腕青筋暴起`、`虎口震裂`、`刃身震颤波纹`。
- 慎用抽象风格化描述：`方向性抖动线`、`灵力乱流取代火星`、`抽象笔触残影`、`通感视效`。AI 对抽象描述的渲染稳定性远低于物理描述，常导致画面崩坏或风格漂移。
- 若必须用抽象元素，**必须搭配至少一项明确物理元素作为锚定**（例：`兵刃相抵迸发火星 + 局部压缩灵力波纹` ✅；单写 `灵力乱流取代常规火星` ❌）。

**D. 单镜自检三连问（每个打斗 row 输出前必跑）：**

1. 本镜动作节拍是否 ≤ 1 个清晰回合？✅ / ❌ 拆 row 重写
2. 本镜核心视觉锚点是否唯一？✅ / ❌ 删特效或留至高潮
3. 是否保留了"留白"给后续高潮？（即终极视觉峰值是否还在后段未释放）✅ / ❌ 调整峰值分布

三问全部通过方可输出该 row。

Professional weapon-duel chain:

1. `入距`: step/blink/air-step enters measure; weapon line is clear.
2. `首触`: blade/weapon binds or meets guard; local contact blur and hit-stop.
3. `滑刃/换线`: edge slides, wrist turns, guard shifts, body angle changes.
4. `破线/反击`: one side opens a line or forces a deflection.
5. `重碰撞`: final clash compresses energy; both sides recoil or displacement begins.
6. `传导`: pressure travels into ground, air, debris, weather, and background objects.
7. `续势`: next row starts from recoil, sliding stance, airborne drift, or unfinished energy expansion.

Blur-frame design for xianxia weapon combat:

- Clear setup frame: readable stance, distance, attack direction.
- Acceleration smear: along sword/saber/halberd path, not across the whole body.
- Contact frame: local blur only around the weapon edge or energy rim; keep the contact anchor sharp.
- Feedback smear: afterimage on retreat, sliding feet, robe drag, thrown debris, or energy trail.
- Camera shake: only on contact or pressure-wave hit, brief and directionally motivated.

Continuity and穿帮 checks:

- If characters are in midair, they cannot carve ground trenches with their feet until a landing or energy-pressure contact is shown.
- If the safety lock says no recognizable face, do not write a clear full-face close-up; use eye-line shadow, jawline silhouette, hand/weapon/robe close-up, or backlit profile.
- If a weapon is locked against another weapon, the next row cannot show the same character freely swinging unless a slide, rebound, disengage, wrist roll, or body turn is shown first.
- If a character is knocked back, maintain screen direction and distance; do not teleport them to the opposite side without a visible route.
- If gravity reverses, first show light debris, dust, rain, cloth, hair, and pebbles lifting before giant stones or pillars move.

## Post-Impact Separation And Axis Lock

Use this when a修仙打戏 contains `对冲 / 碰撞 / 对轰 / 反震 / 倒飞 / 分开 / 稳身 / 对峙`.

The shot block must define a fight map before the first row:

- `坐标系`: stage left-to-right is X, depth is Y, height is Z when airborne matters.
- `180度轴线`: usually the line between fighters; keep the camera on one viewing side unless the row explicitly uses a motivated crossing move.
- `对冲起点`: both fighters' opening coordinates, facing direction, screen direction, and height.
- `碰撞点C`: exact contact coordinate and height, e.g. `C(5.0,1.0,Z=1.8m)`.
- `反震归位`: after collision, each fighter rebounds back along their incoming vector unless a named technique redirects them. If A enters from screen left toward +X, A should separate back toward screen left; if B enters from screen right toward -X, B should separate back toward screen right.
- `最终对峙点`: write both final coordinates, facing directions, distance across the new gap, and what obstacle/energy remains between them.

Post-impact row order:

1. `咬合/压缩`: weapons, spells, or force edges are still locked at C; hit-stop and local contact blur appear here.
2. `脱锁/爆开`: show blade rebound, wrist drop, body rotation, or energy-shell rupture before anyone flies away.
3. `半空倒射`: characters travel away from C with feedback smear, cloth drag, weapon vibration, and readable vectors.
4. `落地触地`: feet, knees, weapon tip, or energy pressure touches the ground first. Only after this can trenches, skid marks, or cracked landing rings appear.
5. `犁退/稳身`: sliding distance, stopping stance, guard recovery, and final screen side are explicit.
6. `对峙继承`: next row inherits both coordinates, distance, eye-line toward opponent, central damage, active particles, and altered gravity state.

Hard failures:

- `倒飞至两端` without naming who returns to which side and what coordinates they land on.
- A fighter crosses through the collision point and appears on the enemy's original side without a dodge, throw, teleport, or camera-side explanation.
- Both fighters are shown `画左` or `画右` after separation unless the composition is deliberately stacked in depth and Y/Z coordinates prove it.
- The row says `战靴犁地` while the previous row still has the character airborne and no landing/energy-contact frame has occurred.
- Background baseline starts with final-state damage (`石柱粉碎`, `重力颠覆`) before the shot actually causes it. Put only the opening state in the baseline, then update damage inside rows.

## Three-Render + 3A Physical Texture

Use this as optional visual language when the user wants high-end xianxia animation, industrial quality, game-CG texture, or东方神漫质感.

Reusable traits:

- Stylized three-render look with clean outline, high contrast, saturated highlights, and deep shadow blocks.
- 3A CG physical feedback: cloth inertia, hair swing, debris, rain, dust, fluid, weapon vibration, ground deformation, light reflection.
- Ink-style supernatural VFX can replace generic glow: ink arcs, ink vortexes, black/white particle bursts, transparent pressure ripples, silhouette afterimages.
- Light must interact with the scene: rim light on rain, reflected light on metal, soft shadow under robes, backlight through dust or mist.

Avoid low-grade VFX:

- Generic glowing ball with no source.
- Whole-body glow that hides action.
- Full-screen explosion with no contact anchor.
- Perfectly symmetrical energy effects.
- Colorful beams added without power-system reason.
- Static standstill casting.

## Xianxia Action Physics

Every supernatural action still needs physical logic:

- `起势`: center of gravity, foot pressure, shoulder/waist rotation, hand seal or weapon route.
- `执行`: acceleration route, blink step, sword-flight path, sleeve/robe drag, weapon arc, VFX source.
- `接触/碰撞`: impact anchor, spell collision edge, shield deflection point, ground contact, local blur.
- `反馈`: recoil, stance change, cloth/hair pull, debris/rain response, particle trail, environmental damage.
- `结果`: distance change, advantage shift, formation opened/broken, enemy displaced, next threat set.

For修仙法术, never write only `灵力爆发`. Specify:

`gesture/weapon source -> energy path -> collision or target -> environment response -> aftermath state`.

## Combat Distance And Blocking

For fight rows, the站位基线 should include combat distance when relevant. Use exact but cinematic distances; do not let enemies stand贴脸 unless the action is a grappling/close-contact beat.

Practical defaults:

- Dagger/short blade close range: `2.0-3.0m`.
- Sword/saber range: `3.5-5.0m`.
- Spear/halberd range: `6.0-8.0m`.
- Ranged spell range: `12.0-18.0m`.
- Spirit-avatar attack range: `18.0-25.0m`.
- Giant avatar / law-form attack range: `35.0m+`.

Distance changes by action:

- Attack wind-up: distance closes by `0.5-1.0m`.
- Strike execution: distance holds or closes sharply.
- Heavy feedback: target is displaced by `2.0-5.0m` or more if stylized.
- Evasion: screen position changes while distance may hold.
- Counterattack: distance closes by `0.5-1.0m`.

For group fights:

- Main character and enemies should have lanes: ground lane, midair lane, rear ranged lane, or flank lane.
- Mark nearest and farthest enemy distance.
- Avoid a flat wall of enemies. Use fan, arc, diagonal depth, or vertical layering.

## Eye-Line Lock

Characters should not look into the camera unless the user explicitly asks for direct-address style.

- Enemies look at each other's brow/weapon/hand seal/attack source.
- Allies look at the speaker, protected target, or shared threat.
- A lone character looks at a meaningful object: blade edge, ground crack, distant gate, spell core, falling rain, or enemy direction.
- Close-ups can look slightly off-lens toward the interaction target.

This prevents "AI portrait posing" and keeps the shot dramatic.

## Xianxia State Router

When these keywords appear, create a state baseline and carry it forward until the source says it ends:

- `元神 / 元婴 / 元神出窍 / 元神离体`: use the Spirit Avatar rules.
- `法天象地 / 法身 / 金身 / 巨大神像 / 巨大虚影`: use the Giant Law-Form rules.
- `破碎虚空 / 撕裂空间 / 空间裂缝 / 虚空裂隙`: use the Space-Rift rules.
- `变身 / 神纹 / 神袍 / 发丝变色 / 觉醒形态`: use the Gradual Transformation rules.
- `御剑 / 剑阵 / 万剑 / 剑气`: use weapon/spell source, trajectory, formation, collision, and aftermath.

Every active state must be represented in:

- `此时此刻背景基线` or `此时此刻站位基线` when it affects space.
- The row's `视觉+运镜`.
- The `[前景]/[中景]/[背景]` prompt layers when visible.

## Spirit Avatar / 元神 Rules

Use when a character's soul/spirit/avatar leaves the body or attacks independently.

Default baseline:

- The spirit avatar inherits the character's identity silhouette unless the user gives a separate reference.
- It appears as a semi-transparent ink or qi silhouette, not as a random glowing duplicate.
- It normally starts above the body's centerline or near the head/upper torso, then may leave the body for attack.
- Body and spirit must have a clear relation: body guides with hand seal, breath, or stance; spirit performs the supernatural release.
- If the spirit leaves the body, keep a visible trail or spatial tether so the audience understands ownership.

Stage route:

1. `启动`: body seals at dantian/chest/brow; pressure gathers.
2. `上升`: a small spirit silhouette forms above or behind the body.
3. `显现`: facial/robe silhouette clarifies; ink/qi edges stabilize.
4. `完全体`: spirit mirrors or amplifies the body motion.
5. `离体攻击`: spirit moves independently, releases spell, or strikes while the body maintains guide posture.
6. `归体/持续`: spirit returns or remains active; record state.

Avoid:

- Spirit appears with no body trigger.
- Body and spirit both randomly fire unrelated effects.
- Spirit vanishes between rows without source reason.
- Spirit overlaps the body or other VFX so badly that ownership is unreadable.

## Giant Law-Form / 法天象地 Rules

Use when a character manifests a huge avatar, law-form, dharma-body, golden body, giant god-shadow, or equivalent.

Default baseline:

- Place the giant form behind and aligned with the body unless the source says otherwise.
- It should share the body's facing direction and intent, usually looking toward the enemy, not the camera.
- The body remains the guide/controller; the giant form amplifies the action.
- Keep enough offset so the giant form does not穿模 with the body, weapons, spirit avatar, or background.

Action rule:

- Body hand seal, step, or weapon route triggers the giant form.
- Giant form performs a larger synchronized version: palm pressure, weapon swing, formation press, space tear.
- The environment must show scale: clouds bend, dust rises, rain curves, buildings/trees silhouette against the avatar.

Continuity:

- Once active, carry it across rows and big storyboards until dissolved.
- Record position, scale, facing, action state, and active VFX in the baseline.

## Space-Rift / 破碎虚空 Rules

Space-rift effects must be staged. Do not jump from hand raise to full battlefield collapse.

Stage route:

1. `空间扭曲`: transparent ripple, refraction, pressure rings, rain/dust bending toward a center.
2. `裂隙产生`: thin black/ink fissure appears, irregular edge, no generic glow.
3. `裂隙扩张`: cracks stretch, ink-like fragments peel outward, edges distort the background.
4. `引力爆发`: nearby debris, cloth, rain, and loose objects pull toward the rift.
5. `完全破碎`: larger void area opens; terrain or architecture bends/deforms safely.
6. `持续吞噬/闭合`: state continues until source says it closes.

Camera:

- Start with hand/gesture or space ripple close-up.
- Use wide angle for scale when the rift grows.
- Track debris or a weapon as it is pulled toward the rift.
- Add brief impact-synced shake only at expansion or collision beats.

Safe visual substitutions:

- Use stylized void, ink, dark mist, particle collapse, and environment deformation.
- Avoid graphic bodily harm.

## Gradual Transformation Rules

Transformation should be staged and tied to action. Avoid instant pose-change, static glow, or costume popping.

Stage route:

1. `触发印记`: brow, hand, weapon, or chest mark appears as texture/engraving, not random light.
2. `发丝/材质变化`: hair, robe, armor, or aura changes from edge to root/hem to body.
3. `衣袍/神袍显现`: original costume structure transforms in layers; fabric motion continues.
4. `全身纹路蔓延`: lines follow muscles, bones, meridians, fabric seams, or armor grooves.
5. `完全体`: final silhouette locks; environment reacts; action continues.

Hard rules:

- Maintain character identity and proportions.
- Keep action moving: transformation can happen during dash, clash, hand seal, or landing.
- Do not freeze the character in a posing stance unless source explicitly requires a ritual stillness.
- If spirit avatar or giant law-form is active, synchronize their transformation progress with the body.

## Multi-Ability Layering

Layer order for readability:

1. Body / weapon / hand seal.
2. Spirit avatar if active.
3. Attack VFX / sword qi / spell trail.
4. Giant law-form if active.
5. Space-rift / formation / background-scale effect.
6. Environment, crowd, weather, architecture.

In the prompt's picture track:

- `[前景]`: body/hand/weapon/contact anchor.
- `[中景]`: active spirit, target, spell path, combat distance, body route.
- `[背景]`: giant law-form, space-rift, weather, terrain response, formation scale.

Do not place all supernatural layers at the same depth.

## Continuity Across Big Storyboards

When an action or special state continues into the next big storyboard, preserve:

- Character stance and screen position.
- Combat distance.
- Eye-line target.
- Active state: spirit avatar, giant law-form, rift, transformation progress, formation, weapon state.
- Physical dynamics: cloth direction, hair direction, particle flow, debris motion, rain/wind direction.
- Light/weather state.
- Camera speed and movement direction, if the shot is meant to be seamless.

The next big storyboard may switch camera type or angle, but it must inherit the action state. Do not reset effects, posture, injuries, weapon position, or transformation progress.

## Short-Drama Xianxia Rhythm

Use compact rhythm by default:

- First 1-3 seconds: visual hook or threat, not slow explanation.
- Every 8-12 seconds: a small action or revelation peak when the scene is action-driven.
- Every 25-35 seconds: a larger payoff if the output spans multiple big storyboards.
- Narration should be sparse in fight scenes. Prefer action, reaction, and exact dialogue.
- Do not let a fight scene start with a long prelude if the source already contains combat.

## Repair Triggers

Rewrite the row or prompt if any of these appear:

- `站桩施法`: spell occurs while the character stands still without body route, gesture, or camera pressure.
- `泛光替代动作`: glow hides the actual gesture, weapon route, collision, or target reaction.
- `镜头重复`: consecutive rows repeat the same camera type and movement without a dramatic reason.
- `看向镜头`: action character faces the camera instead of target or interaction object.
- `贴脸站桩`: enemies are too close or static without grappling reason.
- `神通消失`: active spirit/law-form/rift/transformation vanishes without source reason.
- `层级穿模`: body, spirit, law-form, spell, and background occupy the same unclear layer.
- `瞬间变身`: transformation completes without stage route.
- `虚空一秒完成`: space-rift skips distortion, fissure, expansion, pull, and aftermath stages.
- `动作链回退`: a later row restarts an earlier strike, wind-up, or collision instead of inheriting the previous row's end state.
- `环境破坏跳帧`: pillars, ground, buildings, weather, or gravity change abruptly without visible force propagation.
- `武指缺失`: the row uses vague speed words or VFX without attack line, defense line, contact frame, blur frame, and feedback.
- `穿帮动作`: midair feet carve ground, locked weapons swing freely without disengage, characters teleport sides, safety lock conflicts with face close-up, or gravity changes heavy objects before light debris reacts.
