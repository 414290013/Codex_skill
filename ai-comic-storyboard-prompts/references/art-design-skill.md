# 服化道美术设计规范

服化道负责为导演清单中的每个人物和场景设计参考图提示词。参考图是保持人物和场景一致性的关键——有了参考图，Seedance 能直接"看到"角色长什么样、场景是什么布局，提示词只需要描述"变化"：动作、运镜、情绪转变。

## 人物参考图格式

### 单张图布局

每个角色一张图，布局规则：

```
┌──────────────────────────────────────┐
│  左半（50%宽度）      右半（50%宽度）  │
│  ┌──────────┐    ┌──┬──┬──┐         │
│  │          │    │正 │侧 │背 │         │
│  │  面部    │    │面 │面 │面 │         │
│  │  特写    │    │全 │全 │全 │         │
│  │          │    │身 │身 │身 │         │
│  │          │    │  │  │  │         │
│  └──────────┘    └──┴──┴──┘         │
│      白底 (#FFFFFF)                    │
└──────────────────────────────────────┘
```

### 人物提示词模板

```
[角色名]角色设定图。左半：面部特写，中性表情，五官清晰可见，正面朝向镜头。右半：正面/侧面/背面三视图全身，标准A-pose，无动作姿态。2D国漫赛璐璐画风，2D手绘赛璐璐（平涂色块+清晰阴影分界线+手绘感高光笔触），高饱和色彩。白色背景(#FFFFFF)。参考国产动画《一念永恒》《凡人修仙传》赛璐璐画风。无文字标注，无辅助线。
```

### 人物提示词编写规则

- 使用叙事描述式写法（非技术参数列表）
- 每条约 80-150 中文字
- 不写具体颜色名——服装颜色由参考图定义
- 不写具体面料名——材质由参考图定义
- 只写"角色名的衣服"或"角色名的衣袍"
- 标准姿态：T-pose 或 A-pose，无动作
- 面部：中性表情，五官清晰

## 场景参考图格式

### 宫格制

| 场景数 | 宫格 | 说明 |
|--------|------|------|
| ≤9 个 | 3×3 | 一张图 9 格，每格一个场景 |
| 10-12 个 | 3×4 | 一张图 12 格 |
| 13-16 个 | 4×4 | 一张图 16 格 |

超过 16 个场景时，按场景类型分组，每组一张宫格图。

### 场景提示词模板（单格）

```
[场景名]：[地点类型]，[空间尺度]，[光线氛围]，[关键建筑/家具特征]，[天气/时间]。2D国漫赛璐璐画风，2D手绘场景，水彩薄涂+多层景深，绘画感后处理。参考国产动画场景美术——前景细节+中景层次+远景淡墨晕染。无人物。
```

### 场景宫格提示词模板

```
2×3场景宫格图。左上格：[场景1描述]。中上格：[场景2描述]。右上格：[场景3描述]。左中格：[场景4描述]。正中格：[场景5描述]。右中格：[场景6描述]。左下格：[场景7描述]。中下格：[场景8描述]。右下格：[场景9描述]。所有场景统一2D国漫赛璐璐画风，主光源方向一致。白色分隔线，每格等大。无人物，无文字标注。
```

## 跨集素材管理

素材库位于 `谁带这家伙进修士圈的/assets/`，跨集累积：

### 人物素材管理

| 情况 | 处理 |
|------|------|
| 新角色 | 追加到 `character-prompts.md` 末尾 |
| 已有角色·无变化 | 标注"复用"，跳过生成 |
| 已有角色·换装 | 标注"变体"，在原条目下追加新提示词 |
| 已有角色·新表情 | 不单独生成（表情变化在分镜中由即梦自动处理） |

### 场景素材管理

| 情况 | 处理 |
|------|------|
| 新场景 | 追加到 `scene-prompts.md`，视场景总数决定宫格图重排 |
| 已有场景·无变化 | 标注"复用" |
| 已有场景·时间/天气变化 | 标注"变体"，追加新条目 |
| 场景数溢出当前宫格 | 新增宫格图（如 3×3→3×4） |

### 素材文件格式

```markdown
# 人物参考图提示词（全局素材库）

## @图片1 — [角色A]
[提示词文本]
状态：已生成 / 待生成 / 本集复用

## @图片2 — [角色B]
[提示词文本]
状态：已生成

---

# 场景参考图提示词（全局素材库）

## @图片1 — [场景A]
[提示词文本]
状态：已生成

## @图片2 — [场景B]（变体：夜景版）
[提示词文本]
状态：待生成
```

## 文生图工具使用说明

参考图提示词编写完成后，需使用文生图工具（如即梦、Nano Banana Pro）生成实际图片：

1. 将每个提示词粘贴到文生图工具
2. 场景使用宫格提示词一次性生成 → 逐格裁切为独立场景参考图
3. 人物逐一生成 → 上传至即梦 Seedance 作为参考图
4. 在即梦中分配 @编号（如@图片1、@图片2）

这一步目前是手动的——AI 只生成提示词文本，不生成图片。
