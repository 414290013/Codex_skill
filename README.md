# Codex Skill Backup

This repository backs up the local Codex skill `ai-comic-storyboard-prompts`.

Local source path at backup time:

```text
C:\Users\Administrator\.codex\skills\ai-comic-storyboard-prompts
```

To restore on the same machine, copy the `ai-comic-storyboard-prompts` folder into:

```text
C:\Users\Administrator\.codex\skills\
```
